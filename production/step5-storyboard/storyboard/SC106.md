## 场次概述

在卢老板的餐馆中，老吴和宋小仙正在用餐，小顶在此值班。小顶热情地发现二人是初次来镇上，并意识到他们都来自永安镇——小顶的故乡。三人在看似随意的谈话中建立了同乡的亲切感。老吴和宋小仙敏锐地识别出小顶是他们打入朱围庸势力的关键突破口——小顶身处餐馆工作，地理位置优越，人脉关系单纯，且尚未被朱围庸的势力圈套复杂化。这场戏通过温暖而日常的互动，暗示了即将到来的行动的人性成本。

## 场景总结

**情绪弧线**：温暖的日常 → 同乡的惊喜 → 亲切的连结 → 冷血的计算 → 最后的温暖与隐藏的目的

**关键视觉元素**：
- 老吴的假胡子与粗布衣——伪装的可视化
- 宋小仙的素框眼镜——身份切换的工具
- 小顶的热情与天真——对即将被利用的人物的人道化呈现
- 餐馆的午间光线——温暖而安全的假象中隐藏的危险

**人物弧线**：
- 老吴：从陌生警觉 → 伪装亲切 → 目标确认 → 温暖离别（保持长者姿态）
- 宋小仙：从助手隐身 → 评估分析 → 谨慎提醒 → 无害退出（保持低调）
- 小顶：从热心服务 → 同乡惊喜 → 真诚信任 → 充满期待的离别（完全不知真实意图）

**场景意义**：这场戏通过看似日常而温暖的互动，展现了老吴和宋小仙如何利用"人性的善良"来推进他们的调查。小顶的热情与信任成了他的弱点，而老吴与宋小仙的伪装与计算则成了必要的手段。这场戏埋下了后续"利用小顶"的伏笔，同时也暗示了战争中无辜者被卷入的不可避免性。
