## 场次概述

正值严冬腊月，老吴出门时并未料到会在山林迷路。随着天色完全暗淡，他开始陷入无助的境地——饥肠辘辘、能量消耗极快、大风夹雪花扑面而至。关键时刻，他听到了水流的声音，顺着声音找去发现了一条清澈小溪。沿小溪继续前行，他来到一个圆形小池塘，池塘旁有一块两米多高的大石头。他靠在石头上休息，但天色越来越黑，雪下得越来越大，北风如刀刮过脸庞。体力几乎耗尽，老吴的心态急转直下，他甚至打算在这里等死。绝望中，他走到池塘边，捧起泉水喝了几口——这是他第一次接触到这神秘的泉水，虽然他不知道这意味着什么，但那一刻画面应该有仪式感。饮完泉水后，老吴回到大石头旁躺下，在过度劳累中陷入昏迷。这个场次是全片的"死谷时刻"——最黑暗的一刻，但紧接着就是黎明与奇迹。

**画面描述**

Cinematic still, 2.39:1 widescreen, 35mm film grain texture, medium-high contrast, photorealistic。

**前景**：飘落的大雪片在画面中形成白色的纹理，雪花密集到几乎遮蔽视线。镜头从极高空（约800米）往下推进，雪的粒子感逐步加强，营造出穿过雪幕的沉浸感。

**中景**：山林的树冠呈现出黑色的团块状，枝条在雪的覆盖下勾勒出复杂的纹理。其中一个极其渺小的白色光点在黑色森林中微微摇晃——那是老吴手中的手电筒。手电筒的光线已经很暗，只能照亮他脚下一小圈区域。他的身形在极高的俯视角度下看不清具体样貌，但能看到他的步伐杂乱无序，不是直线前行而是在林间曲折迂回——这是一个迷路者的行走方式。

**背景**：漆黑的群山在雪夜中只能看到隐约的轮廓。北方天空还有极淡的深蓝色，预示着黎明远在数小时之外。远处没有任何灯火，甚至没有人类居住的迹象——老吴已经彻底进入了无人区。

**动态描述**

1. 0-4s：航拍镜头从极高空缓慢向下推进。雪幕越来越密集，视野也越来越模糊。画面中那个白色的光点——老吴的手电筒——在黑色树林中摇晃着前进。他的步伐时而快时而慢，显得犹豫不决。

2. 4-8s：镜头继续下降，高度降至约300米。现在能看到更清晰的地面细节：没有踩踏过的积雪、密集的树干、山林的复杂地形。老吴的身形在画面中逐步变大，但他仍然像一个渺小的蚂蚁。他停顿了一下，转过身看了看来时的方向，似乎想确认能否原路返回。但黑暗中哪里都一样，他又转身继续向前。

3. 8-12s：镜头降至约100米高度。现在能清晰看到老吴的背影——一个穿着厚棉外套的老人，肩膀已经微微耸起（因为寒冷和恐惧），他的手电筒光线在黑暗中显得格外孤独。镜头在此高度悬停，不再下降。远景中能听到风声渐强。

**对白**

无。

**音效提示**

- 环境音：北风呼啸声，声音低沉沉闷，频率约100-300Hz，制造压抑感。雪被风吹过林间的沙沙声，时而清晰时而模糊，暗示风速在变化。
- 动作音：老吴踩踏积雪的声音极弱（几乎听不见），这反而加强了风声的主导地位，强化了自然的压倒性。
- 情绪音：低频弦乐进入，两把大提琴在低音域重复一个不安的音型，节奏与风声缓慢合拍。钢琴在高音域偶尔敲出孤立的音符，宛如绝望的心跳。

**画面描述**

Cinematic still, 2.39:1 widescreen, 35mm film grain texture, medium-high contrast, photorealistic。

**前景**：积雪覆盖的地面占据画面下方三分之一。雪面在手电筒光线下呈现出凹凸不平的质感，某些地方有深色的石头露出来，其他地方被积雪覆盖。镜头的低角度使得观众视线与老吴的视线接近——两人一起仰望前方。

**中景**：老吴的身体占据画面中心。他停下了脚步，抬起头，竖起耳朵。他的身形在手电筒的逆光中显得更加苍白和瘦弱。他穿的厚棉外套已经被雪花打湿，肩膀和头顶都沾满了白色的雪。他的脸部被手电筒的微弱光线照亮——眼睛睁得很大，眉毛因为集中注意力而紧锁。他的嘴巴微张，呼出的热气在冷空气中凝结成细微的白雾。最关键的是他的眼神——从之前的茫然和恐惧，现在转变成了一种急切的希望。他的右手抬起来，做出"竖起耳朵听"的姿态，帽子和手指在手电筒光下投出长长的影子。

**背景**：黑色的树林在镜头背景中成为几乎无法分辨的深色团块。被照亮的树干显示出树皮上的苔藓和冰霜。雪片在镜头前飘落，某些落在他的肩膀上，某些消失在黑暗中。

**动态描述**

1. 0-3s：老吴在行走中突然停下了脚步。他的身体有一瞬间的僵直，随即慢慢转过身来。他的手电筒光线扫过树林，似乎在寻找声音的来源。画面中风声有短暂的减弱，水流的声音开始清晰起来——那是一种细微的、有规律的涓涓流水声，与风的不规则鸣叫形成对比。

2. 3-7s：老吴抬起头，脸上的表情开始变化——从困惑和恐惧，逐步转变成惊喜和渴望。他的嘴角微微上扬，眼睛睁得更大了。他转过身，沿着声音的方向，手中的手电筒光线开始有目标地指向一个方向——那声音中隐含的水，成为了他心中的救命稻草。

3. 7-10s：老吴开始向水声的方向行走。他的步伐比之前更加坚定，不再是迷茫的曲折，而是直奔某个目标。他回过头看了一眼来时的方向，仿佛在确认——是的，我听到了水，我要活下去。镜头保持静止，看着他的身影逐步消失在黑暗中，只有手电筒的光点还在摇晃。

**对白**

无（可以考虑极弱的自言自语"听...听到了..."，但原文中无明确记录）。

**音效提示**

- 环境音：北风声仍然存在但逐步退到背景，水流声逐步增强并成为前景音。水流声应该是清冷的、有韵律的、充满生命力的——与死寂的风声形成对比。
- 动作音：老吴踩踏积雪的声音，脚步声变得更加清晰有力（因为有了目标）。
- 情绪音：之前的不安弦乐逐步消退，被一个新的音型替代——高音竖琴轻轻拨动，暗示希望的闪现。水流声的混响制造出空间感，暗示他正在接近某个开阔的地方。

**衔接说明**

- 与上一镜头：硬切自 SC004-001，从航拍俯视切至地面仰视
- 与下一镜头：硬切到 SC004-003，沿小溪前行的追踪镜头

**首帧合成提示**（供阶段6使用）

- 连续性需求：
  - 前接：SC004-001（同一人物，衣着积雪状态延续）
  - 后接：SC004-003（老吴沿溪流前行，衣着继续积雪）
- 参考图使用建议：夜间山林中低角度仰视一个老人、突然听到声音并抬头的场景。重点是表情的转变——从绝望到希望的瞬间。手电筒应该是画面中唯一的人造光源，周围的树木在其光线下显出怖人的影子。老人的呼吸白雾应该清晰可见。

**画面描述**

Cinematic still, 2.39:1 widescreen, 35mm film grain texture, medium-high contrast, photorealistic。

**前景**：老吴的身体占据画面中心，我们从他身后跟拍。他的肩膀、背部、两条腿清晰可见。他的右手举着手电筒，光线前伸照亮前方的道路。左手时而扶住树干以保持平衡，时而垂下。他的衣物已经被雪花和水溅打湿，背部贴着湿润的面料，显示出身体的轮廓。每一步都显得沉重而谨慎。

**中景**：老吴前方的山溪逐步呈现。溪流在雪夜中显出深色的水体，上面覆盖着薄薄的冰层和浮冰。溪流两侧的卵石被雪覆盖，只有水冲刷的地方显出黑色的石头。溪水虽然严冬但仍在流动，发出清晰的潺潺声。溪流两侧的植被（灌木、低矮的树）在手电筒光下显出黑色的轮廓。水面在月光被乌云遮蔽的情况下反射不出光芒，但手电筒的光线在水面上形成了一条金黄色的光路，宛如指引老吴前进的方向。

**背景**：溪谷两侧的林壁在黑暗中显出只有轮廓的树木。远处的山林完全漆黑，看不清具体形状。唯一的参照是手电筒光线能够到达的范围——大约10-15米开外，之后就是无尽的黑暗。

**动态描述**

1. 0-4s：老吴沿着溪流的上游方向前行。他的步伐虽然沉重但比之前有了方向性。手电筒的光线跟随他的移动而移动，在溪流上形成舞动的金黄色光带。镜头（代表观看者的视线）跟随在他身后，保持中等距离，使我们仿佛也在沿着他的脚步前行。风声渐弱，水流声渐强。

2. 4-9s：溪流开始变宽。老吴停顿了一下，抬头看了看溪流前方。他的呼吸已经变得急促（我们能听到他有力的喘气声）。他用手电筒照向溪流上游，手电筒光线穿过雪片和树间的空隙，照在远处某个地方——那里应该就是泉池的位置，但此刻还看不清楚。老吴的头部轻微地摇晃了一下（在寻找最稳妥的渡溪路线），然后他迈入了溪流。冰冷的水流漫过他的脚踝，他倒吸了一口冷气，但他没有停下来——求生的欲望已经超越了对寒冷的恐惧。

3. 9-14s：老吴继续沿溪流前行。镜头的运动逐步加快（但仍然跟随他的步伐），体现了他行动的加速和心中希望的增长。溪流的潺潺声已经成为画面最主导的声音，压过了风声。老吴的身影在手电筒光和月光稀疏透射的照射下显出灰白色的轮廓。他的呼吸声变得有节奏起来，仿佛他正在对自己说"快到了，快到了"。

**对白**

无。

**音效提示**

- 环境音：溪流声成为绝对主导，清脆的水流声频率约500-2000Hz，与之前低沉的风声形成对比。溪流声的节奏和韵律产生了节拍感，仿佛在引导老吴前进。
- 动作音：脚踩在湿润的石头和积雪上的声音，水流冲过老吴脚踝的声音，衣物被水和冷风吹干的摩擦声。
- 情绪音：高音竖琴的拨弦变得更加活跃，频率加快，与溪流声和谐共鸣。一个新的弦乐旋律在背景中若隐若现，暗示希望正在接近。

## 场次视觉总结

SC004是全片在视觉上最具张力的场景之一。色调从暖色（SC003末尾的施工区）快速转变为冷色（夜间山林），然后进入极限的黑色与白色的对比。老吴的面部表情经历了快速的转变：恐惧 → 绝望 → 虚弱的希望 → 深层绝望 → 虔诚 → 放弃。这个情感弧线应该通过镜头、光线、音效的多维度配合来呈现，而不是依靠台词和表演。

关键视觉要素：
- **光线**：从微弱的手电筒光 → 极弱的月光 → 完全黑暗 → 泉水微光
- **身体语言**：从直立行走 → 蜷缩 → 靠在石头上 → 半蹲 → 完全躺下
- **环境对比**：从稀疏的树林 → 开阔的溪流 → 圆形池塘的宁静
- **声景演变**：从恐怖的风声 → 引导的水流声 → 绝望的寂静 → 仪式感的水声 → 最终的沉默
