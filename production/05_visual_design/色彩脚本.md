# 视觉设计：色彩脚本

## 一、阳间色彩方案

### 1.1 基础色彩体系
#### 核心色调
- **主色调**：温暖的土黄色和金黄色（RGB: 210-240, 170-200, 100-130）
- **辅助色**：翠绿色和浅绿色（RGB: 100-160, 150-200, 80-120）
- **强调色**：生动的橙红色和朝阳金色（RGB: 220-255, 100-150, 40-80）
- **中性色**：米白色、浅灰色和浅棕色（RGB: 220-240, 210-230, 190-210）

#### 色温特征
- **整体色温**：暖色调，色温约为4000-5500K
- **光线品质**：柔和而充满生气，如同被太阳照亮的世界
- **饱和度**：中等至高饱和度，色彩生动而不过度刺眼

### 1.2 生机活力色彩
#### 白天场景
- **早晨**（日出-10:00）
  - 主色：柔和的橙黄色和粉紫色（RGB: 255, 180, 120）和（RGB: 200, 150, 200）
  - 特点：温暖但柔和，充满新生的希望
  - 饱和度：相对较低，给人梦幻感

- **正午**（10:00-14:00）
  - 主色：金黄色和纯白色（RGB: 255, 220, 80）和（RGB: 255, 255, 255）
  - 特点：炽热、明亮、生机勃勃
  - 饱和度：高，色彩鲜明有力

- **傍晚**（14:00-日落）
  - 主色：金橙色和紫红色（RGB: 255, 160, 50）和（RGB: 200, 100, 120）
  - 特点：温暖而深沉，充满诗意的余晖
  - 饱和度：中等，色彩既饱和又柔和

#### 夜晚场景（有光源处）
- **篝火周边**：金黄色和橙红色（RGB: 255, 180, 60）照亮周围
- **月光场景**：清冷的银蓝色（RGB: 200, 220, 240）
- **灯火场景**：温暖的黄白色（RGB: 255, 200, 100）
- **星光场景**：深蓝色背景（RGB: 20, 40, 80）配以稀疏的白点光

### 1.3 希望与生命的视觉语言
#### 自然元素的色彩
- **植物绿**：鲜活的翠绿和橄榄绿（RGB: 100, 180, 80）（RGB: 140, 160, 100）
- **清水蓝**：透亮的浅蓝和天空蓝（RGB: 100, 200, 240）（RGB: 135, 206, 235）
- **土壤棕**：温暖的棕褐色（RGB: 150, 100, 50）
- **花卉色**：多样的彩色点缀，如红花（RGB: 220, 50, 80）、黄花（RGB: 255, 210, 0）

#### 建筑与文化元素
- **传统建筑**：深棕色的木质结构（RGB: 100, 60, 30）配以鲜红色的木漆（RGB: 200, 50, 50）
- **现代建筑**（第五部）：浅灰色和白色的钢结构（RGB: 180, 180, 180）配以玻璃反光
- **宗教建筑**：金色的装饰（RGB: 220, 180, 40）配以深红色的砖墙（RGB: 150, 80, 80）

### 1.4 人物肤色与服装
#### 人物肤色
- **东亚人种**：温暖的黄褐色（RGB: 200, 160, 120）
- **光线下的变化**：阳光下泛红（增加红色成分20-30%），阴影处变暗并偏冷（减少红黄色成分，增加蓝色成分）

#### 服装色彩搭配
- **朱围庸的演变**
  - 乞丐装：脏污的灰褐色和黑色（RGB: 100, 80, 60）
  - 普通衣着：温暖的棕色和绿色（RGB: 140, 100, 60）（RGB: 100, 140, 80）
  - 富商装：深蓝色和金色（RGB: 50, 80, 150）（RGB: 220, 180, 40）
  - 权力者装：深紫红色和黑色（RGB: 100, 50, 80）（RGB: 40, 40, 40）

- **配角服装**
  - 老吴夫妇：朴素的灰色和棕色
  - 商人角色：相对鲜艳的衣着，反映其社会地位
  - 士兵角色：统一的军装颜色，通常为深蓝或深绿

## 二、阴间色彩方案

### 2.1 基础色彩体系
#### 核心色调
- **主色调**：阴冷的深蓝色和深紫色（RGB: 50-100, 60-120, 150-200）
- **辅助色**：灰紫色和石灰色（RGB: 120-140, 100-130, 140-160）
- **强调色**：诡异的蓝紫光和幽灵白（RGB: 100-150, 80-150, 200-255）
- **中性色**：深灰色和黑色（RGB: 60-100, 60-100, 60-100）

#### 色温特征
- **整体色温**：冷色调，色温约为6500-8000K，偏向蓝色
- **光线品质**：诡异、压抑、不自然的光线，如同月光或荧光
- **饱和度**：低至中等饱和度，色彩沉闷而压抑

### 2.2 绝望与痛苦色彩
#### 恶灵区的视觉特征
- **主环境色**：极暗的蓝紫色（RGB: 40, 50, 100）
- **灵魂发光**：蓝紫色的自发光（RGB: 100, 100, 200）
- **岩石色**：黑灰色的坚硬感（RGB: 60, 60, 70）
- **不祥色**：深暗的紫红色（RGB: 80, 40, 80）表达痛苦和堕落

#### 十八层塔的色彩递进
- **第一层**（入口）：深蓝色，仍可见微弱的天光
- **中间层**（7-10层）：深紫色和黑色的混合，完全无天光
- **底层**（15-18层）：纯黑色和极暗的紫色，仅靠灵魂自光可见
- **恶灵区**：诡异的蓝绿色微光（RGB: 50, 100, 80），散发不祥之感

### 2.3 痛苦与酷刑的视觉语言
#### 火焰与酷刑
- **炼狱火**：蓝白色的不自然火焰（RGB: 100, 150, 255），与阳间的黄红火完全不同
- **鞭痕**：深红色和黑色的创伤色（RGB: 100, 20, 20）
- **锁链**：银灰色的冰冷金属（RGB: 150, 150, 160）

#### 灵魂的痛苦表现
- **受伤灵魂**：色彩变暗，蓝紫色降低饱和度，增加黑色成分
- **濒临溃散**：黑色和紫色的交替闪烁，边界模糊不清
- **绝望**：几乎纯黑色的灵魂（RGB: 20, 20, 40）

### 2.4 人物与环境交互
#### 灵魂在阴间的视觉
- **普通灵魂**：蓝紫色自发光，亮度中等，清晰可见
- **恶灵**：暗紫色或黑紫色，亮度较低，显得阴沉
- **强大灵魂**：色彩更鲜艳，亮度更高，如灯塔般突出

#### 建筑与装置
- **塔身**：灰黑色的坚硬岩石（RGB: 80, 80, 100）
- **酷刑器具**：冷硬的黑色和银灰色
- **护栏和链条**：银色和暗灰色（RGB: 140, 140, 150）

## 三、按章节/幕的色彩情绪曲线

### 3.1 第一部（朱围庸诞生与逃脱）

#### 1-3章：老吴的绝望到重生
- **色彩方向**：灰冷 → 温暖 → 希望
- **主要场景色**：
  - 派出所场景：冷色调的官僚气氛，灰色和蓝灰色
  - 山林迷路：冷色的夜幕，深蓝和黑色
  - 发现泉水：逐渐温暖的金黄色光线
  - 篝火取暖：温暖的橙黄色笼罩全身

#### 4-8章：朱围庸的苦难与希望
- **色彩方向**：绝望的黑紫色 → 逃脱的挣扎蓝色 → 重生的金黄色
- **主要场景色**：
  - 恶灵区：阴冷的深蓝紫色
  - 十八层塔：从第一层的冷蓝逐层加深到底层的纯黑
  - 树根通道：灰白色的树根（RGB: 180, 170, 160）
  - 美女榕根部：温暖的土黄色和生机的绿色

### 3.2 第二部（泊岗镇的权力游戏）

#### 10-14章：融入与建设
- **色彩方向**：朴素的棕褐色 → 商业的金色 → 权力的深蓝色
- **主要场景色**：
  - 永安镇：传统的棕色和灰色调
  - 西瓜市场：鲜活的绿色和红色点缀
  - 泊岗镇：海洋的蓝色和商业的金色
  - 黄亮伟办公室：深蓝色和金色的权力象征

#### 15-17章：冲突与胜利
- **色彩方向**：金色的野心 → 蓝色的冲突 → 深蓝色的权力
- **主要场景色**：
  - 海上夜袭：深蓝的海水和黑色的夜幕
  - 灵魂谈判：蓝紫色的灵魂光线与黑暗的对比
  - 港口场景：金色的成功光线

### 3.3 第三部（重聚与牺牲）

#### 18-20章：孤独到决断
- **色彩方向**：孤独的月白色 → 决心的深蓝色 → 行动的冷色调
- **主要场景色**：
  - 月下漫步：清冷的银蓝色月光
  - 决定赴阴间：冷色调的内心戏
  - 美女榕：回到温暖的土黄色和生命绿

#### 21-24章：阴间救援
- **色彩方向**：绝望的黑紫色 → 希望的蓝色 → 成功的温暖色
- **主要场景色**：
  - 通过塔层：逐层加深的蓝紫到黑色
  - 恶灵区相会：蓝紫色的灵魂聚集
  - 还阳过程：蓝色到金黄色的过渡
  - 旅馆场景：温暖的黄白色光线

### 3.4 第四部（战争与陷阱）

#### 25-38章：商业竞争到军事冲突
- **色彩方向**：商业的金色 → 冲突的血红色 → 战争的深蓝灰色
- **主要场景色**：
  - 航运市场：金色和白色的繁荣
  - 价格战：橙色的紧张和压力
  - 军事集结：深蓝色和暗灰色的战争色调
  - 荣康城：传统的棕红色砖墙

#### 39-49章：战争激烈化
- **色彩方向**：深蓝灰的战争 → 血红的暴力 → 黑暗的绝望
- **主要场景色**：
  - 外城之战：混乱的多色混合（战火的红、血的深红、烟的灰）
  - 夜间奇袭：黑色夜幕中蓝紫色灵魂的活动
  - 内城冲突：混乱的光线和色彩
  - 胜利时刻：恢复秩序的深蓝和金色

### 3.5 第五部（现代与救赎）

#### 50-71章：传承到未来
- **色彩方向**：现代的冷色 → 传统的温暖色 → 融合的新色调
- **主要场景色**：
  - 现代城市：灰色、白色和冷蓝色的建筑
  - 传统村落：棕色、黄色的温暖调
  - 混合场景：冷色与暖色的对比与融合
  - 救赎时刻：金色的希望与和解

## 四、关键转场的色彩变化

### 4.1 阳间→阴间的转场
#### 视觉效果
- **渐进式冷化**：温暖的金黄色逐渐冷却为蓝色，再逐渐深化为紫色
- **光线变化**：从自然日光逐渐变为诡异的灵光
- **饱和度变化**：从高饱和逐渐降低至低饱和
- **过程时间**：3-8秒的视觉过渡

#### 具体场景
- **朱围庸的逃脱**：攀登塔层时，色彩从顶部的金黄色逐层变为深蓝紫色
- **灵魂上升**：灵魂从阴间回到阳间，色彩从深蓝逐渐回暖至金黄色
- **附身过程**：新宿主获得新灵魂时，身体色彩短暂冷化再回暖

### 4.2 现代→古代的转场（第五部特有）
#### 视觉效果
- **色彩转换**：冷色的现代城市色彩逐渐转换为温暖的古代色彩
- **质感变化**：光亮的现代材质逐渐变为古旧的纹理
- **光源变化**：人工灯光逐渐转换为自然光或火光

### 4.3 和平→战争的转场
#### 视觉效果
- **色彩升温**：平和的冷蓝色逐渐升温为紧张的橙红色
- **明度降低**：明亮逐渐变为昏暗，强调压抑的战争气氛
- **饱和度增加**：色彩饱和度逐渐增加，强调冲突的激烈
- **光线混乱**：统一的光源逐渐变为混乱的多光源（火焰、手电筒等）

### 4.4 生命→死亡的转场
#### 视觉效果
- **色彩流失**：生机勃勃的多色世界逐渐失去颜色，变为灰色和黑色
- **温度下降**：温暖的色调逐渐冷化为冷灰色和冷蓝色
- **光线暗淡**：光线逐渐熄灭，直至完全黑暗
- **符号化过程**：可见灵魂从宿主身体浮出，显示生命的离去

## 五、实际制作指南

### 5.1 色彩分级建议
- **阳间**：色温设定在4000-5500K，饱和度在60-85%范围内
- **阴间**：色温设定在6500-8000K偏蓝，饱和度在30-60%范围内
- **战争场景**：色温在3000-4000K（火光）混杂8000K（蓝色冷光），饱和度在70-90%

### 5.2 灯光配置
- **阳间自然光**：使用偏暖的灯色（3200K）配合反光板，增加金黄色
- **阴间诡异光**：使用高色温蓝光（7000K+）配合冷光源
- **混合场景**：同时使用暖光和冷光，产生强烈的色彩对比

### 5.3 后期制作
- **色彩分级**：针对不同场景进行分别的色彩调整
- **色彩一致性**：确保同一区间（阳间或阴间）的色彩调性一致
- **过渡效果**：使用渐变和淡入淡出效果处理转场色彩变化
- **符号化处理**：强化色彩的心理学含义，加强视觉说服力

---

**制作备注**：
- 色彩脚本应与音乐和音效的情绪相呼应，形成完整的感官体验
- 演员的化妆和服装应与色彩脚本的色调相匹配
- 外景拍摄时应充分利用自然光的色温变化（早晚的暖光、正午的白光）
- 内景拍摄应提前进行色温测试，确保与色彩脚本一致
- 色彩脚本的执行应贯穿前期筹划、拍摄和后期制作的全过程
