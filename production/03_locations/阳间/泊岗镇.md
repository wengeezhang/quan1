# 泊岗镇 - 场景圣经

## 一、场景概述

**位置**：东方海洋中的一座离岸岛屿，距离大陆约100公里，可通过宁阁码头的客货两用船到达。小镇坐落在岛屿的主要登陆点附近，向内陆蜿蜒而上

**功能**：
- 朱围庸与小顶的新生与发展之地
- 故事第二部的主要活动场景
- 社会矛盾与人性光暗的具体化体现
- 朱围庸施展超能力与改造环境的实验场

**在故事中的重要性**：
- 成长场景：朱围庸从逃亡者转变为行动者，从被动应对到主动改造
- 能力开放：朱围庸首次在人间社会中大规模施展"移魂夺魄"等妖力
- 关键事件：智取裘万财的赌约与"疯癫"结局，成为故事的转折性事件
- 人生抉择：加入黄亮伟团队的决定，标志着朱围庸开始参与到社会权力关系中
- 象征意义：希望之地、自我证明之地、权力与改变的交汇点

## 二、空间结构

**岛屿整体布局**：
- 地形：呈圆形的小岛，周长约30公里，最高点约200米
- 面积：约80-100平方公里
- 植被：郁郁葱葱的森林覆盖约70%的岛屿
- 人口：约8000-12000人

**镇子空间分布**：

### 港口与码头区
- **登陆点**：位于岛屿西侧，是唯一的客货登陆点
  - 规模：可容纳2-3艘大型客货船同时靠岸
  - 设施：简单的栈桥、货物堆放区、售票处
  - 特点：人流频繁，货物堆积，显得繁忙与混乱
  - 黑暗处：地头蛇裘万财的"收费站"，每个游客必须交钱

- **滨海商业区**：位于登陆点上方，沿海边展开
  - 规模：长约1.5公里，宽约300-500米
  - 主要商业：鱼货交易市场、货物仓库、大小饭馆、旅馆
  - 特点：最繁荣的商业地带，人流最密集
  - 臭味：鱼腥与咸味混合的海港气息

### 内陆生活区
- **中心广场/街道**：位于登陆点北侧，向岛屿内陆延伸
  - 布局：宽敞的石板路，两侧是各类商铺与住宅
  - 特点：相对清洁与有序，是生活的核心区域
  - 分区：
    - 靠近码头段：商业与旅馆集中
    - 中段：饭馆、茶馆、日用品店
    - 远段：住宅区与手工业作坊

- **山上住宅区**：位于岛屿高地
  - 特点：环境幽静，多为本地商人与官员的住宅
  - 黄亮伟的家就在这个区域
  - 视野开阔，可俯瞰整个港口与镇子

### 其他关键位置
- **码头搬运区**：港口周边的货物处理区
  - 特点：朱围庸与小顶最初的工作地点
  - 设施：简单的工具、货物、水源

- **餐馆区**：位于中心街道的餐饮一条街
  - 特点：小顶工作的餐馆在此，是故事的另一个重要活动地点
  - 特征：拥挤、嘈杂、食客众多

- **山地与森林**：占据岛屿大部分面积
  - 特点：朱围庸与裘万财对话与施展秘术的地点
  - 视觉：原始森林、蜿蜒山路、神秘洞窟（暗示通往阴间）
  - 气氛：与泉水池塘相似的超自然属性

**空间特征总结**：
- 垂直分层明显：港口（底层，混乱）→ 中心街道（中层，有序）→ 山上住宅（上层，精英）
- 商业与生活混杂，显示小镇的自然生长
- 自然与人为交织，森林与建筑的平衡
- 海岛特性：受海风影响，气候宜人但环境受限

## 三、氛围设计

### 光线设计

**白天场景**：
- 光线色温：明亮而温暖，偏金黄色（6500-7000K）
  - 海岛气候特有的强烈日照
  - 海面反射增强光线强度
  - 空气清晰度高，光线穿透力强

- 时间分段：
  - **清晨（5-7点）**：清冷的蓝白光，显得清爽而宁静
  - **上午（8-11点）**：金黄色阳光，充满希望与能量
  - **中午（12-14点）**：炽热的白光，光影强烈，人们寻求遮阴
  - **下午（15-17点）**：柔和的金色阳光，显得温暖而梦幻
  - **傍晚（18-20点）**：橙红色余晖，洒在海面上闪闪发光

- **特殊光线效果**：
  - 海面反光：闪闪烁烁的金色光芒，提醒观众海岛的特殊性
  - 港口光线：混乱与通透的对比（船与建筑造成的阴影区）
  - 中心街道光线：温暖与清晰的日常光线
  - 山上住宅光线：视野开阔，光线充足且凉爽
  - 森林光线：漫射与斑驳，神秘而压抑

**夜间场景**：
- 光源：月光（明亮的银白色）、街道油灯、店铺灯火
- 颜色：冷蓝色的月光与温黄色的人工光源混合
- 特点：港口区仍然繁忙（人力卸货），中心街道逐渐冷清
- 月圆之夜：特别明亮，强化了裘万财变得"疯癫"的诡异感

**季节光线特性**：
- 春季：清新的黄绿光
- 夏季：炽热的白光，阴影深邃
- 秋季：金黄色阳光，温暖但略显苍凉
- 冬季：清冷的蓝白光（故事发生于冬季）

### 色彩基调

**主色系**：
- 蓝绿色：海水与植被的颜色，象征生机与活力
- 金黄色：日光与海面反光，象征希望与财富
- 红褐色：船舶与建筑的颜色，象征人文与历史
- 青灰色：岩石与街道的颜色，象征沉稳与现实

**辅色系**：
- 白色：浪花、船帆、店铺墙壁
- 黑色：船底、建筑阴影、森林深处
- 绿色：植被与森林
- 棕色：木制建筑与木制船舶

**点缀色**：
- 鲜红色：商人的衣着、店铺招牌、海鲜的颜色
- 橙黄色：灯火与炉火
- 深紫色：夜空与远山

**不同区域的色彩特性**：

| 区域 | 主色 | 氛围 | 象征 |
|------|------|------|------|
| 港口码头 | 蓝色+褐色 | 混乱、繁忙 | 权力与压迫 |
| 滨海商业 | 金黄+红褐 | 活跃、拥挤 | 欲望与交易 |
| 中心街道 | 青灰+金黄 | 有序、温暖 | 日常与秩序 |
| 山上住宅 | 绿色+白色 | 宁静、优雅 | 权力与隐秘 |
| 森林深处 | 深绿+黑色 | 神秘、压抑 | 超越与超自然 |

**朱围庸视角的色彩变化**：
- **初到**：色彩明亮但陌生，充满新鲜感
- **工作期间**：色彩逐渐变得熟悉与亲切
- **对抗裘万财期间**：色彩饱和度升高，对比增强（正义与邪恶的视觉对立）
- **危机事件后**：色彩重新调整，加入权力关系的灰色调

### 声音环境

**港口区声效**：
- 船舶靠岸的声音：木船与木桩碰撞的重击声
- 卸货声：货物重重放下的砰砰声、工人的喊声
- 海浪声：持续的潮汐与海浪拍打声
- 船老板与工人的谈判声：讨价还价，偶尔有争执

**滨海商业区声效**：
- 鱼市场：鱼贩的叫卖声、杀鱼的声音
- 饭馆：餐具碰撞、食客谈笑、厨师喊菜的声音
- 人群：嘈杂的人声、脚步声、交易声
- 背景：海风与海浪的持续低频声音

**中心街道声效**：
- 商业活动：店铺开关门、商人的谈话
- 日常生活：扫地声、生活用品的交易声
- 行人：脚步声、闲聊声
- 背景：远处海浪声、树叶沙沙声

**山上住宅区声效**：
- 相对宁静，主要是自然声：风声、鸟鸣
- 偶尔的人声：佣人的活动声、主人的交谈声

**森林与山地声效**：
- 主导声：鸟鸣、虫鸣、风吹树叶声
- 神秘声效：远处洞窟的回音、地下水流声（暗示超自然）
- 朱围庸施展秘术时：低频的灵魂共鸣声、诡异的重音

**特殊时刻的声效**：
- 满月之夜裘万财变疯时：古怪的笑声、混乱的人声、恐惧的尖叫
- 黄亮伟价格战期间：商人的激烈谈判声、货物交易的混乱

### 气味与触感

**气味层面**：

| 地点 | 主要气味 | 强度 |
|------|---------|------|
| 港口 | 鱼腥、咸味、木材、沥青 | 强烈 |
| 滨海商业 | 食物香气、人体体味、咸海风 | 中等-强烈 |
| 街道 | 各类食物香气、人群气息、清新海风 | 中等 |
| 山上住宅 | 植被香气、木制家具、清新空气 | 柔和 |
| 森林 | 泥土、叶腐、潮湿感、植物香气 | 中等 |

**触感体验**：
- **海风**：咸湿而凉爽，常有强风，吹在脸上有刺痛感
- **地面**：石板路光滑（被人踩踏），有时湿滑
- **建筑物**：木制建筑粗糙，石制建筑坚硬而冰冷
- **衣物**：因为海风与潮湿，衣物常感湿润与黏腻
- **空气**：湿度高（海岛特性），夏日闷热，冬日清爽

**季节性触感**（故事发生于冬季）：
- 清晨有露水，触感湿冷
- 日中温暖，但不炎热
- 傍晚海风变冷，显得凉爽
- 雨后地面湿滑，需谨慎行走

## 四、关键视觉元素

### 必须出现的道具

1. **客货两用船**
   - 尺寸：长约60米，宽8米，吃水深度约3米
   - 结构：木制船体（可能部分铁皮加固）
   - 特点：甲板宽敞，可堆放货物；底舱分为货舱、操作室、休息室
   - 重要细节：黄亮伟的船，显得相对新旧适中，维护得还不错
   - 象征：连接陆地与海岛的通道，黄亮伟商业帝国的基础

2. **港口栈桥与卸货设施**
   - 简单的木制栈桥，可容纳多人同时通过
   - 货物堆放区：绳索、木板、各类货物
   - 收费站：简陋的亭子或台子，裘万财在此"工作"
   - 工人与搬运工具：木制推车、绳索、滑轮组

3. **中心街道**
   - 宽敞的石板路，长约500米
   - 两侧店铺：各类招牌、窗户、门扇
   - 特点：早期阶段保持整洁与有序
   - 关键店铺：小顶工作的餐馆（具体命名待定）

4. **餐馆内部**
   - 约5-8张餐桌，座位拥挤
   - 开放式厨房，可看到厨师工作
   - 收银台与食品展示区
   - 整体显得简陋但生意兴隆

5. **黄亮伟的家**
   - 位置：山上住宅区，视野开阔
   - 规模：中等二层建筑，约200平方米
   - 特点：相对豪华与整洁，显示财富与品味
   - 室内设施：书房、餐厅、卧室等
   - 象征：权力、财富与安全的象征

6. **山间寺庙或秘密地点**
   - 用于朱围庸与裘万财对话的地点
   - 可能是深山的洞穴或废弃建筑
   - 特点：隐蔽、神秘、显示超自然属性
   - 光线：昏暗，只有月光或火光照亮

### 植物元素

1. **岛屿森林**
   - 树种：混合的热带与亚热带树种
   - 特点：郁郁葱葱，覆盖约70%的岛屿
   - 季节性：冬季仍保持绿色，但略显稀疏

2. **港口周边的植被**
   - 为了建设码头而部分砍伐
   - 剩余的树木显得孤立而受创
   - 象征：人类活动对自然的破坏

3. **街道绿化**
   - 街道两侧有零星的大树，提供遮荫
   - 店铺前的花盆与绿植（表现店主的品味）

4. **山上住宅区的花园**
   - 黄亮伟家的花园显示精心打理与经济实力
   - 植物种类多样，显示品味

### 建筑与街景细节

- **木制建筑**：船舶与港口建筑的主要材料，显示海岛特色
- **石制路面**：街道的铺装，显示一定的建设规划
- **招牌与标识**：各类店铺的招牌，体现商业活跃
- **船舶元素**：船舶的索具、帆、木桩等，强化海岛身份
- **灯笼与油灯**：夜间照明工具
- **防浪堤或码头边墙**：港口的基础设施
- **市场摊位**：鱼市场的简陋摊位与冰块

### 人物形象与服饰

**朱围庸**：
- 初到：整洁的普通衣着，显示在永安镇的改造
- 工作期间：粗糙的工人服饰，显示劳动
- 后期：逐渐显示出"智者"的气质，衣着更显品味

**小顶**：
- 餐馆服务生的工作服
- 年轻活力的形象，显示希望

**黄亮伟**：
- 商人的标准形象：得体的衣着、显示财富与品味
- 温暖而有风度的外表

**裘万财**：
- 地头蛇的标准形象：嚣张、粗鲁的衣着与举止
- 后期"疯癫"状态：衣着凌乱、举止诡异

## 五、情绪曲线

### 第一阶段：抵达与适应（第11章后半至第12章初）

**情绪基调**：新奇→疲惫→适应→期待

- 初到岛屿的新鲜感与陌生感
- 收费风波的初次冲击与不公正体验
- 与小顶一起探索岛屿的欣喜
- 游览过程中的放松与融入
- 逐渐意识到现实的生存压力

**视觉变化**：
- 从明亮的港口转向温暖的街道
- 色彩从陌生的蓝绿转向熟悉的金黄
- 场景从拥挤转向开放，再转向亲切
- 最后一天的奢侈感转向现实的压力感

**关键场景转变**：
- 登陆→游览（视野逐渐扩大）
- 旅馆的短暂休息（身心的调整）
- 海边集市的热闹（融入小镇的感觉）
- 钱花完的现实（必须行动的清醒）

### 第二阶段：扎根与奋力（第12章中至第13章初）

**情绪基调**：决定→付出→满足→陪伴

- 朱围庸选择码头搬运工作，主动承担责任
- 日日夜夜的劳动中找到自我价值
- 小顶找到餐馆工作，两人有了经济支撑
- 对小镇的初步融入与安定感
- 与黄亮伟的友谊初步建立

**视觉变化**：
- 白天港口与街道的忙碌显示充实
- 金黄色阳光强化了努力与收获的感觉
- 两个人各自在工作地点的画面显示独立与互助
- 晚间的相聚显示温暖与羁绊

**关键场景转变**：
- 码头工作的艰辛（劳动的真实）
- 与黄亮伟的交流（贵人出现）
- 小顶的工作压力（成长的代价）
- 晚间两人的聚首（情谊的加深）

### 第三阶段：对抗与转折（第13章中至末）

**情绪基调**：正义→愤怒→决断→胜利→复杂

- 遇见裘万财的嚣张，激起正义感
- 与商户们立下赌约，显示自信与勇气
- 面对裘万财时的紧张与对峙
- 施展秘术成功的快感与力量感
- 赌约获胜的荣耀与陶醉
- 但随之而来的复杂情绪（对自身能力的思考）

**视觉变化**：
- 餐馆冲突场景：光线炽热，对比增强（正义与邪恶）
- 赌约期间：色彩饱和度升高，显示紧张与期待
- 山间秘术施展：光线昏暗与诡异，强化超自然感
- 胜利时刻：金光闪烁（重获阳光的象征）
- 裘万财"疯癫"：月圆之夜的冷蓝光，显示诡异与不安

**关键场景转变**：
- 餐馆遭遇→挺身而出（英雄姿态）
- 赌约确立→五日期限（悬念营造）
- 山间对话→秘术施展（高潮时刻）
- 裘万财变疯→道歉与改变（目标达成）
- 金钱与名望→内心的反思（复杂化的感受）

### 第四阶段：权力与抉择（第13章末至第14章）

**情绪基调**：胜利的后遗→被推崇→自我审视→选择性加入

- 朱围庸神秘消失，留下委托信（权力的运用）
- 小顶按照指示行动，完成赌款的分配（信任的体现）

## 十二、制作视觉指南

### 色彩参考

```
港口区：RGB(70, 140, 200) 蓝色 + RGB(180, 100, 60) 褐色
街道区：RGB(200, 180, 100) 金黄色 + RGB(120, 120, 100) 青灰色
住宅区：RGB(100, 180, 100) 绿色 + RGB(240, 240, 240) 白色
森林：RGB(50, 120, 70) 深绿色 + RGB(30, 30, 30) 黑色
```

### 光线参考

- **港口的金黄色日光**：色温6500K，强度高，形成高对比
- **街道的温暖日光**：色温6000K，强度中等，显示亲切
- **夜晚的月光**：色温3500K，强度低，显示神秘
- **火光照亮的秘密地点**：色温2000K，光线不稳定，显示紧张

### 氛围参考

- **港口的混乱度**：70%（人多、货多、噪音大）
- **街道的有序度**：60%（商业活跃但仍有秩序）
- **住宅区的宁静度**：80%（环境优美、人少、清静）
- **森林的神秘度**：90%（光线昏暗、声音诡异、气氛压抑）
