## 二、地理特征

### 2.1 河流走向

```
           上游
            │
            ↓
      ┌─────────────┐
      │   惠泽湖     │ ← 曲江形成的巨大湖泊
      │ (荣康城北)   │
      └──────┬──────┘
             │
        荣康城 ←── 引水通道（隐蔽）
             │
        广通桥
             │
        渡鹿原
             │
            ↓
          下游
```

### 2.2 重要节点

| 节点 | 位置 | 特征 |
|------|------|------|
| 惠泽湖 | 荣康城正北 | 曲江流入形成的巨大湖泊 |
| 引水口 | 荣康城西 | 隐蔽的城市水源通道 |
| 广通桥 | 曲江上 | 宽5米，石墩桥，非常坚固 |
| 渡鹿原 | 曲江对岸 | 一大片荒野 |

### 2.3 与城市的关系
- **荣康城**：几百年来依靠曲江水生活
- **同江城**：曲江流经，形成天然屏障
- 是多个城市的生命线

## 三、战略价值

### 3.1 水源意义
- 荣康城70万人口的饮用水源
- 农业灌溉
- 生活用水
- **控制水源 = 控制城市命脉**

### 3.2 军事意义
- 天然防御屏障
- 渡河困难增加攻城难度
- 广通桥是关键通道
- 封堵引水口可制造城内混乱

### 3.3 朱围庸的战术
- 封堵曲江引水口
- 制造城内水源危机
- 引发恐慌和混乱
- 削弱守军抵抗意志

---

## 四、重要地标

### 4.1 惠泽湖

**位置**：荣康城正北方向

**特征**
| 属性 | 描述 |
|------|------|
| 形成 | 曲江流入形成 |
| 规模 | 巨大湖泊 |
| 功能 | 水源蓄水、调节 |
| 景观 | 波光粼粼、水天一色 |

### 4.2 广通桥

**位置**：曲江上的重要渡口

**特征**
| 属性 | 数值/描述 |
|------|-----------|
| 宽度 | 5米 |
| 结构 | 石墩桥 |
| 特点 | 非常坚固 |
| 功能 | 重要交通要道 |

### 4.3 隐蔽引水通道

**位置**：荣康城西侧

**特征**
- 将城西的曲江水引入内城区
- 隐蔽的水源通道
- 城市生存的命脉
- 成为军事目标

## 八、制作建议

### 8.1 置景要点
- 河流要有流动感
- 广通桥的厚重历史感
- 两岸环境的层次
- 和平与战争的对比

### 8.2 特效需求
- 水流效果
- 波光反射
- 战时的混浊与血色
- 断水的干涸效果

### 8.3 场景转换
- 同一场景不同时期的对比
- 展现战争对环境的破坏
- 突出水源的重要性
