## 二、地理环境

### 2.1 位置与交通

**地理关系**
- 位于永安镇东方约三百里
- 海岛独立于大陆
- 通过船运与外界连接

**交通路线**
```
永安镇 ──(长途汽车约2小时)──> 宁阁码头 ──(船运)──> 泊岗码头
                                    │
                                    └── 预计下午抵达
```

### 2.2 岛屿特征

**整体形象**
- 宛如璀璨珍珠，静躺海天交界
- 宁静祥和的氛围
- 郁郁葱葱的森林覆盖大部分区域
- 偶见高耸房屋从树林中探出

**登岛地形**
```
                    岛内中心区
                        ↑
          ┌─────────────┴─────────────┐
          │                           │
     左侧路线                      右侧路线
    （海边集市）                  （内陆商业区）
          │                           │
          └─────────────┬─────────────┘
                        │
                   50级阶梯
                        │
                    泊岗码头
```

### 2.3 自然景观

**海边风光**
- 波澜壮阔的海面
- 海天交接处景观优美
- 海上天气变化无常

**岛上植被**
- 高大树木，树叶泛黄（秋季）
- 植被稀疏但整洁
- 路边清扫干净

## 三、城镇布局

### 3.1 码头区域

**港口特征**
| 设施 | 描述 |
|------|------|
| 船只类型 | 客货两用大船 |
| 船只尺寸 | 长约60米，宽8米 |
| 运营航线 | 往返泊岗与宁阁码头 |
| 甲板特点 | 堆满货物，临时搭建避雨棚 |

**码头秩序**
- 人头攒动，船运繁忙
- 排队登船、登岛制度
- 存在非官方"登岛费"（地头蛇收费）

### 3.2 道路系统

**人字分叉路设计**

| 路线 | 方向 | 特点 | 功能 |
|------|------|------|------|
| 左侧 | 紧靠海边 | 宽阔热闹 | 大集市所在 |
| 右侧 | 向岛中心 | 较窄蜿蜒 | 生活商业区 |

**右侧内陆路线详情**
- 从码头到旅馆约10分钟步行
- 两侧高大树木，叶片泛黄
- 地面非常干净
- 沿途有饭馆、茶馆等

## 四、重要建筑

### 4.1 海边集市

**位置**：左侧路线，紧靠海边和码头

**特征**
- 主要货物运输枢纽
- 商业中心
- 晚间特别热闹
- 晚上十一点仍人声鼎沸

**氛围**
- 商贩叫卖声
- 讨价还价声
- 人流涌动
- 灯火通明

### 4.2 旅馆系统

| 类型 | 位置 | 价格 | 特点 |
|------|------|------|------|
| 海边旅馆 | 靠近码头 | 较贵 | 位置优越 |
| 内陆旅馆 | 岛内 | 适中 | 安静舒适 |

**设施**
- 休息室
- 清洁住宿空间
- 基本生活便利

### 4.3 黄亮伟宅邸

**主人信息**
| 属性 | 描述 |
|------|------|
| 姓名 | 黄亮伟 |
| 身份 | 船队老板 |
| 年龄 | 约40多岁 |
| 外貌 | 精神矍铄，微微发胖 |
| 经历 | 20多岁迁居，在此奋斗大半辈子 |

**建筑特点**

- 可接待来访旅客

## 十、制作建议

### 10.1 置景要点
- 码头需有真实的木质感
- 阶梯石材要有历史感
- 集市摊位要有生活气息
- 植被要体现秋季特征

### 10.2 特效需求
- 海浪拍岸效果
- 船只摇晃
- 人群流动
- 灯火闪烁

### 10.3 音效设计
- 海浪声（持续背景）
- 船只汽笛（间歇）
- 人群喧哗（集市）
- 叫卖声（远近）

### 10.4 拍摄注意
- 利用自然海光
- 捕捉人流动态
- 体现季节特征
- 展现生活气息
