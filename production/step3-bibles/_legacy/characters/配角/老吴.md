---

## 二、核心定位

**泉水的发现者与见证者** — 超自然元素进入普通世界的桥梁

---

## 三、外貌演变

### 3.1 故事初期（第1章前期）
- **面容**：典型老人面容，皱纹明显
- **眼神**：涣散、疲惫，带焦虑
- **发型**：花白短发，略显稀疏
- **体态**：身体硬朗但行动迟缓
- **肤色**：偏黄，有老人斑

### 3.2 返老还童阶段（第1-2章后期）
- **眼神**：逐渐清澈有神
- **面容**：皱纹减少，皮肤黝黑健康，透出红润
- **体态**：明显强健，肌肉紧绷
- **手臂**：筋肉隆起，布满劳动老茧
- **整体**：仿佛从60岁回春到40岁

### 3.3 故事后期（第45章）
- 记忆力衰退加剧
- 无法跟随年轻人出征
- 只能留守定达

## 四、性格特点

### 积极品质
| 特质 | 表现 |
|------|------|
| 善良热情 | 热情招待访客 |
| 坚毅自信 | 一个月建成房屋 |
| 创意丰富 | 设计泥垛子房屋 |
| 乐于分享 | 向老伴分享秘密 |
| 富有同情 | 救助受困小鹿 |

### 心理特征
- 容易因挫折消沉
- 对变化充满不确定
- 容易被新事物吸引
- 单纯，易信任他人

---

## 五、与泉水的关系

### 5.1 发现经过
- 严冬腊月迷路深山
- 发现清澈小溪
- 沿溪流找到圆形池塘
- 极度口渴时首次饮用

### 5.2 泉水功效
| 变化 | 描述 |
|------|------|
| 思维 | 变得敏捷清晰 |
| 体力 | 身体轻盈有力 |
| 视力 | 恢复年轻状态 |
| 整体 | 年轻20岁 |

### 5.3 认知转变
- 初期：认为是宝物
- 后期：意识到可能是祸源
- 发现小周、白思元变傻后深感自责

## 七、关键场景

### 场景1：户政室挫折（第1章）
- 为孙子办户籍证明
- 记忆力问题连跑三趟
- 遭冷遇后彻底消沉

### 场景2：深山迷路与发现泉水（第1章）
- 在山上游走后迷路
- 陷入冰天雪地
- 发现并饮用泉水
- 身体焕然一新

### 场景3：救助小鹿（第1章）
- 发现被藤蔓缠困的小鹿
- 用石片割断藤蔓
- 用泉水喂养使其恢复
- 与小鹿建立情感连接

### 场景4：建造房屋（第2章）
- 花费一个月完成
- 手臂肌肉紧绷
- 展现身体显著变化

### 场景5：邀请老伴（第2章）
- 展示强健手臂
- 老伴被小屋震撼
- 决定在山上长期居住

### 场景6：发现悲剧（第6章）
- 热情接待各种访客
- 从报纸发现白思元变傻
- 打听得知小周也疯了
- 开始怀疑泉水

## 十二、象征意义

| 象征 | 含义 |
|------|------|
| **泉水发现者** | 超自然进入现实的桥梁 |
| **返老还童** | 泉水神奇功效的证明 |
| **无意帮凶** | 善良也可能带来祸端 |
| **最终衰退** | 自然规律不可抗拒 |

老吴代表了**善良纯真但容易被欺骗的老年群体**，他的故事既是对返老还童的浪漫想象，也是对人性善恶相伴的深刻反思。
