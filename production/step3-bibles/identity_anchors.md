## 一、矩阵说明 | Matrix Overview

本矩阵通过6个维度对53个角色进行系统化差异化设计，确保：
- 每个角色在至少3个维度上与其他角色不同
- 每个角色都有唯一的视觉标记（Signature Mark）
- 身份锚点简洁有力，便于AI绘图提示词生成

This matrix systematically differentiates 53 characters across 6 dimensions:
- Each character differs from all others in at least 3 dimensions
- Each character has a unique visual signature mark
- Identity anchors are concise and impactful for AI prompt generation

### 6个维度说明：

1. **Age Bracket (年龄段)**：teen(15-20), young(20-30), prime(30-45), mature(45-60), elder(60+), supernatural(ageless)
2. **Build (体型)**：gaunt, slim, lean-athletic, average, stocky, heavyset, massive
3. **Face Shape (脸型)**：oval, round, square, long-narrow, diamond, angular-bony
4. **Hair (头发)**：bald/shaved, buzz-cut, short-neat, medium, long-loose, long-tied/bun, white/grey, distinctive
5. **Skin Tone (肤色)**：pale/fair, light, medium-tan, bronze/dark, weathered-rough, smooth-youthful
6. **Signature Mark (独特标记)**：MUST be UNIQUE to each character

```
[IDENTITY ANCHOR - 身份锚点]

[ORIGINAL PROMPT]
{original_ai_prompt}
```

## 七、特别说明 | Special Notes

### 群像角色 (Ensemble Characters) 的处理

对于需要多个具体个体的群像（如"五名古代恶灵"、"旅行团群像"、"泊岗镇1200居民"），建议：

For ensemble characters requiring multiple distinct individuals (like "Five Ancient Spirits", "Tourism Group", "1200 Bogang Residents"):

1. **五名古代恶灵**: 使用基础锚点后，在5个幽灵间添加微妙变化（眼光色泽、半透明度、胸口光晕颜色等）
2. **旅行团群像**: 为每个游客生成独立的容貌（在"20-65岁都市人"基础上加上职业/配饰区分）
3. **泊岗镇1200居民**: 创建3-5个代表性村民肖像，再通过batch processing复制变化

### 超自然角色 (Supernatural Characters) 的特殊处理

对于灵魂、幽灵、神兽等超自然角色，在提示词中明确加入形态特征：

- **灵魂/幽灵**: "ethereal translucent form", "glowing spectral aura", "otherworldly presence"
- **神兽**: "divine beast form", "mystical aura", "celestial bearing"
- **附身者**: "possessed human body", "eyes vacant of original spirit", "unnatural stillness"
