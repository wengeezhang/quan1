## 三、空间结构与地理

### 俯瞰布局

```
    北侧永安镇方向
            │
    ┌───────┴────────┐
    │ 东岸（清平侧） │
  河岸  ┌──────────┐  河岸
  陡峭  │ 河水急流 │  灌木
    │  │(30-50m宽) │  │
    │  └─────┬─────┘  │
    └────────┼────────┘
           西岸
        (永安侧)

    桥梁遗迹处（在河中）
    │残破木桩与梁柱
```

### 纵深与层次

- **前景**：河岸乱石与灌木丛、渔船停靠点、宋小仙隐藏小舟之处
- **中景**：河水涌动、浪花飞溅、桥梁残骸（断木与石桩）
- **远景**：对岸陡峭河岸、远处树林轮廓

### 关键地点

**河水急流区**：宽30-50米，河流在摧毁桥梁后变得更加湍急。夜间水流声轰鸣，能见度极低，救人时的核心场景。

**桥梁遗址**：两座原有的石木混合结构桥梁，被苏伟力的部队从中部破坏摧毁。现存断裂的木梁悬在河中、石桩部分沉水，形成危险的障碍物。

**东岸（清平镇侧）**：河岸较陡，有乱石与灌木，难以攀爬。是宋小仙登陆与观察对岸的位置。

**西岸（永安镇侧）**：河岸同样陡峭，树林茂密，是撤离队伍尝试渡河的出发点。

### 出入口与通道

1. **北侧河道**——连接内陆，战略上最重要的陆路通道
2. **东西两岸的浅滩区**（战前）——原本可涉水的地点，现已不可用
3. **夜间渡河路线**（紧急）——宋小仙借助小舟的冒险路线

### 标志物/视觉锚点

- **破碎的桥梁木梁**——插在河水中的断裂木柱与残骸，象征战争对交通线的摧毁
- **湍急的河水与浪花**——视觉上强调水流的凶险与隔阻性
- **两岸陡峭河岸线**——限制出入，突出地理障碍

## 四、光影与氛围

### 基础光影方案

- **主光源**：日间自然日光；宋小仙渡河场景多为夜间，仅月光微弱照亮
- **色温范围**：日间 5200K-5800K；夜间 4000K（月光冷蓝）；水面反光混杂
- **辅助光源**：月光、水面反光、偶尔火把
- **阴影特征**：日间河岸投出硬阴影；夜间极少清晰阴影，仅月光下微弱身形
- **对比度**：日间中等高；夜间极低对比度（几乎全黑）

### 色彩基调

| 色名 | HEX | 用途 |
|------|-----|------|
| 深灰蓝 | #4A5A7A | 河水的基础色，含泥沙感 |
| 墨绿 | #1A3A2A | 河岸灌木与树林 |
| 冷月白 | #B8C8D8 | 夜间月光与水面反光 |

### 氛围参数

- **空气质感**：河风强劲、潮湿；夜间雾气升腾于河面
- **粒子**：河水飞溅的水雾、河雾升腾（尤其夜间）
- **整体氛围**：白天的战争阻隔感 → 夜间的绝望与凶险 → 生死救援的紧张

- **环境底噪**：河水涌动与撞击岩石的轰鸣声，白天清晰，夜间更显渗透

- **触觉/温度/嗅觉**：河水冰冷刺骨（战时季节）、泥沙与腐朽植物的腥臭、河风凛冽

## 六、场景演变轨迹

望海河在 ch38-39 经历了桥梁被摧毁的重大变化。战前有两座桥梁连接两岸，战时被苏伟力部队破坏，切断了陆路。河流本身无结构改变，但交通功能被彻底摧毁。夜间宋小仙与熊大宝冒激流渡河是该场景最凶险的时刻，战后的河流保持桥梁残骸的状态，成为永安镇撤离的最大地理障碍。

---

## 七、镜头预案

### 标志性构图

- **河流纵深**：从河岸向下俯拍，湍急的河水向远方延伸，桥梁残骸点缀其中，强调河流的宽度与凶险
- **夜间渡河**：低机位从水面拍摄，月光下两个身影在激流中挣扎，浪花飞溅遮挡视线，突出救援的绝望感
- **俯视破桥**：从高处俯瞰桥梁残骸在河中的位置，木梁与石桩的断裂状态清晰，象征战争的破坏

### 空间限制与注意事项

- 河宽30-50米，大景别难以同时容纳两岸细节，多用局部河段
- 夜间场景可见度极低，月光下物体多呈剪影状，需要强烈光影对比
- 水流速度与音效关键，建议结合音景设计突出河流的凶险
