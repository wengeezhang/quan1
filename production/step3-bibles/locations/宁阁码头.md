## 三、空间结构与地理

### 俯瞰布局

```
        陆地
        ────
        候客室
           ↓
    ┌──────────────┐
    │  木栈道      │
    │   ↓↑         │
    │ 船舶停靠     │
    │（客货船）    │
    └──────────────┘
           ↓
        海面
```

### 纵深与层次

- **前景**：候客室、行李堆积、购票窗口
- **中景**：木栈道、船只缆绳、登船处
- **远景**：海平面、远处堤岸、天际线

### 关键地点

**候客室**：简陋木制建筑，售票窗口、长椅、信息板。

**木栈道**：风化的木板铺设，延伸至船舶，宽约3米，晃动感明显。

**客货船**：中等规模运输船，木制船身，舱室与甲板。

### 出入口与通道

1. **陆地入口**——来自镇中心的小路
2. **登船梯**——连接栈道与船舱的木梯
3. **船舱内部**——通往船舱与厨房

### 标志物/视觉锚点

- **候客室的木招牌**——标识"宁阁码头"
- **粗绳系船缆**——缠绕在木桩上的陈旧绳索
- **船舶的木制船身与帆布**——泛黄的布料与船漆剥落

## 七、镜头预案

### 标志性构图

- **推进栈道**：从陆地推进，逐渐靠近船舶，凸显出发的决心
- **侧拍登船**：人物登船梯，背景是船舶与海平线，标志启航
- **俯视海面**：从船舶甲板俯拍海面，海浪与船影形成动感

### 空间限制与注意事项

- 码头空间开放，午间逆光需要精确控制
- 海面反光强烈，容易造成测光困难
- 木栈道摇晃会影响画面稳定性，需要稳定架设
