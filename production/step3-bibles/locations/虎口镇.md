## 三、空间结构与地理

### 俯瞰布局

```
         北部入口
            │
    ┌───────┴────────┐
    │  北部民居聚落  │
    │   (防守位置)   │
    └────────┬────────┘
             │
         镇中心广场
        (露天市集)
             │
    ┌────────┴────────┐
    │  南部码头区域   │
    │  (逃离方向)     │
    └────────┬────────┘
             │
         南部出口
      (通往海岸)
```

### 纵深与层次

- **前景**：狭窄石板街道、商铺棚顶、市集摊位残留物
- **中景**：民居白墙、防守据点的废弃营垒
- **远景**：南侧海平线、码头建筑轮廓线

### 关键地点

**镇中心广场**：露天市集聚集地，平时贸易繁忙，战时成为心理战的舞台——朱围庸势力的宣传与劝降在此进行。

**北部防守阵地**：镇民试图防守的首要位置，部署了简陋木栅栏与火把堡垒，但在心理战面前迅速瓦解。

**南部码头**：渔船停靠处，镇民仓皇逃亡的出发点，通往海岸与外界的最后通道。

### 出入口与通道

1. **北部城门**——陆路出入口，朱围庸军队的进攻方向
2. **南部码头**——水路逃亡通道，镇民撤离的急迫出口
3. **海岸线**——背向大海，镇民逃脱的最后防线

### 标志物/视觉锚点

- **白墙灰瓦的沿海民居**——小镇的身份标识
- **市集广场的棚顶群**——镇中心聚集的视觉中心
- **南部码头的渔船与绳索**——逃亡的象征

## 四、光影与氛围

### 基础光影方案

- **主光源**：自然日光，侧光为主，符合全局 Art Direction 默认光影
- **色温范围**：日间 5200K-5800K（自然日光）
- **辅助光源**：市集下的散射光、火把（防守时期）
- **阴影特征**：中等硬度，方向明确；建筑投影清晰
- **对比度**：中等偏高，符合全局基线

### 色彩基调

- **空气质感**：沿海湿润，海风带盐分，空气通透度适中
- **粒子**：市集有轻微灰尘，码头有盐分水雾
- **整体氛围**：繁华 → 恐慌 → 冷清的心理战演变

## 六、场景演变轨迹

虎口镇在 ch36 经历了朱围庸心理战的打击。镇民防守阵地在宣传与劝降面前迅速瓦解，无大规模战斗，无建筑破坏，仅是人心的溃散。物理面貌保持完好，但氛围从繁华市井转为空城恐惧——这是一个"胜于无声"的占领。

---

## 七、镜头预案

### 标志性构图

- **空城远眺**：从北部城门向镇中心俯视，空荡荡的街道与弃置的市集棚顶，强调人口流失
- **码头逃亡**：从码头望向镇子，渔船争相起航，镇建筑在后景显得孤独无援
- **市集广场的寂寥**：昔日繁忙的交易场所变为鬼城，棚顶影子无人撑举

### 空间限制与注意事项

- 镇街较窄（3-4米），大景别需从高处俯视或街道两端
- 南部码头水域广阔，适合展现镇的地理困境（被海包围）
- 火把防守的视觉不足以形成宏伟防线，宜强调心理战的无声压迫
