## 三、空间结构与地理

### 俯瞰布局

```
         山顶平地（稀疏林木）
              △△△
             /   \
            /刀片道\ (宽50m,最窄5m)
           /   ↓   \
    海岸  /─────────\  内陆
   (永安镇)          (牛台镇)
     ◎                ◎
      \             /
       \  深山隐谷 /
        \(神兽栖地)/
         ╲       ╱
          ╲─────╱

东南侧环抱海岸 — 凤廊山脉 — 西北侧通往内陆
```

### 纵深与层次

- **前景**：刀片道两侧陡峭百米山峰，密集树木与灌木丛林
- **中景**：山腰积雪覆盖，树木密度逐层递减
- **远景**：山顶平地与更远处的起伏山脉轮廓，天际线模糊于雪雾之中

### 关键地点

**刀片道**：凤廊山脉唯一的通行狭道，平均宽50米，最窄处仅5米。两侧是陡峭的百米山峰，树木丛林密集。道路呈之字形蜿蜒上升，坡度陡峭。是永安镇撤离与敌军追击的必经通道。

**山顶平地**：位于刀片道尽头，地形稍平坦，树林较为稀疏。从此处沿着宽阔的山沟向下行约10里，可抵达牛台镇。此处常被积雪覆盖，是行人的中继点。

**白巨隐谷**：深山腹地的隐蔽凹地，约足球场大小。山巅冰封环境下，该处因地热而呈现热气腾腾的景象，四周被高峰环抱。内有三根巨木支撑的大锅（用于炖煮食物）和高约10米的草棚建筑，是神兽三代的栖息地。

**山巅冰原**：海拔最高处，常年积雪，坚硬如冻土。朱围庸初次穿越时艰难行进的区域，寒风刺骨，干粮稀缺。

### 出入口与通道

1. **刀片道入口**——位于凤廊山脉东南侧，永安镇西北出发点，通往山顶平地
2. **刀片道出口**——通往山顶平地，续接向内陆下山的宽阔山沟
3. **深山隐谷出口**——神兽隐谷通向山脉外的密林通道，朱围庸逃脱路线

### 标志物/视觉锚点

- **刀片道的险峻双峰**——山脉的地理标识，远眺可见两侧高耸的百米山壁
- **隐谷的热气烟雾**——在冰封山巅中的异象，标志神兽居所的存在
- **山顶稀疏林木**——与下层密林的视觉对比，表明即将下山进入内陆

## 四、光影与氛围

### 基础光影方案

- **主光源**：自然日光，受厚重云层与雾霾衰减，光线昏沉；顶光为主（光源来自天顶，阴影投向脚下）
- **色温范围**：日间 4500K-5200K（冷调自然光），阴雪天 3800K-4500K（更冷的蓝灰光）
- **辅助光源**：篝火（行进中的临时营火，色温 2800K-3200K），阳光在积雪上的反射光
- **阴影特征**：极软的漫射阴影（雪地反射减弱阴影硬度），雪坡上的阴影呈冷蓝色
- **对比度**：中等，暗部保留 5-10% 细节，积雪亮部不过曝

### 色彩基调

| 色名 | HEX | 用途 |
|------|-----|------|
| 暮蓝 | #3A4A6B | 山脉主色——高海拔冷调的基底色，代表险恶与压抑 |
| 焦褐 | #2C1810 | 露岩、树干、深层阴影的基底色 |
| 冷灰白 | #D3D3D3 | 积雪主色，含微蓝偏冷的白色，绝不使用纯白 |
| 苔绿 | #4A6741 | 密林区域的植被色 |

### 氛围参数

- **空气质感**：极度清冷稀薄的高山空气；雪雾混合产生能见度降低的朦胧感
- **粒子**：飘雪、冰晶飘浮、篝火烟雾（永安镇撤离时连续生火）
- **整体氛围**：苍凉、险峻、原始荒蛮，透出生存的绝望与人类渺小感

## 六、场景演变轨迹

凤廊山脉在 ch9 至 ch67 的全部出场中，地形结构与地理特征保持不变。山脉未遭受战争破坏，刀片道与隐谷的物理面貌始终完好。

不同章节中的气候变化（晴日、积雪、风暴、雾霾）与人物活动引发的临时现象（篝火烟雾、人员往来产生的踏迹、雪崩突发）是叙事层面的变化，通过分镜阶段在具体镜头的 prompt 中处理（如 ch39 永安镇大队伍通过时的繁忙迹象、ch47 朱围庸部队通过时的行军痕迹），不在场景圣经层面拆分阶段。

## 七、镜头预案

### 标志性构图

- **俯拍刀片道**：从山顶俯瞰蜿蜒的刀片道，两侧百米高峰形成深邃峡谷，前景与远景的压迫感突出空间的狭隘与险要
- **隐谷的热气奇景**：在冰封山巅的寒白背景中，前景是腾腾热雾与巨大草棚的剪影，远景是覆雪的山脉轮廓——超现实的视觉对比
- **撤离队伍的长镜头**：低机位或平镜跟随拉长的人队穿过刀片道，队伍渺小在山峡间，强调人类面对自然的脆弱

### 空间限制与注意事项

- 刀片道极窄（最窄5米），横景拍摄机位受限，纵深构图更为可行
- 隐谷四周被山峰包围，俯瞰与仰拍都易产生压迫感，适合表达困境与绝望
- 山顶雪原缺乏遮挡，不宜用极近景（容易显空洞），中远景更能展现空间感
- 大规模人员通过时不宜频繁切镜头（容易破坏行军的连贯感），长镜头推进为佳
- AI 生成需避免"笔直通畅的山路"——刀片道应呈现不规则蜿蜒、陡峭阶梯感
