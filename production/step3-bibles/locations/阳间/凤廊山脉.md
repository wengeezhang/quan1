## 二、地理概述

### 2.1 山脉整体特征

**规模与走向**
- 东北至西南走向
- 绵延约50公里
- 覆盖七八十里的地区范围
- 与牛台山形成半月弯曲状

**地理功能**
- 将牛台镇和沿海城镇隔开的天然屏障
- 形成自然地理分界线
- 连接山地与平原地带的过渡区
- 人间与阴间的自然分隔

### 2.2 牛台山详细参数

| 参数 | 数值 |
|------|------|
| **海拔高度** | 310米 |
| **走向** | 东北至西南 |
| **形态** | 与凤廊山脉呈半月弯曲状 |
| **地下特征** | 200米深处为恶灵控制中心 |

### 2.3 地层结构

```
地表 ─────────────────────────── 0米
```

## 三、自然环境

### 3.1 植被系统

**主要树种**
| 树种 | 特征 | 分布 |
|------|------|------|
| 樟树 | 常青，主要遮挡 | 山腰至山顶 |
| 榕树 | 根系发达，生命力象征 | 溪流两岸 |
| 马尾松 | 挺拔高大 | 山脊 |
| 杉树 | 笔直，四季常青 | 山坡 |
| 落叶树 | 冬季裸露 | 混杂分布 |

**地被植物**
- 厚厚的青苔层覆盖岩石
- 蕨类植物（半枯萎状）
- 野草与灌木（稀疏分布）
- 枯树枝、落叶堆积（冬季）

**整体风貌**
- 郁郁葱葱但显得遥远
- 包围感强，与世隔绝
- 季节变化明显

### 3.2 水文特征

**溪流系统**
- 源自山上，向下流淌
- 流经牛台山各处
- 清澈见底，水质纯净

**泉水池塘（核心水源）**
| 特征 | 描述 |
|------|------|
| 位置 | 牛台山山腰，靠近美女榕 |
| 形态 | 圆形，直径2-3米，深0.5米 |
| 水质 | 极度清澈，可见底部鹅卵石 |
| 特性 | 微微发光，具有神奇功效 |

**花溪**
- 宽约1米，水流湍急
- 溪床大石经年冲刷呈黑绿色
- 两岸野花小草遍布
- 高大榕树林立

### 3.3 气候特征

**季节变化**

| 季节 | 特征 |
|------|------|
| **冬季** | 严寒刺骨，降雪，北风裹雪 |
| **春季** | 新绿萌发，雾气缭绕 |
| **夏季** | 郁郁葱葱，凉爽宜人 |
| **秋季** | 落叶飘零，色彩斑斓 |

**气候体验**
- 山区空气湿润
- 地面常年潮湿（青苔覆盖）
- 晨间清新，雾气带露水
- 冬季极度寒冷

## 四、核心地标

### 4.1 美女榕

**位置**：牛台山半山腰，溪流上方

**外观特征**
- 树干粗大，需两人合抱
- 树根裸露盘旋交织
- 形似女性人形
- 两条细"腿"深入溪水

**神奇性质**
- 与恶灵区有特殊联系
- 可作为灵魂附着的载体
- 朱围庸逃脱后的栖息地
- 地下直通恶灵控制中心

### 4.2 泉水池塘区域

**周边环境**
- 半径10-15米的开放平地
- 周围密集树木形成包围
- 远处山林层次递进

**标志性巨石**
- 高约2米的大灰岩
- 位于池塘东侧
- 表面风化纹理和苔藓
- 一侧有凹陷（被磨圆）
- 下方堆积枯树枝

**水流走向**
- 北侧山谷 → 池塘中心 → 南侧溪流

### 4.3 老吴的小院

**位置**：牛台山半山腰

**建筑结构**
- 三间宽房子（客厅+两卧室）
- 独立厨房
- 竹子围成的院子

**建筑特点**
- 黄泥垛子堆砌（冬暖夏凉）
- 松木大梁
- 茅草覆盖屋面
- 隐蔽不易被发现

## 五、氛围与感官体验

### 5.1 光线特征

**白天场景**
- 自然光透过树叶缝隙洒落
- 明暗交错的斑驳效果
- 阳光温暖但不炽热
- 色温：清晨偏冷（5500K），午间中性（6500K）

**夜间场景**
- 月光明亮（冬季月圆时）
- 池面银白色反射
- 灰蓝色阴影区域
- 梦幻而诡异的冷色调
- 泉水微微发光

### 5.2 色彩基调

**主色系**
| 颜色 | 代表元素 | 象征 |
|------|----------|------|
| 翠绿 | 树木植被 | 生机 |
| 银白 | 泉水月光 | 纯净神圣 |
| 灰褐 | 岩石树枝 | 沧桑 |

**辅色系**
- 淡青色：冬季树影与晨雾
- 金黄色：阳光透过叶隙
- 深蓝色：夜间天空与远山

### 5.3 声音环境

**主要环境音**
- 潺潺流水声：持续不绝
- 风吹树叶声：沙沙作响
- 偶尔的鸟鸣：远处传来
- 虫鸣：夏夜此起彼伏

**特殊声效**
- 冬夜寒风呜咽
- 雪花飘落沙沙声
- 冰冻树枝咔嚓声
- 灵魂显现时的低鸣共鸣

### 5.4 气味特征
- 湿润泥土与青苔清香
- 冬季干枯落叶陈旧味
- 清爽矿物质泉水气味
- 松林淡淡香气
- 雨后泥土青草味

## 十、制作建议

### 10.1 置景要点
- 使用真实或高仿植被
- 水质处理保持清澈透明
- 岩石表面需有青苔纹理
- 营造包围感的树木布局

### 10.2 特效需求
- 持续流水声与视觉
- 光束穿透树叶效果
- 雾气/薄雾效果
- 泉水微光渲染

### 10.3 天气特效
- 飘雪机制
- 风向控制
- 积雪真实感
- 北风音效

### 10.4 拍摄注意
- 广角展现环境规模
- 中焦捕捉细节纹理
- 长焦压缩山脉层次
- 自然光优先
