# 定达国都 | 场景圣经

## 一、基本信息

**场景名称**：定达国都
**所属区域**：国家首都
**首次出现**：第44章
**末次出现**：第70章
**出现章数**：10+章
**重要性**：★★★★★

**核心定义**：国家权力与末日的最后堡垒。从象征权力的黄金宫殿，演变为围困中的绝望之城。

**空间层级**：
- 权力中枢（宫殿、行政中心）
- 特权阶层（贵族区、精英学校）
- 隐秘设施（秘密实验室）
- 防御体系（城墙防线、出入口）
- 普通民众（平民区、商业区）

**色彩体系**：
- 权力期：暗金 #DAA520 + 暖白 #FFF8DC
- 腐败期：暗金逐渐褪色，阴影加深
- 围城期：暗紫 #5B3A6B + 暗红 #8B1A1A + 烈焰橙
- 禁用色：#FFFFFF（纯白破坏场景品质）

## 二、阶段注册表

### 第一阶段：权力期（44-50章）
**时间跨度**：初入国都
**主要特征**：
- 权力结构清晰，等级森严
- 黄金建筑沐浴在正午光线下
- 官僚体系正常运作，表面繁荣
- 主角初次接触权力阶层
- 贵族学校代表"精英教育"幻象

**关键场景**：宫殿谒见、贵族学校、行政区游历
**光影特征**：直射的金色光线，清晰的阴影边界
**氛围关键词**：庄严、虚伪、等级、繁荣假象

### 第二阶段：腐败期（51-59章）
**时间跨度**：权力运作的真相暴露
**主要特征**：
- 权力结构开始扭曲，暗流涌动
- 秘密实验室活动加强
- 贪污、压制、非人道研究的迹象
- 城市底层开始出现不安情绪
- 官方叙事与真实世界的割裂

**关键场景**：秘密设施探访、腐败官员活动、贵族区的暴力维持秩序
**光影特征**：金色逐渐被阴影侵蚀，走廊与地下设施的昏暗对比
**氛围关键词**：压抑、虚伪崩塌、恐惧、无力

### 第三阶段：围城期（60-70章）
**时间跨度**：末日降临
**主要特征**：
- 城市被全面包围
- 权力结构彻底瓦解
- 防御体系激活，城市变为战场
- 社会阶级壁垒破碎
- 城市沦为孤岛，生存成为唯一主题

**关键场景**：城墙防线、沦陷的行政中心、难民区、最后的庇护所
**光影特征**：火光与烟雾，暗紫色的夜空，暗红的火焰反射
**氛围关键词**：绝望、崩塌、末日、挣扎

## 三、空间结构与地理

### 宫殿与行政中心
**位置**：国都中心，地势最高
**建筑特征**：
- 黄金穹顶在权力期闪闪发光
- 多层级的内部空间：公开大厅、私密议事厅、地下密室
- 围绕宫殿的护卫防线逐渐加强
- 围城期成为最后的抵抗据点

**功能变化**：权力象征→腐败中心→末日堡垒

### 贵族区
**位置**：宫殿周围环绕
**建筑特征**：
- 精致的宅邸，私家花园（权力期保持整洁）
- 贵族学校代表特权教育系统
- 私密的聚集地点
- 围城期率先沦陷或成为难民集中地

**社会功能**：特权维持、信息流通、权力演变的观察点

### 秘密实验室
**位置**：隐藏在地下或偏僻建筑内
**特征**：
- 与官方权力结构有着微妙的关系
- 进行着"进步"名义下的非人道实验
- 权力期较为隐蔽，腐败期活动加强
- 围城期可能被发现或摧毁
- 冷色灯光、冰冷的仪器、痛苦的痕迹

**意象**：权力与科学扭曲结合的象征

### 城墙防线
**位置**：城市周边
**特征**：
- 权力期：象征城市的边界与安全
- 腐败期：开始强化，显示权力的不安
- 围城期：成为最后的防线，逐渐被突破
- 观察哨塔、储备物资、伤兵营地

**功能演变**：边界→警惕→绝望的防线

### 平民区
**位置**：城市外围
**特征**：
- 权力期：被权力中心遗忘，贫困与不平等
- 腐败期：镇压加强，不安情绪蔓延
- 围城期：难民涌入，成为最混乱的区域
- 市集、居住区、小型宗教场所

**社会功能**：展现权力黑暗面的镜像

## 四、光影与氛围

### 权力期的光景
**主光源**：正午的烈日，照亮黄金穹顶
**光质**：直射、清晰、热烈
**色温**：暖色，金色占主导（#DAA520）
**阴影**：清晰、有力度，强化建筑的宏伟感
**细节**：
- 宫殿走廊中的金色反射
- 贵族区的私家园林沐浴在柔和阳光下
- 城墙上的卫兵在强烈阳光下显得坚定

**氛围感**：秩序、确定性、虚伪的安全感

### 腐败期的光景
**主光源**：逐渐加强的人工光（灯火、火把）
**光质**：混合，部分区域变暗
**色温**：金色褪去，暖白（#FFF8DC）变得苍白、不自然
**阴影**：加深，不规则，隐藏秘密
**细节**：
- 秘密实验室的冷白灯光形成对比
- 官邸中的烛光摇晃，显得不稳定
- 地下通道的昏暗，强化压抑感
- 夜间的城市开始显露不安的光芒

**氛围感**：压抑、虚伪、隐藏与暴露的冲突

### 围城期的光景
**主光源**：火光、爆炸、烟火
**光质**：混乱、闪烁、不稳定
**色温**：暗紫（#5B3A6B）夜空为背景，暗红（#8B1A1A）火焰为前景
**阴影**：被火光取代，轮廓模糊不清
**细节**：
- 城墙上的火焰与烟雾弥漫
- 坍塌的建筑在火光中显得更加绝望
- 难民营地的零散篝火
- 夜幕下，城市变成火海与暗影的混合体

**氛围感**：末日、绝望、混乱、挣扎

### 光影过渡策略
- **渐进式加强阴影**：从清晰→复杂→混乱
- **色彩温度下降**：从温暖金色→冷白→冷紫冷红
- **光源变化**：从自然光→人工光→火光
- **对比度变化**：从清晰→模糊→混乱

## 五、声景与感官

### 权力期的声景
**主导声音**：
- 宫殿中的正式宣告，回音显得庄严
- 贵族学校的规律钟声
- 行政中心的文书沙沙声、笔触纸张的细微音
- 城市广场的商业喧嚣（经管制的）
- 卫兵的整齐脚步声

**细节音**：
- 黄金器物碰撞的清脆声
- 高跟鞋在大理石地面上的敲击
- 水喷泉的潺潺声（贵族区）
- 城门沉重的关闭声

**感官体验**：秩序感、控制感、隐约的紧张

### 腐败期的声景
**主导声音**：
- 秘密对话的低沉耳语
- 机器的嗡鸣与嘶鸣（实验室）
- 镇压行动的粗暴声响
- 谣言的传播声（不同声调、重复变形）
- 门被重重关闭的声音

**细节音**：
- 脚步声变得更加急促或沉重
- 破碎声（秩序开始崩坏）
- 警钟的鸣响
- 压低的、充满恐惧的交谈声

**感官体验**：不安、压抑、隐藏的暴力

### 围城期的声景
**主导声音**：
- 爆炸与枪火声（持续、沉闷）
- 建筑坍塌的轰鸣
- 难民的哭喊、尖叫
- 城墙防线上的喊杀声
- 火焰的噼啪声

**细节音**：
- 破碎玻璃的尖锐声
- 余震引起的物体滚动声
- 人群的混乱噪音
- 沉默（死亡、绝望的瞬间）

**感官体验**：末日、混乱、生死危机

### 其他感官
**嗅觉**：
- 权力期：香氛、贵族区的花香、书籍的纸张味
- 腐败期：消毒水与腐烂气味混合（实验室附近）、烟雾、不安的汗味
- 围城期：烟火、血腥、死亡的腐臭、灰烬

**触觉**：
- 权力期：精细的织物、冰冷的大理石、柔软的绒毯
- 腐败期：凝聚的冷汗、被束缚感、粗糙的压力
- 围城期：尖锐的碎片、灼热的火焰、冰冷的恐惧

## 六、场景演变轨迹

### 城市功能的演变

**权力期**（章节44-50）
```
功能完整 → 权力中心高效运作
- 行政系统正常
- 供给链完整
- 社会等级明确
- 对外部威胁的否认
```

**腐败期**（章节51-59）
```
功能扭曲 → 权力结构腐烂
- 行政系统开始失效
- 资源被秘密工程挪用
- 社会不安加剧
- 外部威胁逼近
```

**围城期**（章节60-70）
```
功能瘫痪 → 末日生存
- 行政系统彻底瘫痪
- 资源枯竭
- 社会等级破裂
- 外部威胁变为现实包围
```

### 人口流动的演变
- **权力期**：贵族在中心，平民在外围，秩序稳定
- **腐败期**：平民开始逃离，城市中部出现难民涌入迹象
- **围城期**：大量难民涌入中心区域寻求保护，贵族区沦为难民营

### 城市面貌的演变
- **权力期**：干净、整洁、精心维护的城市秩序
- **腐败期**：部分区域开始显露破败，维护工作明显减弱
- **围城期**：建筑被轰炸、烧毁，街道变成战场与废墟

### 权力象征的演变
- **权力期**：黄金穹顶闪闪发光，代表稳定与永恒
- **腐败期**：黄金开始暗淡，穹顶上出现裂缝与污迹
- **围城期**：黄金穹顶被摧毁或被火光照亮，变成末日的背景

## 七、镜头预案

### 权力期的典型镜头

**镜头1：宫殿谒见**
- 视角：从下而上仰拍，强化宫殿的高大
- 光线：金色阳光从穹顶洒下，形成神圣的光柱
- 人物：身着正式服装，表情严肃，等级明确
- 时间：正午或傍晚（权力的黄金时刻）

**镜头2：贵族学校走廊**
- 视角：跟随镜头，展现建筑的精细与秩序
- 光线：柔和的人工光线，强调精致
- 人物：穿着统一，姿态得体，言行规范
- 细节：精美的壁画、整洁的地面、精确的几何结构

**镜头3：城墙巡视**
- 视角：广角，显示城市的全貌与防御体系
- 光线：清晰的日光，强调防线的坚固
- 人物：卫兵在清晨或傍晚时分的影子显得坚定
- 细节：整齐的防线、清晰的视野、有序的巡逻

### 腐败期的典型镜头

**镜头4：秘密实验室探访**
- 视角：逼仄的角度，强调压抑感
- 光线：冷白色的灯光，产生不自然的苍白
- 人物：压低身体，谨慎移动，恐惧的面部表情
- 细节：复杂的机器、受害者的痕迹、隐蔽的设施

**镜头5：腐败官员的活动**
- 视角：从暗处或偏角观察，制造偷窥感
- 光线：混合的烛光与人工光，产生不稳定感
- 人物：表情虚伪，动作隐蔽，言语充满密码
- 细节：贵重物品的交接、密室的多重锁、证据的销毁

**镜头6：社会不安的迹象**
- 视角：街道平视，捕捉普通民众的不安
- 光线：傍晚或黄昏，光线变暗，投射长影
- 人物：聚集的人群、低沉的交谈、警觉的眼神
- 细节：被损毁的公共设施、被加固的大门、宣传的撕裂

### 围城期的典型镜头

**镜头7：城墙防线被攻击**
- 视角：广角或甚至俯拍，显示防线的脆弱
- 光线：爆炸的闪光、火焰的照亮、烟雾的遮挡
- 人物：卫兵的混乱、伤兵的哀嚎、难民的逃散
- 特效：爆炸、烟火、建筑的坍塌

**镜头8：城市的沦陷**
- 视角：从高处俯瞰，显示城市全貌的变化
- 光线：火光与烟雾主导，逐渐吞没城市的其他光源
- 人物：人群的逃散、组织的混乱
- 细节：建筑的燃烧、街道被占据、秩序的彻底瘫痪

**镜头9：难民营地的绝望**
- 视角：近景与中景结合，显示个人的绝望与宏观的混乱
- 光线：零散的篝火、月光、残留的建筑反射
- 人物：疲惫的脸孔、无望的眼神、相互依靠的身体
- 细节：简陋的庇护所、稀缺的食物、传播的疾病迹象

**镜头10：最后的堡垒**
- 视角：低角度，强化最后抵抗的决绝
- 光线：火光与暗紫色夜空的对比
- 人物：疲惫但坚定的守卫者、受伤的幸存者
- 细节：加强的防线、紧张的气氛、绝望中的希望
