## 三、空间结构与地理

### 俯瞰布局

```
       入口走廊 (地下楼梯)
            │
     ┌──────┴──────┐
     │              │
  控制室       扫描区
  (办公       (灵魂
   监控)      注入)
     │              │
     └──────┬──────┘
            │
        培养室群
     (5-6间小室)
         │  │
         └──┘
```

### 纵深与层次

- **前景**：不锈钢走廊、白色门框、触屏面板
- **中景**：扫描仓库与培养液槽、密集的医疗级仪器、线路纵横
- **远景**：实验室尽端的厚重隔离门、冷冻储存舱

### 关键地点

**控制室**：不超过40平方米，配备多块显示屏实时监测培养进度，办公桌、计算机、样本冷藏柜，空气循环系统的主控制面板和警报装置。吴启思的个人科研阵地，极其整洁，每个仪器都标记精确的检测参数。

**扫描舱**：约2m×1.5m×2m的金属密闭舱，内部排满电极和光学传感器，舱顶有旋转扫描臂，底部有灵魂容器的对接口。此处进行"灵魂注入"——将液态灵魂通过管道泵入目标数字人肉体。整个过程由控制室远程操控，扫描舱内部从暗蓝色待命状态切换为白光扫描状态（强刺激感）。

**培养室**：5-6间小室，每间约20平方米，玻璃隔断透视内部。培养槽呈横排排列，每槽盛有微绿色的培养液，槽内浸泡着发育中的数字人肉体（人形轮廓，肤色苍白），槽顶有营养液滴管、温度传感器、脑脊液循环管。液面下1米处设置细微的气泡供氧。培养液散发的绿光是该区域唯一的光源。

**储存与隔离区**：实验室深端的密封舱，存放完成品（已植入灵魂的数字人）和备用脑脊液容器。隔离舱门厚达15cm，内部维持超低温以保持生物活性。

### 出入口与通道

1. **地下楼梯入口**——从AA大学地下层进入实验室的唯一通道，隐蔽于建筑基础深处，配备刷卡门禁
2. **应急逃生口**——实验室最深处的密闭逃生通道，通往地下停车场（第71章被启用）
3. **物资接收口**——控制室侧面的真空递送舱，用于无人接触地接收培养液、试剂、肉体样本

### 标志物/视觉锚点

- **扫描舱的旋转扫描臂**——启动时发出白光和轻微的机械声，是"注入开始"的视觉标识
- **培养液的绿色微光**——在漆黑的走廊中如鬼火般诡异，标记实验进行中
- **控制室的多屏显示墙**——呈现数字人成长进度的图表和生命体征曲线，象征科技之力

## 四、光影与氛围

### 基础光影方案

- **主光源**：人工冷白LED灯板，色温5600K-6000K（接近医疗标准），均匀分布在天花板，产生无影/极柔影效果
- **辅助光源**：培养槽内的绿色生物荧光（低强度），扫描舱内的冷蓝扫描光（激活时高强度刺目），控制室屏幕的蓝白反光
- **阴影特征**：无明显阴影（医疗级均匀照度），仅在仪器背面有微弱的设备遮挡影
- **对比度**：极低（接近无对比的均匀亮度），整个空间的平均亮度为75-85%
- **色彩基调**：冷白 #F5F5DC + 医疗绿 #A8D8A8（培养液），禁用任何暖色

### 氛围参数

- **空气质感**：完全密闭，无外界空气流通；循环系统产生的低频白噪声（约45分贝）充斥整个空间
- **粒子**：极少，偶见培养液蒸汽形成的微细雾气；无自然光线，故无灰尘粒子可见
- **温度感**：冷硬，恒温18-20°C，给人"非自然环境"的不适感
- **整体氛围**：诡异的超洁净感、冷酷的科技进步感、道德边界被冲破的不安感

## 五、声景与感官

- **环境底噪**：恒定的机械低频嗡鸣（空气循环泵）、液体泵送的脉动声、温度调节器的间歇启停声
- **标志性声音**：扫描舱启动时的"嗖——"高频扫描音，控制室按键的轻快电子音，培养液槽的偶尔气泡声
- **触觉/温度/嗅觉**：极冷的空气（可感受的肌肤刺痛）、不锈钢表面的光滑冰冷、淡淡的消毒液气味（酒精/过氧化物），无生物气息

## 六、场景演变轨迹

吴启思实验室在 ch53-ch71 的全部出场中，作为秘密科研设施，内部结构保持不变。虽荣康城在 ch55 陷落，但实验室因位置隐蔽且被继续利用，设备状态始终维持正常运作。

第71章的最后一具数字人制造过程中，整个实验室进入"应急运转"状态——吴启思日以继日工作，但灯光、温度、仪器流程无物理改变，仅人物活动的紧迫性和工作强度增加。

## 七、镜头预案

### 标志性构图

- **扫描舱全景**：冷白灯光下的金属密闭舱，旋转扫描臂在顶部缓缓转动，投射出微弱的阴影曲线——科技与人类之间的物理边界
- **培养室深视**：从走廊朝向培养室群，绿色荧光透过玻璃隔断形成竖向光柱，人形轮廓在液体中若隐若现——制造生命的梦幻与诡异的融合
- **控制室监控视角**：吴启思身影投在多屏显示墙上，屏幕曲线跳跃，反映在眼镜上——科学家与他的造物的关系浓缩

### 空间限制与注意事项

- 实验室为标准医疗级空间，走廊仅2.5m宽，不适合展开大场景叙事
- 扫描舱启动时产生强高频音，不利于同步对白，适合静默蒙太奇或音效强化
- 培养液的绿光具有高度视觉识别性，可用作"秘密科研进行中"的视觉暗示
- AI生成时避免"科幻未来感"过度（赛博朋克风格），应保持"现实医疗设施的冷硬升级"
