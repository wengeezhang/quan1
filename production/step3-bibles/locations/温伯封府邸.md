## 三、空间结构与地理

### 俯瞰布局

```
┌──────────────────────────────┐
│       街道临街面              │
├──────┬──────────────────┤
│ 正门 │  会客厅/用餐区   │

│      │                  │

│ 走廊 │  书房            │
│(昏暗)│ (权力中枢        │

├──────┼──────────────────┤
│楼梯  │  卧室            │
│(木质)│ (隐蔽后方)       │
└──────┴──────────────────┘
```

### 纵深与层次

- **前景**：铁制大门与门槛、玻璃与精木门框、门口侍卫身影
- **中景**：宽敞会客厅、黑檀木家具、精美走廊
- **远景**：书房内侧光线微暗的深处、楼梯盘旋上升至隐隐约约的二楼

### 关键地点

**会客厅**：府邸的门面，宽敞华贵，黑檀木方桌与高背椅座位讲究，墙上挂有字画，天花板有精细的木质纹样。这是温伯封宴请重要客人的场所，鸿门宴在此展开。

**书房**：温伯封的权力中枢，内有账簿、航运地图、商业信函。昏暗的灯光下，案台堆满文书。门口通常有贴身侍卫守护。朱围庸无法近身，成为其后来被克制的原因。

**卧室**：府邸后方的隐蔽空间，温伯封夜间休息之地。胡酹伪装送菜伙计，在此地遇到独处的温伯封，实施"智取"与灵魂交换。

**走廊**：连接各个房间的幽深通道，光线昏暗，墙壁为深棕色，隐约听得见脚步回音。

### 出入口与通道

1. **正门**——街道临界，两扇精木大门，通常有侍卫把守
2. **后院小门**——通往内院与厨房，胡酹伪装送菜人员的进口
3. **楼梯通道**——连接各层楼，木质扶手精良

### 标志物/视觉锚点

- **铁制大门**——泊岗镇最显眼的豪宅标志，冷硬的权力象征
- **黑檀木家具**——府邸内部的贵气所在，黑色木质的深沉感
- **书房的账簿**——权力与商业秘密的承载，最终被胡酹掌握

## 四、光影与氛围

### 基础光影方案

- **主光源**：自然日光仅从前窗与走廊高处窗口进入，侧光与底光混合；内室依靠油灯与烛台照明
- **色温范围**：会客厅日间 5200K-5800K（窗光），书房与卧室 3200K-3800K（烛火与油灯）
- **辅助光源**：书房油灯、卧室烛台、走廊墙上的壁灯
- **阴影特征**：深色木质家具投射硬阴影，走廊昏暗形成柔软的暗区；书房深处几乎无光
- **对比度**：中等偏高，前厅明亮（60-70%），后室压抑（30-40%），符合权力压制的视觉隐喻

### 色彩基调

| 色名 | HEX | 用途 |
|------|-----|------|
| 焦褐 | #2C1810 | 黑檀木家具的主色，代表权势与压抑 |
| 铜绿 | #5F7A61 | 墙壁与走廊的暗绿色调，华贵中的沉闷感 |
| 暗金 | #DAA520 | 烛火反射与装饰纹样的点缀色，权力的金色暗示 |

### 氛围参数

- **空气质感**：室内沉闷，窗帘与重木家具吸音；走廊弥漫陈年木料与上等家具蜡的气味；后室更显压抑与隐密
- **粒子**：仅窗口日光处可见尘埃浮动，内室则笼罩于微弱烛火下的烟雾中
- **整体氛围**：华贵而压抑的权力场所，充满隐蔽的杀机与商业秘密

- **环境底噪**：走廊脚步的回音、侍卫的低语、远处厨房隐隐约约的声音、烛火微弱的噼啪声
- **标志性声音**：精木大门的沉闷关合声、书房门被推开时的吱呀声、温伯封的权势话语与强硬语调
- **触觉/温度/嗅觉**：室内阴凉与书房的压抑冷意；家具精良但冷硬的触感；上等蜡烛与木料的幽香混合古旧感

## 六、场景演变轨迹

温伯封府邸在 ch14-ch21 的全部出场中，建筑与内部空间结构保持完全不变。宅邸作为泊岗镇权力象征的物理载体，其华贵的装饰与森严的布局始终如一。

从鸿门宴对峙到胡酹智取再到灵魂互换，这些事件虽在此地发生，但宅邸本身的空间状态无永久改变——烛火依旧照亮书房、家具依旧排列整齐、权力的物理象征依旧屹立。事件过后宅邸被新的"温伯封"（胡酹）接管，但其建筑风格与内部格局在视觉上保持一致。

## 七、镜头预案

### 标志性构图

- **正门外仰拍**：从街道仰望府邸正门与周围城市轮廓，强调其在泊岗镇的权力地位与视觉压迫感
- **走廊穿行**：从会客厅穿过幽深的走廊望向书房，光线逐渐暗淡，暗示权力中心的隐蔽与不可侵犯
- **书房书案视角**：从温伯封的位置俯拍，案台上的账簿与灯光，窗口透出的海景——权力的私密中心

### 空间限制与注意事项

- 室内深度有限，需利用走廊制造深度感
- 厚重的遮光窗帘限制自然光，人物面部需靠烛火与补光照亮
- 书房与卧室空间狭窄，大景别需在走廊或会客厅取景
- 黑色家具易产生死黑，需通过烛火产生高光来避免
