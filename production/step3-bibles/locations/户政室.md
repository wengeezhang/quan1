## 三、空间结构与地理

### 俯瞰布局

```
            入口（玻璃门）
                 ↓
      ┌─────────────────┐
      │  等候座椅区域   │
      │  塑料座椅排列   │
      ├─────────────────┤
      │  │  │  │  │  │  │
      │  窗口1-5        │
      │  办事台区       │
      └─────────────────┘
```

### 纵深与层次

- **前景**：玻璃门、等候区的塑料座椅
- **中景**：多个办事窗口、身份证打印机、档案架
- **远景**：办事台后方的墙壁，挂着各类办事指南

### 关键地点

**办事窗口**：玻璃隔板分隔，柜台后有电脑与打印机。窗口标识牌显示办理项目。

**等候区**：塑料座椅排列整齐，座椅磨损泛白。墙上贴着"请取号"的提示牌。

**叫号机**：红色LED显示屏，显示当前号码与办事窗口。

### 出入口与通道

1. **玻璃门主入口**——连接室外走廊，推拉自动门
2. **后方职员通道**——通往办公后台

### 标志物/视觉锚点

- **荧光灯管阵列**——顶部密布的日光灯，强冷白光
- **窗口分隔玻璃**——透明隔板上贴满办事流程说明
- **叫号机屏幕**——红色LED与蓝色显示，闪烁的号码

## 四、光影与氛围

### 基础光影方案

- **主光源**：顶部荧光灯阵列，均匀分布的冷白光（约5500K）
- **阴影特征**：极少自然阴影，人工光源产生的微弱影子
- **色彩基调**：冷灰白色主调，文件夹的焦褐色点缀
- **氛围参数**：空气静滞，无空气流动感，偶有纸张摩擦声和键盘敲击声

## 五、声景与感官

- **环境底噪**：键盘敲击声、打印机运作声、柜台小窗口的滑动声
- **标志性声音**：叫号机的电子语音播报、办事人员的冷淡回应
- **触觉/温度/嗅觉**：空调制冷的室温、塑料椅子的冰冷、纸张与碳粉的刺鼻气味

户政室在 ch1 的出现中，建筑与空间结构保持标准政府办公形态，无物理变化。该场景仅作故事起点，无后续出现。

## 七、镜头预案

### 标志性构图

- **推轨进入**：从玻璃门外推进，窗口玻璃反光暗示冷漠隔膜
- **俯视窗口**：从高机位俯拍办事窗口与等候者，凸显渺小无助感
- **窗口特写**：隔玻璃拍摄办事人员的漠然表情，突出官僚冷漠

### 空间限制与注意事项

- 室内空间较小，大景别受限
- 荧光灯光强度高，容易过曝，需要精确曝光控制
- 玻璃隔板产生反光与透视变形，需要机位选择避免强烈眩光
