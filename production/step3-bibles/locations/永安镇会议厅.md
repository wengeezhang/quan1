## 三、空间结构与地理

**地理位置特征：**
- 位于永安镇镇政府建筑内（可能为县衙或政府大楼）
- 处于镇中心行政区
- 交通便利，易于全镇居民集结
- 建筑具有一定权威性与庄严感（相比民居）

**室内空间布局：**

**主厅（公开集会区）**
- 规模中等，可容纳100-200人
- **座位布局**：
  - 镇长杜镇长主坐台前高位
  - 副镇长及主要官员坐于两侧
  - 普通居民与听众坐于下方区域
  - 后排预留站立空间

- **建筑特征**：
  - 木质框架，古朴庄严
  - 天花板中等高度，无宫殿的压迫感但有权力暗示
  - 前台可能有讲演台或升高的说话区域
  - 窗户供应自然光（白天）
  - 油灯或早期电灯照明（夜间）

**里屋（关门密谈区）**
- 与主厅相邻或隔壁
- 规模较小，容纳5-15人关键决策者
- **特征**：
  - 隔绝声音，可私密交流
  - 木质方桌或长形会议桌
  - 仅供镇领导与重要证人/顾问使用
  - 灯火昏暗但足以阅读文件

**走廊与连通空间**
- 主厅与里屋之间的走廊或通道
- 可能有登记处或门卫值守区
- 会议召集的动线清晰

**材质与纹理：**
- 枯草黄木质 #8B7355：座椅、墙壁装饰板、讲演台
- 焦褐色木框 #2C1810：门框、窗框、梁柱
- 暖黄油灯光 ：照明主色，营造古朴官署氛围
- 布料：粗麻布座垫（廉价但坚实）
- 地面：铺砌石板或泥地板（镇级规模）

## 四、光影与氛围

**光线特征：**

**Ch38白天/傍晚场景：**
- 主光源：窗口自然光（南窗可能面对街道）
- 补光：油灯或蜡烛开始点燃（随时间推进）
- 光线随故事进展从明亮逐渐昏暗
- 会议从白天进行到夜晚，光线演变构成戏剧时间感

**Ch39凌晨场景：**
- 纯人工光源（油灯、蜡烛、火把）
- 光线集中在讲演台与主桌区域
- 后排阴影深邃，低光照下人影显得诡异
- 凌晨的寒冷与灯光的孤独感

**氛围营造：**

**Ch38的转变弧线：**
1. **开场**：阳光明媚，众人闲散散坐，带有"日常"的无所谓
2. **宋小仙陈述开始**：光线集中于讲演台，众人面向前方
3. **嘲笑与怀疑阶段**：横向的交流，低声讨论，光线相对均匀
4. **信服与恐惧**：光线更集中，阴影增强，众人身体向内紧缩
5. **五天之约确立**：全场寂静，仅有镇长声音在回响，灯光强化庄严

**Ch39的决策氛围：**
- 凌晨时刻的疲惫与紧张并存
- 熊大宝作证的公开性 + 里屋密谈的隐秘性形成对比
- 从信息披露到决策确认的压力递进
- 最后全镇撤离决定的沉重与必然

**色彩配置：**
- **枯草黄主调** #8B7355：座椅、讲演台、温暖的权力象征
- **焦褐次调** #2C1810：门框、结构、历史沉淀感
- **暖黄油灯光** ：照明与氛围的主色，随时间推移变得昏暗
- **阴影深化** ：随着故事进展，阴影逐渐吞没房间
- **禁用色** ：#FFFFFF（不得使用纯白光照）

## 五、声景与感官

**听觉元素：**

**Ch38会议过程中：**
- **宋小仙的陈述**：激动、急促、带着求生欲望的声音
  - 描述恶魔的细节，声音中混有恐惧与说服的力度
- **众人的反应**：
  - 开场：笑声、嗤笑声、低声评论
  - 中段：逐渐安静，有人开始追问细节
  - 后段：沉重的吸气声、紧张的窃窃私语
- **镇长杜镇长的声音**：权威、理性、但逐渐显露震惊
- **背景声**：座位挪动声、人群躁动声、脚步声

**Ch39凌晨场景：**
- **紧急敲钟声或通知声**：将众人从睡眠中唤醒
- **熊大宝的证词**：沙哑、疲惫但坚定的声音，逐字强调恶魔的真实性
- **镇长的决策宣言**：可能通过扩音（小铜钟）传向全镇
- **沉默**：里屋机密商议期间的压抑寂静
- **撤离动员声**：最后可能有镇长的撤离通知与民众的哭泣、怨声

**触觉体验：**
- 座椅的坚硬与长时间坐卧的不适
- 凌晨冷空气的刺骨感
- 紧张中出汗的湿润感
- 擦肩而过的陌生人接触

**嗅觉线索：**
- 油灯与蜡烛的燃烧气息
- 密闭空间的人体体臭积累
- 古旧木材的陈香
- 凌晨的潮湿与霉变气味

## 六、场景演变轨迹

**时间轴与情感弧线：**

**Ch38阶段（信息披露与态度转变）：**

1. **前期-怀疑与嘲笑**
   - 时间：下午或傍晚，众人懒散聚集
   - 宋小仙突然陈述恶魔威胁
   - 众人初期反应：这是谣言、怕什么、少女疯癫之言
   - 笑声充满会议厅
   - 镇长杜镇长也带有疑虑

2. **中期-证据与动摇**
   - 宋小仙详细描述恶魔特征、来历、威胁时间表
   - 某些细节令人无法否认（如未来已知的事件）
   - 众人笑声逐渐止息
   - 有人开始严肃提问
   - 镇长神色渐变，从怀疑转向重视

3. **高潮-绝望与确认**
   - 宋小仙的陈述抵达最高潮
   - 众人真正感受到威胁的现实性与紧迫性
   - 从集体怀疑转向集体恐惧
   - 镇长做出决策：五天之约生效

4. **尾声-沉重的新常态**
   - 会议落幕，众人面色凝重离开
   - 小镇即将面临的命运被众人接纳
   - 准备阶段开始

**Ch39阶段（决策与行动方案）：**

1. **开幕-紧急召集**
   - 时间：凌晨1-2点（距离恶魔约定时间仅余3-4天）
   - 急促的敲钟或信使通知
   - 镇民从睡眠中被唤醒，仓促来到会议厅
   - 氛围紧张与疲惫交织

2. **证人出庭-熊大宝作证**
   - 熊大宝详细描述自己的遭遇与确认
   - 进一步验证了宋小仙的预言
   - 消除了任何剩余的怀疑者
   - 众人的恐惧达到顶峰

3. **密室决策-里屋商议**
   - 镇长与核心官员进入里屋
   - 讨论撤离路线、时间、物资
   - 凌晨的沉默中响起激烈讨论声
   - 最终方案确定：每天50里，5天到凤廊山脉刀片道

4. **动员令-全镇撤离决定**
   - 镇长从里屋出来，宣布决议
   - 决定内容：全镇撤离，假称"大洪水"
   - 年轻人强行协助撤离（半强制性）
   - 众人的绝望与接纳并存

5. **尾声-行动开始**
   - 会议厅逐渐清空，众人离开准备物资
   - 镇长独自留在会议厅，沉思着承担全镇命运的重量
   - 或者镇民的哭泣与怨声成为最后的背景音

**关键转折点：**
1. 从嘲笑到信服的转变（Ch38）
2. 五天之约的生效确认
3. 熊大宝作证的最终确认（Ch39）
4. 全镇撤离的不可逆转决定
5. 从被动等待到主动逃亡的转变

## 七、镜头预案

**镜头1：会议厅全景进入**
- 推轨或跟拍镜头，进入主厅
- 呈现座位排列、众人散坐的场景
- 窗外逐渐暗去的光线
- 油灯逐盏被点燃
- 强调从日常到非常的转变

**镜头2：宋小仙陈述的对峙**
- 宋小仙站于讲演台或中心位置
- 众人围坐成半圆或方阵，面向她
- 众人脸部特写切割：嘲笑→惊讶→恐惧→沉默
- 宋小仙身姿与表情的坚定与绝望
- 光线逐渐集中于她身上

**镜头3：众人的群体反应**
- 宽景镜头呈现整个厅的反应
- 人群身体语言的变化：懒散→直立→内紧缩
- 各个区域不同人物的脸部表情蒙太奇
- 笑声逐渐停止，寂静降临的转折

**镜头4：镇长的决策时刻**
- 镇长杜镇长坐于主位，眼神从怀疑→确信→沉重
- 他缓缓起身，众人的目光聚焦于他
- 他的声音回荡在整个厅堂
- 宣布"五天之约"的那一刻，众人的绝望凝固

**镜头5：凌晨敲钟的惊醒**
- 镜头从黑暗中开始，突然一声钟声
- 快速蒙太奇：镇民被惊醒、穿衣、起身、赶向会议厅
- 急促的脚步声、混乱的人声
- 光线与阴影的剧烈对比

**镜头6：熊大宝的证词**
- 熊大宝站起身，全厅寂静听他发言
- 他的口述被认真记录
- 众人面色更加凝重
- 可选特写：某个听众的泪水滑落脸颊

**镜头7：里屋密谈的暗示**
- 镇长与几人进入里屋，木门关闭
- 从主厅看，隐约听到里屋的讨论声
- 等待中的众人的焦灼（可用长镜头或静止镜头）
- 时间感通过背景的安静变化而显现

**镜头8：最终决议的宣布**
- 里屋门打开，镇长走出
- 他走向讲演台，全厅屏息以待
- 宣布决议：全镇撤离、假称洪水、每日50里行进
- 众人反应：有人哭泣、有人怨声、有人麻木接纳
- 最后落幕于全厅的哭声与压抑的空荡

**镜头9：撤离前夜的会议厅**
- 会议厅逐渐清空
- 椅子被推开，灯火渐熄
- 镇长独自站于讲演台，望向空旷的座位
- 肩负全镇命运的沉重感与孤独感
