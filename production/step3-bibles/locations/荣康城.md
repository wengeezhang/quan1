## 三、空间结构与地理

### 俯瞰布局

```
                    定达方向（北）
                         ↑
            ┌─────────────┼─────────────┐
            │   北部区域   │             │
            │   北城门     │             │
     ┌──────┼─────────────┼─────────────┼──────┐
     │ 西部 │  外围区域   │  中央大道   │ 东部 │
     │ 薄弱 │  (商贸区)   │  (20米宽)   │ 密集 │
     │ 防线 ├─────────────┼─────────────┤ (东北热闹)
     │ 曲江 │   护城壕沟   │   10米宽   │      │
     │ 流过 │   水下尖桩   │             │      │
     │      │             │             │      │
     │ 南部 ├─────────────┼─────────────┤ AA大学
     │ 南门 │  内城墙      │   内部区域  │ (南侧)
     │ (突破│  (10米+高)   │   (城市核心)│      │
     │ 口)  │             │             │      │
     └──────┴─────────────┴─────────────┴──────┘
                    南门/曲江方向
```

**说明**：
- **外围区域**：商贸、行政、水利部门集中；中央大道环绕；东北部最繁忙，西南部防守薄弱
- **内城区域**：10米以上高城墙围绕；城市核心政治、军事、民居混杂
- **防御特点**：双层城墙；南门为最易突破点

### 纵深与层次

- **前景**：城墙石砖/夯土纹理、防守器械、士兵身影
- **中景**：外围中央大道上的牌坊（高8米，每500米一座）、市集建筑、兵营
- **远景**：内城城墙轮廓、内部高大建筑（政务院、AA大学建筑群）、远方天际线

### 关键地点

**外围区域**：荣康城商贸与行政枢纽，宽20米中央大道穿行其中，每隔500米矗立牌坊展示千年历史。东北部热闹繁华（贸易往来频繁），西南部因靠近曲江而防守相对薄弱。

**内城城墙**：高10米以上的坚固防线，城门包括东门、南门、北门。外城陷落后，守军退守内城；内城防线是第二道关键防御。

**指挥部**：郭云成的军事指挥中心，位于内城内部。后被曹元艺潜伏渗透并通过栽赃策略接管。

**AA大学**：位于荣康城内南侧，被改为两万内应部队的隔离营地。校园宽敞、易于隔离，内有实验室（吴启思的数字人项目）、食堂、宿舍等设施。

**南门**：朱围庸攻城的突破口，防守相对薄弱；也是渗透者入城的主要通道。

### 出入口与通道

1. **北城门**——通往定达方向，接受官方援助的通道
2. **东门**——商贸通道，胡酹混入信使队伍的入口
3. **南门**——最薄弱环节，朱围庸攻城的重点突破位置
4. **西门**——外部区域有限，因曲江流过而防守较弱
5. **引水渠**——城市供水通道，胡酹派手下剪断铁丝网作为伏线传递通道（后用于曹元艺与胡酹的秘密联络）
6. **内城墙通道**——外城撤退时的路线，被渗透者混入的关键缝隙

### 标志物/视觉锚点

- **外围中央大道的牌坊群**——每500米高8米的牌坊，记录荣康城千年历史，是俯瞰或远景时的视觉标识
- **内城城墙**——高耸的双层防线，城墙上旗帜飘扬，防守器械排列整齐
- **AA大学建筑群**——校园建筑集中、可识别的学术建筑特征，战争期间被隔离为营地
- **曲江**——城西流过的河流，围城策略中的自然屏障

## 四、光影与氛围

### 基础光影方案（防御期）

- **主光源**：自然日光，侧光为主（30°-60°夹角），符合全局 Art Direction 默认模式
- **色温范围**：日间 5200K-5800K（自然光），阴天或雨天 6500K-7500K（冷灰）；夜间 3200K-3800K（火把/油灯补光）
- **辅助光源**：城墙防守火把、指挥部油灯、广场商铺照明
- **阴影特征**：中等硬度自然阴影，方向明确；城墙上防守者产生的长影；内城复杂阴影因建筑密集而深重
- **对比度**：中等偏高，暗部保留 5-10% 细节，亮部不过曝

### 色彩基调

| 色名 | HEX | 用途 |
|------|-----|------|
| 焦褐 | #2C1810 | 城墙夯土与石砖基色、建筑木质结构、暗部基调 |
| 枯草黄 | #8B7355 | 日光下外围建筑、牌坊木质、民居屋顶的温暖基调 |
| 暮蓝 | #3A4A6B | 防守紧张、战争阴天、夜间城内的压抑色 |
| 苔绿 | #4A6741 | 曲江周边植被、西侧防线背景（点缀用） |

### 战时光影特征（防御期後期）

- **火光与烟尘**：攻城期间出现火光（照明弹、火把堆积），烟尘笼罩城墙与中央广场
- **频闪效果**：近距离"移魂夺魄"发生时，室内灯光短暂频闪（3-5Hz），持续 0.5 秒
- **混乱光影**：夜间攻防时，光源多方向、不规律，产生混乱多向阴影

### 光影偏移（陷落期）

- **空间状态**：城墙破损、城门被攻破，原有防守设施残破或被占用
- **光影变化**：破损城墙露出内部，改变光线通透路径；占领后火光减少，恢复为占领军的日常灯光方案（偏向朱围庸部队的冷紫色调）

### 氛围参数

- **空气质感**：城市内商业气息浓厚（防御期初期），战时烟尘增加，通透度下降；雨天时曲江带来湿润感
- **粒子**：防守期有烟尘（火堆、防守活动）；陷落期有战火烟尘与灰烬；夜间火把烟雾可见
- **整体氛围**：防御期为坚固、紧张、压抑；陷落期为混乱、崩溃、权力更迭

## 五、声景与感官

- **环境底噪**：防御期——士兵脚步声、旗帜飘动声、弓弦绷紧声；商业区的市集嘈杂（初期）
- **标志性声音**：城墙防守的锣鼓预警、内外城门的沉重关闭声、指挥塔上的军号
- **战争声景**：攻城时的冲锋呐喊、梯子碰撞城墙声、石块滚落声、弓箭破空声、士兵倒地的沉闷声
- **触觉/温度/嗅觉**：冷硬的城墙石砖、防守者身上的铠甲与汗味、火焰烘烤时的热浪、烟尘呛鼻；陷落期战后的尸体腐臭与火烧焦臭

## 六、场景演变轨迹

#### 防御期（ch48-ch53）

**空间状态**：双层城墙防御体系完整，外城与内城分界清晰。郭云成指挥的守军占据城墙、指挥部和主要街道。外围区域商业繁忙，内部区域人口密集。

**光影与色彩偏移**：防守期初（ch48）为正常日间商业城市光线，随着战事升级（ch49-ch52）光线逐渐暗沉（多阴天），夜间火光增加。内城陷入压抑的暮蓝与焦褐色调；郭云成权力被动摇后（ch51-ch52），指挥部周围光线更加昏暗，反映士气低落。

#### 陷落期（ch54-ch55）

**空间状态**：外城城墙被攻破，防线崩溃。南门城门被破坏，两万内应部队涌入。朱围庸势力接管城市，郭云成等战俘被关押。AA大学被改为隔离营地。城内秩序重建为占领军体制。

**光影与色彩偏移**：城门破损后光线通透性改变；朱围庸占领后，占领军的冷紫色调逐步覆盖城市（相对防守方的暖黄）；AA大学被隔离，周围灯光明亮但诡异（对比校外）；实验室内部呈冷白科技感光线（吴启思的设施）。

## 七、镜头预案

### 标志性构图

- **城墙远景**：从定达方向俯瞰或远景，双层城墙的壮观防线清晰可见，外围中央大道与牌坊形成纵深感，暗示城市的庞大与防守的坚固
- **攻城俯拍全景**：从高空俯视攻城战场，胡酹的梯队攀登外城、城墙上混乱的防守者、破晓的光线扫过战场——展现大规模军事行动的混乱与力量对比
- **内城城墙防守线**：低机位或平视，内城高耸的城墙占据画面大部分，城墙顶部防守者的剪影与旗帜，体现防线的不可逾越感
- **AA大学隔离营地**：校园建筑群被隔离圈包围，远景是城市，近景是岗哨——暗示秘密基地与城市的分离

### 空间限制与注意事项

- 外城中央大道较宽（20米），但两侧建筑限制了大景别的展开，适合纵深构图或街道视角
- 内城城墙高度（10米+）限制了从城外仰拍的有效角度，需要从高处或远处才能拍摄城墙全貌
- AA大学校园宽敞，适合表现集团行动、隔离感、以及内部秘密活动的隐蔽性
- 夜间防守场景光源受限（火把、油灯），需避免过度照亮城墙细节，保留战场的混乱与危险感
