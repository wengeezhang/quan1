## 三、空间结构与地理

### 俯瞰布局

```
         山顶寺院
            │
    ┌───────┴────────┐

             │

    │  内堂          │
    │  (神圣收藏)    │
    │  十八元神      │
    │  水晶棒所在    │
    └─────────────────┘
```

### 纵深与层次

- **前景**：寺前台阶、石刻栏杆、香炉与灯笼
- **中景**：木质大殿、檐角翘起的屋顶、金色琉璃瓦闪闪
- **远景**：山脉环抱、古树参天、晨曦或夕阳照亮的山林

### 关键地点

**寺前广场**：香客朝圣的空间，通常寂静无人。广场中央有古老的石刻栏杆围绕的空地，昭示这里的神圣性。

**大殿**：寺院的主体建筑，木质结构、金色琉璃瓦屋顶、檐角高翘。内部供奉佛像，光线昏暗神秘。

**内堂**：大殿后方的密室，存放十八元神水晶棒的秘密地点。仅有悟道者与灵魂使者可进入，散发超自然的金光。

### 出入口与通道

1. **寺前石阶**——进入寺院的正式通道，数十级台阶向上
2. **山林通道**——通往周边山区的自然小路
3. **内堂密道**——连接大殿与秘密内堂，仅可通行灵魂体

### 标志物/视觉锚点

- **金色琉璃瓦屋顶**——在日光下闪闪生辉，象征神圣与永恒
- **寺前的香炉与灯笼**——信仰的物质载体
- **古树参天**——见证千年岁月的自然圣物

## 四、光影与氛围

### 基础光影方案

- **主光源**：自然日光，通常侧光或晨光；超自然金光（内堂）
- **色温范围**：户外 5200K-5800K；内堂 2800K-3500K（超自然暖金色）
- **辅助光源**：烛火、香烟缭绕光线、超自然灵光
- **阴影特征**：硬度中等（户外）→ 极软（内堂）；方向明确（外）→ 无方向（内）
- **对比度**：中等高（外）→ 低（内），暗部保留神秘感

### 色彩基调

| 色名 | HEX | 用途 |
|------|-----|------|
| 古木褐 | #6B4423 | 木质建筑与檐柱 |
| 暗金 | #DAA520 | 琉璃瓦与超自然金光 |
| 深古铜 | #8B6914 | 内堂的古朴感 |

### 氛围参数

- **空气质感**：清冷的山间空气、香烟与木香混杂、超自然的洁净感
- **粒子**：香烟缭绕形成的光影斑驳、神圣金光的微粒闪烁
- **整体氛围**：古老、宁静、神圣、超越时间、救赎与永恒

## 六、场景演变轨迹

少林寺在 ch65 作为故事的终局圣地出现，仅有一次出场但意义重大。寺院本身是古老与不变的象征——它见证了千年历史，也见证了最后的灵魂仪式。场景的视觉状态保持古刹的原始与神圣，无任何破坏或时间改变。这是一个"永恒之地"的设定，代表故事超越肉身与时间的最终升华。

## 七、镜头预案

### 标志性构图

- **寺院全景**：从山下仰拍寺院，金色琉璃瓦在晨曦或夕阳中闪烁，山林环绕，显示其与世隔绝的神圣性
- **大殿正面**：直视大殿正门，对称的建筑、翘起的檐角、昏暗的内部，象征神圣与秘密的交界
- **内堂光影**：超自然的金光从内堂溢出，照亮周围石地，营造灵魂仪式的神圣瞬间

### 空间限制与注意事项

- 寺院建筑的对称性需强调，机位多用中轴线或侧角
- 金光效果需克制，避免过度渲染而失去写实感
- 山林背景需营造"世外之地"的隐蔽感，远景模糊但存在
