## 三、空间结构与地理

### 俯瞰布局

```
       雪坡（来处）
          │
      ┌───┴───┐
      │       │
   岩壁  热气凹地
    (冷)  │
      │  ┌─┴─┐
      │  │大锅│(中心)
      │  └─┬─┘
      │    │
    ┌─────┴─────┐
    │   草棚     │
   (北)  (10m高) (南)
    └─────┬─────┘
          │
       雪坡(逃脱方向)
```

### 纵深与层次

- **前景**：积雪地面、凸起的岩块、铁链与木柱
- **中景**：大锅与三角架、草棚的木骨架与草苫顶
- **远景**：凹地四周的陡峭白雪覆盖的岩壁、天际线上的山脊

### 关键地点

**热气腾腾的凹地**：山谷中部的微凹地形，地下存在热源（火山地热或温泉），导致该区域冬季仍保持无冻状态，地表温度约10-15°C，产生持续的白色蒸汽。空气中弥漫着硫磺与温水的混合气味。凹地是整个谷地的"温暖中心"，白巨一族的生活核心。

**大锅与三角架**：凹地中央竖立三根粗大木干（直径0.5米，高约4米），上方用粗铁链与木栓固定一个巨大的金属锅（直径3米），下方熊熊燃烧的篝火煮炖食物。锅内常年保持炖煮状态，是白巨一族的主要食物来源。

**草棚**：凹地北侧，用粗木搭建的框架结构，覆盖厚重的干草与兽皮，高约10米（适应神兽的身型），内部足以容纳三个身高5-6米的生物。棚内地面铺着草垫与兽皮。棚口朝南向凹地，可观察外界。

**囚禁点**：凹地西侧的一根木柱，粗大如树干，上面缠绕着粗铁链（长3-4米），链端有铁锁扣。第9章中朱围庸被锁于此，成为该空间的"人类标本"。

**逃离之路**：凹地南侧的雪坡，相对缓和但需翻越，直接通往山脉深处的丛林。朱围庸最后的脱逃路线。

### 出入口与通道

1. **北侧雪坡（来时路）**——从山脉主干道下滑进入谷地，陡峭且易失控
2. **南侧雪坡（逃生路）**——通往山脉更深处，林地密集，追踪困难
3. **西侧岩壁**——陡峭无法攀援，仅作为地形背景

### 标志物/视觉锚点

- **持续升腾的白色蒸汽**——漫天雾气，是"隐谷存在"的唯一视觉线索
- **三角架大锅**——从远处就可辨识，像一座原始的"庙宇"
- **白毛巨兽的身影**——身高5-6米的轮廓，相对于人类的渺小对比
- **闪闪发光的铁链**——被日光照亮时显眼，代表"囚禁"的视觉符号

## 四、光影与氛围

### 基础光影方案

- **主光源**：自然日光，从上方（天顶光），因山谷四周岩壁遮挡，光线呈竖直投射，产生极端高对比
- **色温范围**：日间 5200K-5800K，雪面反光强烈加热至 6000K+（刺眼），阴影处偏冷蓝 4000K
- **辅助光源**：篝火的橙红热光（约 2200K），与日光形成冷暖对立
- **阴影特征**：极其锐利的硬边界阴影，草棚、木柱投射出清晰的黑色阴影
- **对比度**：极高，雪地亮部与岩壁暗部、阴影区的对比至少 1:8

### 色彩基调

| 色名 | HEX | 用途 |
|------|-----|------|
| 雪白 | #F5F5F5 | 地面与神兽毛发 |
| 焦褐 | #2C1810 | 岩壁、草棚、火光阴影 |
| 炎橙 | #FF6B35 | 篝火核心 |

### 氛围参数

- **空气质感**：冰冷刺骨的山顶空气（-10°C 左右）与凹地温暖蒸汽形成极端对比；蒸汽弥漫导致能见度不稳定（10-30米起伏）
- **粒子**：漫天白色蒸汽构成整个谷地的"雾幕"；雪地反光形成的极强阳光粒子；篝火的飞灰与炭粒
- **温度感**：触觉上的极端二元性——雪地冰冻与凹地温暖的瞬间切换
- **整体氛围**：超现实的荒凉、原始粗粝的生存环境、宿命的陷阱感、神话与现实的物理碰撞点

## 五、声景与感官

- **环境底噪**：持续的蒸汽喷鸣声（低频沙沙声）、篝火的噼啪燃烧声、冰雪在重压下的微微开裂声
- **标志性声音**：白巨一族的低沉嗓音（人类无法发出的低频音）、铁链拖曳与摩擦声、神兽脚步踏破冰雪的沉重声响
- **触觉/温度/嗅觉**：刺骨寒冷的冰风、脚踩积雪的硬脆质感、温泉硫磺的刺鼻气味、篝火烤肉的浓郁肉香、兽皮与草的陈腐气息

## 七、镜头预案

### 标志性构图

- **俯视凹地全景**：从山谷顶部俯瞰，蒸汽漫天，大锅与草棚成为烟雾中的黑色轮廓，三个白毛巨兽如山丘般伏卧——人类视角的绝对渺小感
- **白毛巨兽与人的尺度对比**：灵闽（5-6米高）面朝朱围庸（1.7米高），两者在画面中的身体比例展现压倒性的物理差异
- **篝火与冰雪的冷暖对立**：橙红火光照亮草棚，背景白雪被火光映成淡红，呈现视觉上的暖冷分割

### 空间限制与注意事项

- 山谷凹地的视角有限，大广角镜头需从进入谷地的坡道俯视或从草棚顶部平视
- 蒸汽的浓度变化（浓时能见度仅10米，稀时可见50米）可用作镜头的有意模糊/清晰切换，制造"幻觉"与"现实"的交替
- 神兽身材巨大，近景难以容纳整个躯体，更适合部分身体特写（如巨掌、长耳、白毛质感）与远景全身
- 铁链的细节（链结、生锈纹理）可强化"囚禁"的视觉意象
