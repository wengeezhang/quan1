## 三、空间结构与地理

### 整体空间分层

**第一层：地上树冠（天空）**
- 高度：30-50米以上
- 特征：密集枝叶，形成自然穹顶
- 光线：散射日光，构成昏黄色光幕
- 温度：相对温暖，与外界相似

**第二层：树干与分支（中界）**
- 高度：5-30米
- 特征：多条粗大树干，互相缠绕
- 纹理：沧桑龟裂，苔藓覆盖
- 视觉焦点：树干背面与根部交界处存在隐蔽空间

**第三层：根系网络（地表至浅地下）**
- 深度：地表至10米以下
- 特征：
  - 主干根粗壮如同人体躯干，形似仰卧女性躯体
  - 左侧根系呈"女性右臂"姿态，延伸向西北
  - 右侧根系呈"女性左臂"姿态，延伸向东南
  - 头部根系盘旋上升，形成类似"头颅"的球形根团
  - 中央根系向下扎入深处，形成"脊椎"通道
- 表面：粗糙，覆盖湿润苔藓与菌类
- 空隙：根系间形成自然通路，可供人类进入

**第四层：阴间十八层塔（地下深处）**
- 深度：10米以下，向下无限延伸
- 特征：
  - 根系不再是树的身体，而是连接两个世界的"脐带"
  - 通道壁面由古老石质与树根交织而成
  - 每向下延伸，光线颜色从冷绿变为冷紫，最后变为近乎黑色的深紫
  - 温度逐层下降
  - 声音从寂静变为低频共鸣

---

### 核心视觉特征：根系的"仰卧女性形态"

**头部区域**
- 位置：榕树北侧，距树干5-8米处
- 形态：球形根团，宛如女性头颅侧卧
- 特征：
  - 表面光滑相对光滑，仿佛"脸颊"与"额头"
  - 根须如同"长发"披散向下
  - 空洞与纹理形成类似"眼窝""鼻孔"的幻觉
- 视觉意义：此处是观察者与榕树"对视"的位置，产生强烈的被"凝视"感

**躯干区域**
- 位置：树干与主根交界处
- 形态：宽阔的根系网络，宛如女性胸腹部
- 特征：
  - 根系排列形成类似"肋骨"的纹理
  - 中央凹陷处如同"脐部"，向下延伸成深通道
  - 左右对称的根系如同"乳房"高隆
- 视觉意义：此处是进入阴间的主入口，也是"生命与死亡的交界点"

**四肢区域**
- 位置：树干周围向外延伸
- 形态：
  - 左侧根系向西北方向延伸，如同"右臂"（以仰卧者视角）
  - 右侧根系向东南方向延伸，如同"左臂"
  - 两侧根尖端向上翘起，如同"掌心向上"的姿态
- 特征：根系逐渐变细，须根密集，如同"指骨"与"指尖"
- 视觉意义：此处是"接纳"与"释放"的通道，人物可从四肢根系进出

**脚部区域**
- 位置：树的南侧或东侧
- 形态：根系向下扎深，形成"双足"姿态
- 特征：
  - 粗壮的脚部根系深扎地下
  - 细根如同"脚趾"散开
  - 此处地面最不稳定，易出现沉陷或裂隙
- 视觉意义：此处是最古老的、最深与地下世界相连的位置

### 空间丈量与标志点

**主体周长** | 约80-120米（基于树冠直径）
**最高点** | 50米以上（树梢）
**最低点** | 地下30米以下（根系延伸至阴间）
**观察最佳点** |
- 榕树东侧：可观看根系全貌与树干立体感
- 榕树北侧：可对视"头部"根团，产生凝视感
- 树冠内部：可感受天然穹顶与光影交互

**危险区域**
- 树冠正下方：易被落叶、断枝击中
- 根系缝隙：易陷入，时空感扭曲
- 中央凹陷处（脐部）：吸引力强，接近时易失控

## 四、光影与氛围

### 地上部分：自然光线系统

**日光环境**
- **晨间（5-8点）** |
  - 光线角度低，树影拉长
  - 光线从东侧透过枝叶，形成金色光束（立体感强）
  - 地面光斑稀疏，大多数区域仍在阴影中
  - 氛围：神秘、沉静，生机初显

- **午间（9-16点）** |
  - 光线垂直向下，树冠完全阻挡
  - 地面昏暗，如同永恒的黄昏
  - 散射光线在根系表面形成微弱高光
  - 氛围：压抑、诡异，时间感失效

- **傍晚（17-19点）** |
  - 光线从西侧低角度射入，树体逆光呈剪影
  - 部分光线穿透稀疏处，形成金色光柱
  - 树影投向地面，覆盖面积最大
  - 氛围：苍凉、悲伤，分界明显

- **夜间（20点-次日4点）** |
  - 完全自然光缺失
  - 月光难以穿透树冠
  - 地面完全黑暗
  - 仅超自然光可见（见下文）

**光质特征**
- 绿色光（树叶过滤后的日光） |
  - RGB: (74, 103, 65)
  - 冷绿色，带有古老感
  - 苔藓与腐殖质反射，加强此色调

- 金黄色光（散射日光）|
  - RGB: (200, 170, 80)
  - 仅在特定时间、特定角度出现
  - 与暗色调形成强烈对比

---

### 根系部分：超自然光线系统

**默认状态光线**
- 无主动发光源
- 仅反射地上散射光
- 根系表面呈灰白色，略显阴冷
- 色温：冷色调，约4000K

**激活状态光线**
- 冷紫色荧光从根系内部散发
  - RGB: (91, 58, 107)
  - 强度随故事进展逐步加强
  - 不刺眼，但具有强烈穿透感

- 根系间缝隙泛出冷光
  - 颜色从紫转向蓝、再向深紫
  - 光源位置不明确，似来自地下深处
  - 制造悬浮感与超现实感

**终局状态光线**
- 暗紫光主导，完全压制自然光
- 光线如脉搏般搏动，有生命感
- 根系与光交融，难以分清边界
- 色温：极冷，接近2000K的深紫色域

---

### 氛围营造要点

**视觉氛围**
- 古老感：树龄的沧桑，根系的粗糙纹理，苔藓的覆盖
- 神圣感：对称的根系形态，光影的庄严性，空间的宏大感
- 诡异感：根系形似人体的不安感，超自然光的离奇性，时空扭曲感
- 压迫感：树冠压低，根系包围，视觉逐层下沉

**听觉氛围**
- 自然声：枝叶沙沙声，昆虫鸣叫，雨水滴落（见《五、声景与感官》）
- 超自然声：低频共鸣，无源呼吸声，远古低吟

**触觉氛围**
- 温度递进：地上温暖 → 根系处凉爽 → 深处极冷
- 湿度递进：地上干爽 → 根系处湿润 → 深处潮湿阴冷
- 触感递进：粗糙树皮 → 光滑苔藓 → 冰冷石质根通

## 五、声景与感官

### 声景层次

**第一层：地上自然声（日间）**
- 枝叶沙沙声（风速依存）
  - 微风时：柔和沙沙，间歇性，宁静感
  - 强风时：鬼哭狼嚎，持续性，恐怖感
  - 无风时：寂静，反而显得诡异

- 昆虫鸣叫声
  - 蝉鸣：高频，在树冠最高处
  - 蛙鸣：低频，在根系潮湿处
  - 蟋蟀：中频，间歇性，暗示夜幕降临

- 鸟鸣声（稀少）
  - 很少有鸟类进入此处
  - 偶有鸟鸣显得诡异，预示事件发生

- 雨水声（雨季）
  - 初期：轻快的滴落声
  - 持续：汇聚成溪流声，从树冠流向根系
  - 暴雨：强烈的冲击声，树体共鸣

**第二层：地下通道声（全时段）**
- 低频共鸣
  - 源头：根系网络的结构震颤
  - 频率：约20-40Hz，人类感知为振动而非声音
  - 功能：提醒进入者已进入超自然领域

- 呼吸声
  - 缓慢的吸气与呼气
  - 似乎来自地下深处
  - 节奏不规律，增强不安感

- 低吟声
  - 频率极低，音色古老、原始
  - 无法辨认来源与含义
  - 听者感到被"唱诵"

**第三层：终局期声景（激活全状态）**
- 多层声音同时进行
  - 地上声与地下声混合
  - 自然声与超自然声交织
  - 制造音响的复杂度与压力

- 时空扭曲声
  - 回音的逻辑混乱（前方的声音从后方回响）
  - 声音距离感不准确（远处声音显得近，近处声音显得远）
  - 增强失方向感

---

### 感官体验要点

**视觉体验演进**
- 初入：视线清晰，能识别周围环境
- 深入：视线逐步受限，色彩逐步冷色化
- 激活时期：色彩饱和度增加，对比度强化，产生幻觉感
- 终局期：视觉接近失效（过度刺激），仅能感知光的脉动

**听觉体验演进**
- 初入：正常听觉范围
- 深入：低频声音逐步凸显，高频声音逐步消失
- 激活时期：听觉敏感度异常升高，细微声音也显得巨大
- 终局期：听觉过载，声音混成一体，难以分辨

**嗅觉体验演进**
- 初入：古木与腐叶的陈旧气味
- 深入：潮湿、霉腐的气味加强
- 激活时期：不属于人世的奇异气味（难以描述）
- 终局期：气味消失，代之以冰冷感

**触觉体验演进**
- 初入：温暖、干爽
- 深入：逐渐变凉，湿度增加
- 激活时期：极冷，似有无形之物接触皮肤
- 终局期：温度感消失，仅余冷的抽象感

**成长阶段（第11-30章）：觉醒与异变**
- 关键事件：

  - 第一次明显的光线异常
- 场景状态：逐步激活，异常现象增加
- 观看者心理：困惑、恐惧、着迷

  - 阴阳两界的力量直接交汇

**终局阶段（第61-71章）：和解与重生**

- 场景状态：光线与声音逐步消退或转变
- 观看者心理：解脱、希望、沉思
- 视觉变化：根据结局性质，光线向金色或深色转变
- 功能：承载故事的终极答案与新的开始
- 镜头语言：宽景与空景，强调静寂与永恒

### 与人物互动的动态变化

**人物在场景中的影响规律**
- 主角进入 → 场景微微共鸣，低频振动增加
- 冲突发生 → 光线波动，根系纹理凸显
- 真相揭示 → 声音达到最大强度，光线最亮
- 人物离开 → 场景逐步恢复平静，光线衰退

## 七、镜头预案

### 标志性构图一：透过根须缝隙的远景视角

**视觉特征**
- 前景：粗壮根系形成自然框架
- 中景：根系缝隙透出的世界片段（树冠、人物、或阴间空间）
- 背景：因光线被分割而形成的多层次空间感

**镜头参数**
- 焦距：50-85mm（标准到中焦）
- 光圈：f/4-f/8（保持前景与背景清晰）
- 景深：浅景深，突出根系纹理
- 色温：4500K-5500K（自然光）或3000K（超自然光）

**象征意义**
- 对观看者视野的束缚与引导
- 两界之间的分界与连通
- 透过表象看清本质

**应用场景**
- 首次到访：根须缝隙透出好奇的发现
- 冲突阶段：根须缝隙透出扭曲的真相
- 终局阶段：根须缝隙透出选择与道别

**参考镜头范例**
```
镜头 A：初期
- 摄影机位：根系球形头部附近，距地面1.2米
- 透过根须网络向上看，见树冠光斑
- 人物小身影出现在光斑中
- 音效：自然风声 + 微弱低频

镜头 B：高潮期
- 摄影机位：中央根系缝隙内
- 透过根须看向四周，景象扭曲、多重曝光
- 阴间的光线与地上的光线同时可见
- 音效：混乱的声音层次，回音错位

镜头 C：终局
- 摄影机位：低角度，根系缝隙边缘
- 透过最后的根须看向外界，逐渐清晰
- 光线从深紫色逐步转变为金黄色
- 音效：声景从混乱逐步转向单一、寂静
```

### 标志性构图二：俯瞰根系全貌视角

**视觉特征**

- 景深：深景深，展现完整空间

**象征意义**

- 整体观与细节的统一
- 人类的渺小与自然的伟大

**应用场景**
- 故事开篇：介绍场景的宏大感

- 音效：低频共鸣，如同心跳
