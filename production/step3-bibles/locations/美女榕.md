## 三、空间结构与地理

### 俯瞰布局

```
                        牛台山林间
                      树冠覆盖区域
                     （约150-200㎡）

                  ┌───────────────┐
                  │               │
                  │     树冠       │
              ┌───┤  气根垂地    ├───┐
              │   │   枝叶茂密   │   │
              │   │   向四方伸   │   │
              │   │   展         │   │
              │   └───┬───────┬──┘   │
              │       │树干   │       │
              │       │（粗）│       │
              │       │       │       │
              └───────┼───────┼───────┘
                      │根部   │
                      │系统   │
                      │（地下│
                      │数百米）
                      │       │
                   地下阴间
            （十八层塔 → 恶灵区）
```

### 纵深与层次

- **前景**：树冠下方的地面，铺满枯叶、青苔与湿润的黄土；气根垂地，形成天然的屏障与通道
- **中景**：粗壮的树干（直径可达3-5米），褐色树皮沟壑纵横，苔绿与地衣覆盖其表面；树根从树干基部向四方伸展，形成复杂的根系网络
- **远景**：树冠上方的林隙，可见天空和周围山林；树冠与周围树木的交界处，暗示美女榕在这片山林中的独特地位

### 关键地点

**树冠区**：枝叶繁茂，密集的绿色叶片形成天然的顶篷，遮挡大部分天光，内部昏暗而氤氲。是灵魂进出现实世界的入口点。

**树干中段**：粗壮而古老，树皮上的沟壑与裂纹记录了数百年的岁月。树干表面的湿润与青苔生长显示其深处与地下水系的连通。

**根部膨胀区**：树根在地表附近的膨胀、扭曲，形似仰卧女性的身体轮廓——这是美女榕名称的源头。根须向下延伸，逐渐隐没于黄土之中。

**地下根系通道**：树根向下延伸至地下数百米，与十八层塔结构相交，形成有机的、曲折的通道网络。是阴阳两界的物理连接。

**根系与塔的交界处**：树根网络的最深处与十八层塔基部相合，这是阴阳世界的临界点，既属于自然生态，也属于超自然空间。

### 出入口与通道

1. **树冠入口**——灵魂从地下破出，进入树冠；同时也是灵魂进入人间的最后一道关卡
2. **树根通道**——从树干基部的根系网络向下，连接地下的阴间世界
3. **树根缝隙**——老吴等人物可以沿着树根间隙在根系中行走或攀爬（如第3章的场景）

### 标志物/视觉锚点

- **树干的粗壮感**——横亘于山林中的视觉标识，直径可达3-5米，在远处即可辨认
- **下垂的气根与榕树独特的根系形态**——与周围普通树木的明显区别
- **树冠的蓬勃与黑暗**——从远处看是山林中最密集的绿色区域；走入其中则陷入昏暗
- **根部的女性人形轮廓**——根须扭曲形似仰卧女性，是故事中"美女"形象的视觉来源

## 四、光影与氛围

### 基础光影方案

- **主光源**：自然日光，但因树冠密集而大幅减弱；在树冠内部，主要光源来自树冠顶部的林隙散射光，呈现碎光斑效果
- **色温范围**：树冠外 5200K-5800K（自然日光）；树冠内部 3200K-4200K（散射冷绿光+林间昏暗）
- **辅助光源**：树根与地下空间中，超自然光源为主（灵魂紫光、泉水蓝光等）
- **阴影特征**：树冠内部产生柔软、弥散的大面积阴影；树根区域的阴影极深，仅由超自然光源照亮
- **对比度**：中等至低，树冠内部亮部与暗部差异不大；地下根系区域对比度极高（紫光与黑暗的对比）

### 色彩基调

| 色名 | HEX | 用途 |
|------|-----|------|
| 苔绿 | #4A6741 | 树冠、叶片、苔藓的主色调，代表生机与古老 |
| 焦褐 | #2C1810 | 树干、树皮、老化树木的表面，代表时间痕迹 |
| 暗紫 | #5B3A6B | 灵魂紫光通过时的光影映照，代表超自然能量 |
| 土黄 | #8B7355 | 地下黄土、根部周围的泥土色 |
| 深蓝黑 | #1A1A2E | 地下深处、阴间的背景色，代表无光的深渊 |

### 氛围参数

- **空气质感**：树冠下潮湿沉闷，充满林间腐叶的味道与湿润的苔藓气息；地下根系则是冷而干的矿物味
- **粒子**：树冠内林隙处有明显的尘埃粒子在光斑中飘浮；地下无自然粒子，仅超自然光源照亮的根纤维
- **湿度感**：树冠下极端潮湿，树皮与根部都带有露水与青苔；地下则呈现一种被古老树根吸干的干涩感
- **整体氛围**：神圣与诡异的混杂——地表上方是生命的象征，地表下方是死亡的入口；树木本身既容纳生机也容纳死亡

## 五、声景与感官

- **环境底噪**：树冠上层的鸟鸣与风吹树叶的沙沙声；树冠中层逐渐减弱为沉闷的林间底噪；根部区域则是诡异的沉默，仅远处若隐若现的塔内嘈杂（灵魂呐喊、铁链摩擦）
- **标志性声音**：树根摩擦的干涸吱呀声；灵魂通过时的紫光产生的电子音脉冲；树根与土壤之间的空心共鸣
- **触觉/温度/嗅觉**：树皮粗糙而湿冷，苔藓柔软；地下的树根坚硬而干冷；树冠下的空气腐蚀感强（老化植物、湿土的味道）；地下则是矿物与古老有机物的混合气味——既不腐烂也不新鲜，而是超越时间的古老气息

## 六、场景演变轨迹

美女榕在故事中作为阴阳通道的容纳体，其本身的物理结构保持不变。树体从第2章首次出现到第71章的末次出现，建筑完整度保持恒定——既不被摧毁，也不被改造。

树体的"变化"体现在其所承载的超自然活动的增减：早期（ch2-ch10）朱围庸首次通过树根逃脱，树体呈现灵魂出入的超自然迹象；中期（ch11-ch40）树根通道保持活跃但相对平静；后期（ch41-ch71）随着故事高潮的临近，灵魂活动频繁，树体在视觉上呈现更频繁的紫光脉动与光影变化。但这些变化属于叙事与情绪层面，不属于物理层面的改变。

## 七、镜头预案

### 标志性构图

- **从下仰拍树冠**：镜头低机位，向上仰视树冠的蓬勃与庞大，天空透过树冠的林隙可见，形成壮观的绿色穹顶——展现美女榕作为空间标志的视觉冲击力
- **树根女性轮廓俯视图**：从高处俯视树根部分，清晰呈现树根扭曲形似仰卧女性的轮廓，根须交叉形成衣褶与身体曲线——是故事中"美女"形象的关键视角
- **灵魂通过树干破出**：镜头在树干前方的侧位，紫色光球从树干内部或树冠处破出，冲向画面——展现阴阳转换的超自然时刻
- **地下根系迷宫穿梭**：镜头在地下，沿着树根通道跟拍紫光高速移动，根系在紫光映照下显出棕色与土黄色的纹理——强调空间的古老与有机性

### 空间限制与注意事项

- 树冠内部光线极弱，机位在树冠深处时需保证画面可见度，建议补充超自然光源或调高亮度以维持可读性
- 树根区域为弯曲、不规则的有机形态，镜头运动需要跟随根系的曲线，避免生硬的直线推进
- 树干的粗壮性限制了机位选择，建议使用超低机位（地面高度）或超高机位（无人机）来突出其体量
- 树根与地下空间的转换需要清晰的视觉界线，建议在树根进入塔基的交界处设置过渡镜头
- AI 生成时需强调树根的有机纹理与湿润感，避免生成过于规则或塑料感的树根形态
