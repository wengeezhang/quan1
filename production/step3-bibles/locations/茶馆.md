## 三、空间结构与地理

### 俯瞰布局

```
        街道（北侧入口）
             ↓
    ┌──────────────┐
    │ 柜台  账本  │
    │             │
    │  茶桌  椅  │
    │  1    1    │
    │             │
    │ 后房（阁楼）│
    └──────────────┘
        厨房（后）
           ↓
```

### 纵深与层次

- **前景**：茶桌、茶杯、小炉子
- **中景**：柜台、账本、油灯
- **远景**：后房帘子、阁楼入口

### 关键地点

**茶桌区**：靠近街道的几张矮木桌，配以竹椅，是低声密谈的场所。

**柜台**：老板娘所在，账本与茶叶罐摆放。

**后房/阁楼**：隐蔽的上层空间，郝大川与小顶密谈的最终地点。

### 出入口与通道

1. **街道正门**——连接镇中心街道
2. **厨房后门**——通往镇内其他地方
3. **阁楼梯**——通往后房隐蔽空间

### 标志物/视觉锚点

- **油灯与烛火**——昏暗的光源，见证秘密对话
- **茶杯与茶叶**——茶馆身份的标识
- **木制家具的年代感**——陈旧与沉默的见证者

## 四、光影与氛围

### 基础光影方案

- **主光源**：油灯与蜡烛，点光源，强度弱，颜色偏黄（3200K）
- **色温范围**：3000K-3500K，烛光暖黄
- **阴影特征**：极软的烛火阴影，摇曳不定，方向不稳定
- **氛围参数**：烟火气浓厚，茶香弥漫，光线不足产生压抑感，暗角明显

## 五、声景与感官

- **环境底噪**：隐约的街道喧闹声隔绝在外，只有轻微的风声
- **标志性声音**：茶杯轻轻碰撞、小声耳语、老板娘脚步声、油灯的细微嘶嘶声
- **触觉/温度/嗅觉**：温暖的室内温度、陶瓷茶杯的温热、茶叶与烟火的混合气味、老木材的陈旧气息

## 七、镜头预案

### 标志性构图

- **烛光特写**：近距离拍摄茶杯与人物面部半隐在烛光中，强调秘密与恐惧
- **后房阁楼**：低机位看向阁楼梯，暗示上升与逃离
- **茶杯碰撞**：极近景拍摄茶杯碰撞与茶水涟漪，暗示内心的战栗

### 空间限制与注意事项

- 室内极暗，主要光源为烛火，曝光控制极为困难
- 烛火摇曳，画面会出现光线不稳定，可考虑稳定曝光或接纳微弱闪烁
- 木制家具容易产生过度暖色，需要色温平衡
