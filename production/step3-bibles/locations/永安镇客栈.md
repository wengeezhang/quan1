## 三、空间结构与地理

### 俯瞰布局

```
              主入口（朝向市集广场）
                      ↓
        ┌──────────────────────┐
        │   一楼大堂/柜台        │
        │   (收银、休息区)       │
        └───┬────────────┬──────┘
            │            │
       楼梯  │    走廊    │ 楼梯
            │            │
    ┌───────┴────────┬──┴──────┐
    │  二楼走廊        │ 三楼走廊 │
    │  (5-6间房)      │(3-4间房)│
    │  朱围庸 宋小仙   │         │
    │  相邻！          │         │
    └─────────────────┴─────────┘
```

### 纵深与层次

- **前景**：客栈大门、门框石板步、街道与市集摊位的边界
- **中景**：一楼大堂、服务柜台、楼梯入口
- **远景**：二楼走廊深处、三楼轮廓

### 关键地点

**一楼大堂**：客栈的接待区，约15×10米，一侧放置简陋的柜台与账本，柜台后放有房钥匙架。地面为不规则的木板或石板。大厅内放置几张木凳供客人等待。内部光线昏暗，仅靠门窗与油灯照亮。

**楼梯**：从大堂通往二、三楼的木制楼梯，较窄（约1.2米宽），踏步磨损明显，上行时会吱呀吱呀作响。

**走廊**：二、三楼的狭长通道，两侧是客房门，每扇门都简陋地钉着数字房号。走廊采光依靠端部窗户，昏暗压抑。

**客房**：约12-15平方米的小房间，内设简单行军床或木架床、水盆架、一张简陋的小桌。墙壁粗糙，门框不甚紧密，隔音很差——隔壁房间的声息清晰可闻。

### 出入口与通道

1. **主大门**——朝向市集广场，是唯一的主要出入口
2. **二楼窗户**——可能的应急逃脱（故事中朱围庸可能考虑但未使用）
3. **楼梯**——连接各层的唯一通道

### 标志物/视觉锚点

- **简陋的柜台**——旅馆的商业标识
- **房间数字门牌**——最廉价房间的标记（如"105"号）
- **陈旧的木楼梯**——客栈年代感的象征

## 四、光影与氛围

### 基础光影方案

- **主光源**：日光通过一楼大门与二楼走廊端部窗户入射，但传播到走廊深处已极弱
- **色温范围**：门窗附近 5200K-5800K（自然日光），走廊深处 3500K-4500K（日光衰减+油灯补光）
- **辅助光源**：油灯或烛火放在大堂柜台与走廊墙上
- **阴影特征**：楼梯与走廊深处阴影深且压抑，房间门框投射清晰阴影
- **对比度**：低至中等，亮部（门窗附近）与暗部（走廊深处）反差大

### 色彩基调

| 色名 | HEX | 用途 |
|------|-----|------|
| 焦褐 | #2C1810 | 老旧的木制楼梯、房门、家具 |
| 灰褐 | #5C4033 | 墙壁粉刷、走廊地板 |
| 深蓝灰 | #3A4A6B | 走廊深处的压抑暗色 |

### 氛围参数

- **空气质感**：室内空气滞闷，带有陈旧木料、霉湿与油灯烟雾的混合气味
- **粒子**：油灯烟雾在走廊飘散，日光中可见尘埃飘浮
- **整体氛围**：压抑、廉价、危险伏伏、隔音差的紧张感

## 五、声景与感官

- **环境底噪**：市集广场的嘈杂声透过墙壁传入；隔壁房间的声息清晰可闻（人声、脚步、水声）
- **标志性声音**：楼梯吱呀吱呀的踏步声、房门吱呀呀的开合声、柜台后的钥匙碰撞声
- **触觉/温度/嗅觉**：粗糙的木制扶手、潮湿阴冷的走廊空气、霉味与烟味混合、脚踩木板的声音反馈

## 六、场景演变轨迹

永安镇客栈在 ch10 仅出现一次，建筑结构保持完好。朱围庸在此的遭遇（发现宋小仙）是叙事事件，不改变建筑本身的视觉状态。客栈作为"廉价、压抑、隔音差"的空间，这些特性在分镜层面渲染（如宋小仙房间内的声息、朱围庸的听觉恐惧）。

## 七、镜头预案

### 标志性构图

- **走廊纵深**：从二楼走廊的起点向深处拍摄，两侧房门沿走廊逐渐消失于黑暗，强调压抑与孤立
- **柜台角度**：从客人进门的视角拍摄，大堂柜台与背后的钥匙架、油灯，昏暗中闪烁的微弱光
- **楼梯仰拍**：从一楼向上仰拍楼梯，踏步与扶手的质感，隐喻向上但艰难的逃脱

### 空间限制与注意事项

- 走廊狭窄（1.2米），不适合多人并排通行
- 内部采光严重不足，夜间场景需依靠油灯与烛火
- 楼梯转角急，机位受限
- 隔音差的设定需在声景层面处理，视觉上难以表现
