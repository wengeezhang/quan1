## 二、外貌与形态

### 基础外貌

- **身材体型**：瘦高书生形象，肩膀偏窄，手臂修长，整体显得纤弱但不失力度感。姿态长期伏案导致脊椎微微前倾，在实验室中更为明显。
- **脸型**：长脸偏瘦，颧骨略高，下颌线条棱角分明，面相显得年轻但眼神沧桑——知识分子的典型特征。
- **眉毛**：深色浓眉，眉间常有皱纹，思考时两眉紧皱成"川"字状。
- **眼睛**：深色眼眸，眼窝略深，被黑框眼镜隐藏了大部分表情。泉水充足时眼神清亮冷静；伦理觉醒期眼神开始浮现困顿和矛盾；危机时眼镜反光遮挡眼神成为视觉隐喻。最大特征是眼镜这道"屏障"——镜片反光时隐藏眼神的真实，这是他内心冷漠的视觉表现。
- **鼻子**：直挺鼻子，鼻梁高，与瘦长脸型相匹配，显出理性严肃的气质。
- **嘴唇**：薄唇，说话时嘴角几乎不动，给人冷漠疏离之感；在最终危机时可能出现轻微的嘴角抖动——理性开始破裂的第一个信号。
- **皮肤**：偏白皙，长期在室内实验室工作缺乏日晒，皮肤显出病态的洁白。面部几乎无皱纹（除了眉间和眼角），显得皮肤很"冷"。
- **发型**：深色短发，梳理得整齐贴头皮，发质显示长期缺乏护理但保持整洁——对外表的忽视源自对技术的专注。
- **标志性微动作**：推眼镜（思考或说出重要数据时的条件反射）；掐指尖计算（在心中快速演算复杂公式）；单手托腮（陷入深度思考时）；双手交叠于胸前或实验台上（防御性姿态，暗示内心的自卫机制）。

### 单一着装与气质

**衣着**：标准白色实验服，干净整洁，口袋中插着笔或记录本。黑框眼镜是身体的一部分——即使在室内也不摘下。可能戴着一条简约手表（科研工作者的标准配置）。冬季可能套一件简素毛衣或夹克在实验服外。

**气质演变轨迹**：
- **项目盛期**：冷静、理性、科学至上、与人保持距离。面容清冷理性，眼神专注如激光射向前方的屏幕或数据表。眼镜镜片反光频繁，成为隐藏眼神的"面具"。与导师吴启思互动时眼中闪过一丝尊敬与共谋的光芒，但此刻理性完全压倒良知——他的脸上看不到丝毫道德的阴影。
- **伦理觉醒期**：理性的裂缝、内在的不安、压抑中的疲惫。面容仍保持冷静但眼角开始有疲惫的阴影。推眼镜的动作频率明显增加——每次推眼镜都成为一个"暂停"的信号，暗示他在重新组织思维或压抑内心的不安。眼神在某些时刻会短暂地显出迷茫，但随即被理性的防御重新覆盖。独处时姿态可能会松散下来，微微驼背，显出精神上的疲惫。白色实验服仍未改变，但可能出现皱褶或沾染痕迹（长期工作未及休息的标志）。眼镜可能沾上指纹或灰尘，显示他对外表的进一步忽视。
- **道德崩裂期**：理性的瓦解、人性的苏醒、极端的困境。面容首次显出明显的情感波动——眉间的皱纹加深，嘴角可能微微抖动。眼睛在最关键时刻可能短暂脱离眼镜的遮挡（眼镜脱落或他下意识地摘下），这是整个角色弧线中最脆弱的一刻。肤色可能显出疲惫的苍白，眼窝的阴影加深。站立时身体可能首次出现不稳定的迹象——腿部微弱或靠向墙壁/实验台。实验服可能显得更加凌乱，袖子可能卷起（表示长期工作的紧迫感）。眼镜可能松动或被推到头顶。

## 三、行为模式与标志性动作

### 3.1 冷静的理性机器

于永智的核心行为特征是**完全被逻辑支配**。他说话前总是短暂停顿，组织最精确的表达；每句话都经过计算，像代码一样严谨。在团队中，他用数据和逻辑而非情感说话——即使在传达坏消息时，语调也保持平稳如镜面般的水面。走进实验室，他的第一个动作是环顾四周检查所有仪器的运行状态，动作有条不紊，没有任何多余的手势。

### 3.2 与实验数据的亲密关系

于永智与屏幕的互动远多于与人的互动。他长时间凝视数据、代码或全息投影，眼神透过镜片射向虚拟世界。手指在键盘或触摸屏上跳动时显出精细的肌肉控制，但同时也显出一种机器般的重复性。偶尔会出现"掐指计算"的动作——虽然大多数时间他是在做心算，但这种身体语言暗示他在不断地在脑海中推演复杂的公式和算法。

### 3.3 "推眼镜"——理性的仪式

推眼镜是于永智最常见的微动作，也是其心理状态的直接反映。当他做出这个动作时，通常代表以下几种心理状态：(1) 深度思考——眼睛离开屏幕，右手的食指和中指推动眼镜向上；(2) 说出重要数据——在关键信息处，他会习惯性地推眼镜，仿佛在重新对焦；(3) 压抑不安——在伦理觉醒期，这个动作的频率剧增，成为他压制内心动摇的"癖好"。

### 3.4 语言的精准与冷漠

对话中，于永智不使用口头禅或感叹词。他的句子结构完整，充满专业术语，但这些术语通常会被他自己解释（这是他理性的表现——确保信息的完整传递）。在导师吴启思面前，说话会更加谨慎，带有学生的尊敬。对兄长于永龙，他用冷淡的语气，显出对实用主义的贬低。对弟弟于崇宝，他用防御性的逻辑去击碎对方的情感论点。

### 3.5 身体的疲惫信号

长期投入项目导致于永智的身体呈现出过度工作的迹象。他经常在办公椅上坐数小时不动，直到需要起身时才意识到腿部僵硬。食物对他而言是补给而非享受——他常常忘记吃饭，只在感到明显饥饿时才匆匆进食。在最终危机时期（ch63），他可能出现明显的生理信号——手指颤抖、眼神散焦、呼吸不稳——这些是理性防线即将崩溃的身体证据。

**一句话弧线**：从完全的理性执行→理性的质疑→理性的瓦解——于永智的人生是**科学主义信仰的坍塌与人性的惊人苏醒**。

### 关键阶段详解

**项目盛期（ch61-ch62前）**

- **核心特质**：被理性完全支配的执行者——科学信仰的狂热信徒
- **行为表现**：ch61初登场时立即展示对项目的全身心投入，用数据和逻辑与导师沟通；向团队成员解释技术细节时娓娓道来，仿佛数字人是完全合理的科学成就；对项目可能带来的伦理后果视而不见，或用"科学无价值判断"的论点予以驳斥。
- **内心驱动**：对技术完美的追求、对导师的崇拜、对自身能力的自信。他相信科学是中立的，伦理问题不属于技术领域的考量。
- **视觉状态**：清冷苍白的皮肤、锐利的眼神被眼镜完全遮挡、整洁但冰冷的白色实验服——整体呈现一个"完美的机器"形象。光线多为实验室的冷白色LED，强烈的高对比度映照出他的冷漠。

**伦理觉醒期（ch62中-ch62后）**

- **核心特质**：理性的第一次动摇——良知从地底下苏醒的声音
- **行为表现**：开始在关键时刻停顿更久，推眼镜的频率上升；用逻辑去论证项目的合理性，但论证过程中逻辑链开始出现漏洞；独处时可能长时间凝视屏幕但脑海中在反复一个问题而非计算；与导师的互动中，尊敬开始掺入一丝不易察觉的质疑；可能主动向弟弟于崇宝询问某些他自己也想问的道德问题，但用冷淡的语气去"驳斥"对方的答案。
- **内心驱动**：理性与良知的第一次交锋。他不是突然转变，而是被逻辑的缝隙折磨——数据无法完全解释伦理困境。
- **视觉状态**：眼镜反光增多成为心理不安的视觉映射；推眼镜动作成为"重新确认理性"的仪式；独处时姿态松散，显出精神疲惫；肤色可能微微泛黄（长期睡眠不足的迹象）。

**道德崩裂期（ch63）**

- **核心特质**：理性堡垒的最后瓦解——人性的极端觉醒
- **行为表现**：在面对最终的道德抉择时，他无法再使用逻辑去自我欺骗。可能会爆发——打破沉默用吼声发泄，或陷入极度的寂静而无法言语。眼镜可能被脱下、摘掉或脱落，这是整个角色弧线中最重要的视觉转折：眼镜这层"理性的假面"被移除，露出真实的眼神——可能是恐惧、后悔、绝望或决绝。
- **内心驱动**：现实击穿了理论。他无法再活在数据的世界中，必须面对人性的重量。
- **视觉状态**：最脆弱的一幕——脱掉眼镜后显出完整的脸孔，眼神首次完全暴露，可能出现眼泪（这对于他来说是绝对的突破）；白色实验服显得如同诘问的法官的衣袍；身体可能显出不稳定，需要靠向实验台或墙壁来支撑自己。

### 关键转折点

1. **ch61 初登场** ——理性权力的顶峰，全身心投入项目
2. **ch62中 第一次质疑** ——推眼镜频率上升，逻辑第一次出现缝隙
3. **ch62后 与弟弟的对话** ——用冷淡语气问出了自己心底的问题
4. **ch63 终极抉择时刻** ——眼镜脱落，理性完全瓦解，人性苏醒

### 视觉叙事线索：理性与人性的对抗

```
眼镜完全遮挡(ch61) → 眼镜反光暗示不安(ch62) → 眼镜脱落露出真眼(ch63)

白色冷光(ch61) → 白色光线中的阴影加重(ch62) → 逆光中的剪影(ch63)

整洁实验服(ch61) → 微微皱褶的实验服(ch62) → 凌乱沾污的实验服(ch63)
```

于永智的视觉签名是**眼镜与眼神的关系变化**。眼镜从"完美理性的象征"逐渐变成"压抑人性的枷锁"，最终在ch63被移除——这一刻，理性的假面彻底破裂，人性的真实显露。

## 五、人物关系

### 与吴启思（导师）

**关系定位**：崇拜与共谋的师徒关系——由学生→共谋者→最终的良知对抗

于永智对吴启思的尊敬源自深深的崇拜。导师代表了他所追求的一切——智慧、权力、科学的绝对性。在项目初期，于永智是吴启思意志的完美执行者。但伦理觉醒期开始，他在导师身上看到了自己逐渐看清的东西——对伦理的视而不见、对权力的执着、对人性的漠视。ch63时两人可能面临最后的对峙：于永智用曾经学到的理性逻辑去质疑导师的理性。这是整个故事中最具张力的关系之一。

### 与于永龙（兄长）

**关系定位**：理解的缺失与分化的痛楚——工程师vs理论家

兄长于永龙是务实的工程师，而于永智是沉溺理论的学者。两人在项目期间的隔阂加深。永龙能感受到永智的变化但无法进入他的思维世界；永智则对永龙的"非学术"态度显出冷漠与贬低。他们有过多次无效的沟通，在同一个家族但生活在不同的星球。

### 与于崇宝（弟弟）

**关系定位**：精神救赎与理性防御的碰撞——温暖vs冷漠

弟弟试图用精神与情感去拯救永智，但永智用冷淡的逻辑去击碎弟弟的每一个论点。然而，在伦理觉醒期，于永智可能主动向弟弟提出某些问题——这不是认可，而是他用冷淡面具包裹的求救信号。弟弟的温暖最终可能成为他在崩裂时刻的唯一救赎。

### 与数字人

**关系定位**：创造者与创造物之间模糊的边界——工具vs生命

于永智将数字人视为完美的技术成就，是科学的胜利。但随着项目推进，他无法回避一个问题：这是否是生命？在伦理觉醒期，他可能开始看到数字人不仅是代码，而是某种他创造出的"存在"。这种认识会加速他理性防线的崩溃。

## 七、连贯性约束

### 视觉标识

**不变的**：
- 黑框眼镜——全程标志，即使在最终危机时也会存在（可脱落或推至头顶）
- 瘦高书生体型——任何阶段不可改为粗壮或虚弱
- 深色短发——整洁贴头皮的发型风格保持一致
- 苍白皮肤——长期室内工作的标志，保持一致的病态洁白

**可变的**：
- 眼镜位置从完全遮挡→偶尔脱落→最终脱掉
- 实验服从整洁→微皱→凌乱
- 肤色可从苍白微微偏黄（疲惫）→在最终危机时可能微微泛红（情绪激动）
- 眼神从被眼镜遮挡→开始透露不安→完全暴露真实

### 行为模式

**不变的**：
- 推眼镜的习惯性动作——从第一场到最后一场（最终时刻可能动作停止）
- 说话前的停顿——思维的精确性保持一致
- 对理性/逻辑的执着——即使在崩裂时仍然试图用逻辑解释

**可变的**：
- 推眼镜的频率从偶尔→频繁→停止
- 说话语速从稳定→加快或急促→可能出现颤抖
- 身体姿态从挺拔→微微前倾→可能需要支撑

### 情绪弧线

**不变的**：
- 于永智本质上是理性压倒一切——即使在最后关头，他的冲突也是理性vs人性的冲突，而非道德vs邪恶
- 他的转变不是背叛，而是发现——发现理性的边界
- 与弟弟的互动中隐含着他渴望理解但无法表达的情感

**可变的**：
