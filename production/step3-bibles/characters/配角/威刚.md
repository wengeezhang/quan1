## Ch3 外貌与形态

### 形态概述

威刚拥有两种截然不同的视觉形态：**人类仆人形态**和**元神巨人形态**。两种形态的极端反差是角色的核心视觉设计。

### 形态一：人类仆人形态（ch66-ch68前半）

**面部特征**：
- 中年男性面容，约35-40岁外貌（实际年龄不可估量）
- 面部线条清瘦但不枯槁，颧骨微微突出
- 眼睛：平时看似普通的深色瞳孔，但偶尔在特定光线下闪过一丝不属于凡人的深邃——这是唯一暴露超凡本质的细节
- 表情：平静内敛，不卑不亢，但绝不讨好——仆人的身份是伪装而非本性
- 皮肤偏白净（非常年户外劳作的质感，暗示其非普通仆人）

**体型**：
- 身高约180cm，瘦削精干（"瘦削精干"是原文用词）
- 身体比例匀称，四肢修长，关节灵活
- 肌肉不明显但实际力量远超外表——属于"看着瘦实际极强"的类型
- 动作轻盈无声，行走时几乎不发出脚步声

**服饰**：
- 深色朴素仆人衣物（深蓝或黑色短褐）
- 衣物整洁但无任何装饰
- 束腰带，便于行动
- 布鞋或草鞋，无声移动

### 形态二：元神巨人形态（ch68后半-ch70）

**整体轮廓**：
- 高度：20-30米以上，超越城墙高度（城墙约50米，巨人可与之平视或更高）
- 保持人类身体比例但放大至巨型
- **半透明质感**：身体并非完全实体，而是半透明的发光状态，可以透过其身体隐约看到背景
- **发光属性**：全身散发明亮白光，如同一座行走的灯塔

**颜色与光效**：
- 主体：温暖白光（#FFF8DC 暖白，非冷白），从体内向外发散
- 轮廓边缘：更亮的白光光晕，边界模糊
- 眼睛：**蓝光瞳孔**（#4169E1 皇家蓝），在黑暗环境中如两盏蓝色探照灯
- 头顶：变身瞬间从头顶冠部爆发白光（变身触发点）

**质感**：
- 非实体——"光即材质"（引用art_direction天界设定）
- 表面如同凝固的光线，有微弱的光波流动感
- 不反射外部光线，而是自身发光
- 在水面/地面留下光的倒影/投射

**特殊物品**：
- **分割带**：戴在手腕的银白色环状物（#C0C0C0），平时为装饰，战斗时可弹射为精准切割武器
- **泡泡**：从口中吹出的巨型半透明蓝色球体，内部发光，表面折射彩虹光

### 变身过程视觉序列

```
1. 闭眼（人类形态静止）
2. 头顶冠部白光爆发（变身触发）
3. 身体开始膨胀/发光
4. 周围水面/地面被光照亮
5. 衣物消融于光中（非撕裂，而是被光取代）
6. 完成巨人形态（半透明+自发光）
```

### 两种形态视觉对比

| 视觉要素 | 仆人形态 | 元神巨人形态 |
|----------|---------|-------------|
| 身高 | ~180cm | 20-30米+ |
| 体质 | 实体，瘦削精干 | 半透明，发光 |
| 颜色 | 深色衣物，正常肤色 | 温暖白光全身 |
| 眼睛 | 深色普通瞳孔（偶闪异光） | 蓝光瞳孔如探照灯 |
| 动作 | 轻盈无声 | 缓慢庄严（体量巨大） |
| 存在感 | 极低（仆人伪装） | 极强（照亮战场） |

## Ch4 能力与视觉表现

### 核心能力

**1. 飞壁（ch68）**
威刚能以极速垂直奔跑于50米高的城墙表面，如同地面是水平的一样。
*视觉表现*：月光下，一个人形剪影以近乎垂直的角度贴着石壁飞速上升，双脚触壁点扬起石粉但完全无声。城墙守卫毫无察觉。整个过程如同一只在墙面奔跑的壁虎的人形放大版。

**2. 蓝光视觉（ch68）**
在水下或黑暗环境中，威刚双眼发出蓝色光芒照亮周围，如两盏水下探照灯。
*视觉表现*：血红色浑浊水域中，两道锐利的蓝色光柱（#4169E1）穿透混沌，照亮隐藏在水下的机关和悬梯。蓝光在水中产生散射和焦散效果。

**3. 巨人变身（ch68）**
从人类体型膨胀为20-30米高的半透明发光巨人。
*视觉表现*：闭眼→头顶白光爆发→身体如充气般膨胀→水面被冲破→巨人从水中升起，高度超越城墙→全身散发温暖白光，如一座行走的灯塔。箭矢射来完全无效，如投入光中便消融。

**4. 泡泡囚笼（ch69）**
从口中吹出巨型半透明蓝色球体，可包裹目标形成无法逃脱的囚笼。
*视觉表现*：巨人张口，一颗巨大的半透明蓝色球体缓缓吹出，球体表面折射周围景色产生彩虹光效。灵尤被包裹其中后疯狂挣扎但球壁完全不变形——绝对的降维碾压。

**5. 分割带（ch69）**
手腕上的银白色环状物可弹射为精准切割武器。
*视觉表现*：威刚拇指一弹手腕上的银白色环带（#C0C0C0），银光如一条直线飞射而出，精准切入泡泡将其一分为二——一半爆裂（朱围庸安全落地），一半收缩困住灵尤。

### 能力视觉规范

| 能力 | 视觉表现形式 | 光效色彩 | 禁忌 |
|------|------------|---------|------|
| 飞壁 | 月光剪影+石粉轨迹+无声 | 无光效（仅月光） | 不加速度线或残影 |
| 蓝光视觉 | 双眼蓝色光柱+水中焦散 | #4169E1 皇家蓝 | 不全脸发光，仅眼睛 |
| 巨人变身 | 头顶爆光+膨胀+半透明体 | #FFF8DC 暖白 | 不使用闪电/雷鸣等西方变身特效 |
| 泡泡囚笼 | 半透明蓝色球体+彩虹折射 | 淡蓝+彩虹 | 不使用铁笼/锁链等物理囚禁意象 |
| 分割带 | 银光直线弹射+精准切割 | #C0C0C0 银白 | 不使用激光或能量束视觉 |

## Ch5 角色发展轨迹

### 全局弧线

```
潜伏仆人期         神力初显期          元神降世期
[隐藏的天神] ──→ [显露的力量] ──→ [降临的审判]
  ch66-ch67          ch68              ch69-ch70
```

### 各阶段详细轨迹

**潜伏仆人期（ch66-ch67）**
*情感基调*：平静内敛，超然旁观
*关键事件*：威刚以仆人身份潜入杨弘控制的城池进行侦察。穿着朴素仆人衣物，混迹于城中而不引起注意。行动无声高效，收集城防情报，为后续攻城做准备。
*视觉转变标记*：威刚混在普通仆人中几乎不可辨认→偶尔在特殊光线下眼中闪过一丝不属于凡人的光芒

**神力初显期（ch68）**
*情感基调*：从隐忍到释放，如弦断般的力量爆发
*关键事件*：
- 夜间飞壁攀越50米城墙，月光下无声剪影
- 潜入深沟水下，蓝光瞳孔在血红水域中探索，发现隐藏悬梯
- 首次变身：闭眼→头顶白光→身体膨胀→从水中升起超越城墙的发光巨人→举起巨型悬梯为攻城部队开路
- 敌军箭矢对巨人形态完全无效
*视觉转变标记*：人类剪影→蓝光双眼→白光爆发→半透明巨人。这是全片最震撼的视觉变身之一。

**元神降世期（ch69-ch70）**
*情感基调*：天界力量的碾压性降临，不是愤怒而是秩序的执行
*关键事件*：
- 灵尤杀死胡酹后追击朱围庸
- 威刚以巨人形态口吹泡泡将灵尤和朱围庸同时包裹
- 以分割带精准将泡泡一分为二——救出朱围庸，囚禁灵尤
- 全面展现元神级别对凡间力量（包括神兽级别）的绝对碾压
*视觉转变标记*：巨人形态从ch68的被动救援→ch69的主动出击和精准控制

## Ch6 人物关系

### 关系网络

```
    上神界（天界秩序）
         │ 派遣
         ▼
       威刚（元神执法者）
       ┌─┴──────────┐
       │             │
  朱围庸          灵尤
 （守护对象）    （囚禁对象）
                     │ 秘密武器
                  杨弘
               （间接被碾压者）
```

### 核心关系

**威刚 → 朱围庸**：守护者→效力对象
威刚以仆人身份服务于朱围庸阵营，在ch69中冒死救出被灵尤追击的朱围庸。这种关系不是简单的主仆，而是天界秩序对朱围庸使命的认可和支持。威刚的存在证明了朱围庸的反抗之路有天命支撑。
*视觉关系语言*：朱围庸在泡泡中安全落向草地 vs 灵尤在泡泡中困兽犹斗——威刚精准的区别对待展现守护者意志。

**威刚 → 灵尤**：高维 vs 低维
灵尤虽为4-5米高的神兽，拥有毁灭性的蛮力，但在元神级别的威刚面前如同凡人。泡泡囚笼将灵尤完全封锁，灵尤的一切挣扎毫无意义。这展现了存在层级的绝对碾压。
*视觉关系语言*：20-30米发光巨人 vs 4-5米白色猛兽——体量和存在维度的双重碾压。

**威刚 → 杨弘**：天界秩序 vs 人类极限
威刚从未与杨弘直接对话，但其超自然力量完全粉碎了杨弘的一切军事部署。黑布阵、大蒜战术、精密防御工事——人类智慧的极致在元神面前如同儿戏。杨弘最终的精神崩溃部分源于见证威刚的力量后意识到人类的渺小。
*视觉关系语言*：从不同框出现，但战场上威刚的光芒映在杨弘震惊恐惧的脸上。

## Ch7 视觉设计

### 色彩系统

**阵营归属**：天界/元神级（最高超自然层级）
**主色调**：暖白 #FFF8DC（元神发光体）+ 皇家蓝 #4169E1（蓝光瞳孔/泡泡）

**辅助色**：

| 色彩 | 色值 | 用途 |
|------|------|------|
| 暖白 | #FFF8DC | 元神巨人形态主体发光色 |
| 皇家蓝 | #4169E1 | 蓝光瞳孔、泡泡囚笼内光 |
| 银白 | #C0C0C0 | 分割带、手腕环饰 |
| 深藏蓝 | #1A1A3E | 仆人衣物色（潜伏期） |
| 暗金 | #DAA520 | 变身瞬间头顶爆发光的金色分量 |
| 彩虹折射 | — | 泡泡表面的彩虹光效（无固定色值） |

### 阶段色彩变化

| 阶段 | 主色调 | 光效 | 整体氛围 |
|------|--------|------|---------|
| 潜伏仆人期 | 深藏蓝 #1A1A3E（衣物） | 无光效，仅月光 | 隐匿、朴素、不引人注目 |
| 神力初显期 | 暖白 #FFF8DC + 皇家蓝 #4169E1 | 蓝光瞳孔→头顶白光爆发→全身发光 | 从隐忍到震撼的释放感 |
| 元神降世期 | 暖白 #FFF8DC 为主 | 全身持续发光+泡泡蓝光+银白分割带 | 天界力量的绝对降临 |

### 视觉签名：存在维度跃迁

威刚的视觉签名是**存在维度的跃迁**——从最不起眼的存在变为最压迫性的存在：
- **潜伏仆人期**：混在人群中几乎不可见——"零存在感"
- **神力初显期**：从零到满的爆发——照亮整个战场的白色巨人——"最大存在感"
- **元神降世期**：稳定的神级存在——半透明发光巨人如同天界法则的具现化——"超越存在感"

### 光影方案

**潜伏仆人期**：无特殊光影处理，威刚处于场景中的暗处或阴影中，仅偶尔被月光或火光扫过时眼中闪过异光。整体暗调，与周围环境融为一体。
**神力初显期**：月光提供基础照明→蓝光瞳孔成为水下唯一光源→头顶白光爆发瞬间全场被照亮→巨人形态本身成为光源，反向照亮周围环境（水面、城墙、士兵的惊恐面部）。
**元神降世期**：威刚本身就是场景主光源。巨人形态在日间战场上的白光与日光叠加，在夜间则独自照亮方圆百米。泡泡的蓝色内光和银白分割带的飞射光线成为辅助光效。

## Ch8 连贯性约束

### 不可变特征（全阶段恒定）

| 特征 | 规范 | 容错范围 |
|------|------|---------|
| 面部轮廓（人类形态） | 清瘦中年面容，颧骨微突 | 巨人形态面部放大但保持比例 |
| 蓝光瞳孔色值 | #4169E1 皇家蓝 | 不同环境下饱和度可微调±10% |
| 元神发光色 | #FFF8DC 暖白 | 不得变为冷白#FFFFFF或其他颜色 |
| 分割带 | 银白色 #C0C0C0 手腕环饰 | 两种形态均佩戴 |
| 身体比例 | 人类标准比例 | 巨人形态等比放大，不变形 |

### 可变特征（随阶段变化）

| 特征 | 变化规则 |
|------|---------|
| 身高 | 仆人期180cm→巨人形态20-30米+ |
| 透明度 | 仆人期完全实体→巨人形态半透明 |
| 发光 | 仆人期不发光→巨人形态全身发光 |
| 衣物 | 仆人期深色短褐→巨人形态衣物融入光中消失 |
| 眼睛 | 仆人期深色正常→战斗时蓝光 |
| 存在感 | 仆人期极低→巨人期照亮战场 |

### 形态切换约束

- **人→巨人变身**必须经过完整视觉序列：闭眼→头顶白光→膨胀→完成。不得省略或瞬间完成
- **巨人形态**下不得出现完全不透明的实体质感——必须保持半透明+发光
- **蓝光瞳孔**可在人类形态和巨人形态中均出现，但仆人期仅在水下/黑暗环境中激活
- **变回人类形态**的过程在原文中未详细描述，可以是光芒逐渐收敛+身体缩小

### 禁忌

- **禁止**使用西方超级英雄式的变身光效（闪电、雷鸣、肌肉暴涨）——应为东方式的"内光外放"
- **禁止**元神形态使用冷色调（冷白、冰蓝）——天界色调为暖白+暗金
- **禁止**元神形态出现完全不透明的实体感——半透明是区分天界存在和凡间存在的关键视觉标记
