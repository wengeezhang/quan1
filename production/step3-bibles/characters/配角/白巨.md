## Ch3 外貌与形态

### 基础形态描述（神兽形态，非人类）

**整体轮廓**：
四足巨兽，站立高度5-6米，体型如小型山丘。身体覆盖约20厘米长的纯白色长毛，毛质浓密柔软，在风中如白色瀑布般波动。面部完全无毛，露出纯黑色皮肤，五官简化但轮廓分明。头顶两只长耳直立，如鹿角般向上延伸。

**面部特征**：
- 面部皮肤：纯黑色，质感如磨砂石，与全身白毛形成极致反差
- 眼睛：深邃而大，瞳孔中映射星光，传递悲悯与古老智慧
- 五官：简化的兽类面部结构——宽阔鼻梁、不明显的嘴部线条，表情主要通过眼睛传达
- 双耳：长而直立，耳廓内侧呈深灰色，可灵活转动感知环境

**身体结构**：
- 四肢粗壮如石柱，前肢略高于后肢，呈现稳固的坡形体态
- 爪部宽大，指甲呈灰黑色，可抓取物品（使用石锅烹饪）
- 尾部短粗，白毛覆盖
- 整体骨架厚重，肌肉线条被长毛覆盖，仅在前肢和颈部隐约可见

**毛发系统**：
- 颜色：纯白色（#F5F5F5），非纯白，带有自然的象牙白光泽
- 长度：约20厘米，全身均匀覆盖
- 质感：丝绸般柔顺，微风中轻柔波动，大风中如旗帜飘扬
- 特殊区域：颈部鬃毛更长更密（约30厘米），形成天然"项圈"效果

**动作特征**：
一切动作极其缓慢，带有仪式感。每一步、每一次转头都显得深思熟虑，如同古庙中的祭祀动作。绝不急躁、绝不惊慌。即使在紧急时刻（如制止灵尤），也仅以最小幅度的力量精准制止，而非暴力对抗。

### 阶段间视觉差异

| 视觉要素 | 族长全家期 | 囚徒启悟期 | 山巅守望期 |
|----------|-----------|-----------|-----------|
| 毛发状态 | 浓密完整，火光映照呈暖金白 | 不变，月光下呈银灰色调 | 被山风吹拂飘动，边缘微乱 |
| 眼神 | 威严慈悲，带有父亲的温暖 | 深邃沉静，映射星光 | 远眺人间的超然审视 |
| 姿态 | 居中端坐，三角构图顶点 | 星空下端坐，与囚徒对视 | 屹立峰顶，如石雕般不动 |
| 面部状态 | 黑色面部在火光中呈琥珀边缘 | 黑色面部在月光中呈银蓝边缘 | 黑色面部在日光中呈纯墨色 |
| 周围环境 | 隐谷、篝火、石锅、家族陪伴 | 雪山星空、铁链囚徒 | 山巅古树、云海、远处人间 |

### 视觉辨识标记（跨阶段不变）

- **黑白极致反差**：纯黑面部 + 纯白长毛，任何光线条件下均一眼可辨
- **双耳直立**：头顶两只高耸长耳，是区别于其他大型动物的关键轮廓特征
- **仪式感动作**：无论何时，动作速度始终极慢，带有远古祭祀般的庄严
- **体量压迫**：5-6米的身高在任何场景中都形成巨大的比例反差

## Ch4 能力与视觉表现

### 核心能力

**1. 超自然体能**
白巨拥有远超凡间生物的力量与耐久力。能以单掌制止灵尤的暴走，能在3000米以上的极寒雪山中自如生存。其力量表现为"克制的压迫"——从不全力出击，仅以必要的最小力量达成目的。
*视觉表现*：制止灵尤时仅微微抬起前掌按压，毛发纹丝不动，体现力量的绝对优势。地面在其脚下微微震动。

**2. 宇宙知识/创世感知**
白巨拥有超越凡人认知的宇宙真理与创世知识。这种知识不通过法术展示，而是通过对话传递。
*视觉表现*：与朱围庸对话时，镜头从两者逐渐拉远至星空全景，暗示其言语所涉及的宇宙尺度。白巨眼中映射星辰，如同眼睛本身就是通向宇宙的窗口。

**3. 族长威压**
作为神兽族长，白巨拥有不怒自威的天然威压，能以眼神和姿态控制暴烈的灵尤。
*视觉表现*：当白巨转向灵尤时，灵尤的狂暴动作瞬间冻结。白巨眼中闪过一道警告性的微光（非魔法光效，而是瞳孔深处的自然反光），灵尤立即退缩。

### 能力视觉规范

| 能力 | 视觉表现形式 | 禁忌 |
|------|------------|------|
| 超自然体能 | 极简动作+环境微震+风压感 | 不使用能量光环或粒子特效 |
| 宇宙知识 | 眼中映射星空+场景尺度拉伸 | 不使用头顶光环、知识符文等 |
| 族长威压 | 眼神微光+对方冻结+压迫性构图 | 不使用冲击波、可见力场 |

## Ch5 角色发展轨迹

### 全局弧线

```
族长全家期          囚徒启悟期           山巅守望期
[威严的父亲] ──→ [启悟的哲人] ──→ [超然的守望者]
  ch9前半            ch9后半-ch38          ch39-ch69
```

### 各阶段详细轨迹

**族长全家期（ch9前半）**
*情感基调*：威严慈悲，父亲的温暖
*关键事件*：朱围庸坠入隐谷，初见三兽围坐篝火用餐。白巨居中端坐，左右是争抢食物的灵尤和温顺的灵闽。
*视觉转变标记*：篝火映照三兽形成稳定的三角构图→朱围庸坠入隐谷打破宁静

**囚徒启悟期（ch9后半-ch38）**
*情感基调*：深邃沉静，哲学思辨的宁静
*关键事件*：白巨用铁链囚禁了闯入领地的朱围庸，在约一个月的囚禁期间，于月光星空下与朱围庸进行关于宇宙创世、宿命与自由的深层对话，揭示宇宙真理。朱围庸最终逃离。
*视觉转变标记*：从隐谷篝火的家族场景→星空下巨兽与渺小人类囚徒的哲学对话构图
*核心台词主题*："所有生命都应享有自由"——这句话将影响朱围庸此后全部的反抗信念

**山巅守望期（ch39-ch69）**
*情感基调*：超然审视，不干预的悲悯
*关键事件*：七万难民穿越凤廊山脉雪山撤离时，白巨与灵尤作为"神秘山灵"在峰顶远处观察。此后持续以山巅守望者的身份存在，远距离注视人间战争。
*视觉转变标记*：从山谷内部→山巅外部，白巨的视角从近距离互动变为远距离俯瞰

## Ch6 人物关系

### 关系网络

```
        灵闽（次子）
         │ 父子
    ┌────┴────┐
    │  白巨   │── 朱围庸（囚徒→启蒙对象）
    └────┬────┘
         │ 父子
        灵尤（长子）
```

### 核心关系

**白巨 → 朱围庸**：囚禁者→哲学启蒙者
白巨在ch9用铁链囚禁了穿越凤廊山脉的朱围庸，将其关押约一个月。但在这段囚禁期间，白巨并非以暴力对待，而是在星空下与朱围庸进行了关于宇宙创世、宿命与自由的深层对话。这段对话从根本上改变了朱围庸对"自由"的理解，成为他此后反抗命运的哲学根基。
*视觉关系语言*：白巨5-6米的庞大体型与朱围庸人类身躯的极端比例差，以及铁链的物理束缚，形成"力量与智慧的不对称对话"的画面感。

**白巨 → 灵尤**：父亲→制约者
灵尤暴烈、冲动、渴望暴力，白巨以族长威压和父亲权威加以制止。两者在性格上形成极端对比：白巨的缓慢仪式感 vs 灵尤的爆发暴烈。
*视觉关系语言*：灵尤每次暴走时白巨仅需微微动作即可制止，体现力量差距。

**白巨 → 灵闽**：父亲→温柔之子
灵闽是白巨最温柔的孩子，性格好奇、友善、略带羞怯。在三兽之中，灵闽最接近白巨内心的柔软面，与暴烈的灵尤形成鲜明反差。
*视觉关系语言*：三兽同框时，灵闽总是贴近白巨身侧，形成温暖的父子依偎，与灵尤的躁动距离形成对比。

## Ch7 视觉设计

### 色彩系统

**阵营归属**：神兽族
**主色调**：纯白 #F5F5F5（毛发）+ 纯黑 #1A1A1A（面部）
**辅助色**：

| 色彩 | 色值 | 用途 |
|------|------|------|
| 象牙白 | #F5F5F5 | 毛发基础色，全身覆盖 |
| 墨黑 | #1A1A1A | 面部皮肤、爪甲 |
| 深灰 | #3A3A3A | 耳廓内侧、鼻部细节 |
| 火映金 | #C9A032 | 篝火场景中毛发反射色（仅族长全家期） |
| 月光银 | #B8C4D0 | 月下场景毛发反射色（仅囚徒启悟期） |
| 日光灰 | #D4D0C8 | 山巅日光场景毛发色调（仅山巅守望期） |

### 阶段色彩变化

| 阶段 | 主色调 | 环境反射色 | 整体氛围 |
|------|--------|-----------|---------|
| 族长全家期 | #F5F5F5 + #1A1A1A | 火光暖金 #C9A032 | 原始、温暖、野性的家族感 |
| 囚徒启悟期 | #F5F5F5 + #1A1A1A | 月光银蓝 #B8C4D0 | 宇宙级的孤独、哲学的宁静 |
| 山巅守望期 | #F5F5F5 + #1A1A1A | 日光灰白 #D4D0C8 | 超然、石雕般的永恒感 |

### 视觉签名：眼瞳深度

白巨的**眼瞳深度**是贯穿全片的视觉签名元素：
- **族长全家期**：眼瞳中映射跳动的篝火，温暖而警觉
- **囚徒启悟期**：眼瞳中映射满天繁星，如通向宇宙的窗口——这是白巨最具标志性的画面
- **山巅守望期**：眼瞳中映射远处人间的微小景象，超然而悲悯

### 光影方案

**族长全家期**：篝火作为主光源从下方照射，在白巨面部形成戏剧性的明暗反差。火光将白毛染成暖金色，黑色面部边缘出现琥珀色光晕。
**囚徒启悟期**：月光从上方/侧方照射，白毛呈冷银色调，黑色面部几乎融入夜色，仅眼睛映射星光作为焦点。整体画面趋向银灰单色调。
**山巅守望期**：自然日光/阴天散射光，毛发呈自然灰白色，无强烈色彩倾向。逆光时轮廓边缘产生白色辉光（非特效，而是自然逆光效果）。

## Ch8 连贯性约束

### 不可变特征（全阶段恒定）

| 特征 | 规范 | 容错范围 |
|------|------|---------|
| 体型 | 高5-6米，四足巨兽 | ±0.3米（远景透视变化） |
| 毛色 | 纯白 #F5F5F5 | 不同光照下允许色温偏移，但基础白色不变 |
| 面部 | 纯黑无毛 #1A1A1A | 不同光照下边缘可带环境反射色 |
| 双耳 | 长耳直立，头顶位置 | 耳朵方向可随注意力转动 |
| 毛发长度 | 约20厘米，颈部30厘米 | ±3厘米 |
| 动作速度 | 极慢，仪式感 | 任何情况下不得出现快速/惊慌动作 |

### 可变特征（随阶段变化）

| 特征 | 变化规则 |
|------|---------|
| 毛发反射色 | 随环境光源变化：火光→月光→日光 |
| 眼瞳映射 | 随场景变化：篝火→星空→人间远景 |
| 姿态 | 从居中端坐→星空下独坐→峰顶屹立 |
| 构图角色 | 从三角顶点→哲学对话双人构图→孤独剪影 |

### 比例约束

| 参照对象 | 比例关系 |
|----------|---------|
| 白巨 vs 朱围庸 | 约3:1至3.5:1（人类站立时仅到白巨肩部） |
| 白巨 vs 灵尤 | 约1.3:1（白巨略大于成年灵尤） |
| 白巨 vs 灵闽 | 约1.8:1（灵闽为未成年体型） |
| 白巨 vs 树木 | 约1:1至1:1.5（与大型古树接近） |

### 禁忌

- **禁止**白巨做出快速、惊慌、暴躁的动作——他的每一个动作都是经过深思熟虑的
- **禁止**在白巨身上添加任何魔法光效、符文、光环等——他的超自然感完全通过体量、动作、眼神传达
- **禁止**将白巨的白色毛发渲染为纯白 #FFFFFF——应始终保持自然的象牙白 #F5F5F5
- **禁止**白巨做出讨好、服从任何人类的姿态——即使面对强者，他的姿态也始终是平等甚至俯视的
