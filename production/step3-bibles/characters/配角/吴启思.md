## 三、外貌与形态

### 基础外貌

- **身材体型**：消瘦但精神矍铄，长期在实验室工作导致体型偏瘦，但肩膀微驼显示学者的典型体态；手指修长灵巧，指尖常沾有实验室的痕迹（墨迹、灼痕、化学染色）——是一双"思考者的手"
- **脸型**：长脸颧骨凸显，面部线条深刻，饱经风霜的深皱纹覆盖额头和眼角，颌骨线条清晰但肌肉略显松弛
- **眉毛**：花白浓眉，略显杂乱，思考时会不自觉皱起
- **眼睛**：圆形偏大，黑色虹膜后藏着睿智与执着；眼神清澈有神——是整张脸上最有生命力的部分。观察力敏锐，看人时常有"透视"的感觉，仿佛在扫描对方的思想
- **鼻子**：高挺的鼻梁，鼻翼较窄，鼻尖有明显的凹陷——长年推眼镜的习惯动作
- **嘴唇**：薄唇，嘴角天生微下垂显得有些厌倦或冷淡；笑容罕见且短暂，但真诚时会露出意外的温暖
- **皮肤**：泛黄偏白，因长期缺乏日照呈现不健康的苍白；面部有老人斑和色素沉着，显示年岁和实验室化学品接触的痕迹
- **发型**：花白头发蓬乱而不羁，似乎从不精心梳理，经常前额毛躁翘起；头顶略显稀疏——过度用脑的"代价"
- **标志性微动作**：频繁推眼镜（每句话说不完就要推一次），双手做出抽象的空间手势以解释复杂概念，坐下时习惯性地用食指在桌面敲出思考的节奏，兴奋时眼睛会放大闪闪发亮

### 单一着装与气质

**衣着**：长期穿着米色/浅灰色的旧实验服，口袋装满笔记本、数据线、小工具；内层穿着皱巴巴的深色棉质衬衫；脚上是陈旧但舒适的黑色运动鞋，鞋底已经磨平。实验服随处可见痕迹：袖口处的墨迹、腋下的汗渍、袖口处的烧伤痕迹、膝盖附近的烧伤痕迹——这是他"作战装"。随着故事推进，磨损程度加重（走出实验室更频繁），偶尔出入外部场景时改穿简素深色便装开衫。

**气质关键词**：执着、古怪、专注、隐隐的焦虑 → （后期）保留的执着、浮现的疑虑、被迫的世俗化

## 四、行为模式与标志性动作

### 3.1 科学家的思考节奏

吴启思最明显的行为模式是**停顿与爆发的交替**。他在对话中会经常突然停顿——眼睛看向远方，嘴唇微微鼓起，整个身体陷入完全的静止状态。这种静止可能持续3-5秒，直到他突然呼出一口气，转身面向对方，快速而清晰地说出他的想法。这不是心不在焉，反而是最专注的时刻——他在脑内运行复杂的逻辑演算。

### 3.2 推眼镜的思考密码

推眼镜是吴启思的标志性行为，频率与他的思维强度成正比。感到困惑时推眼镜一次，产生新想法时推眼镜两次，陷入深度思考时会不停地推，直到眼镜因为不断推动而在鼻梁上留下明显的红印。在实验室专注工作时，这个动作可能每十秒发生一次。与他人争论时，每一个反驳前都会伴随一次有力的推眼镜——这是他的"论证开场白"。频率随思维强度增加而递增，随伦理困惑加深而加倍。

### 3.3 手势中的逻辑

吴启思的手势充满空间几何感。解释数字人技术时，会用食指在空中画出神经网络的框架——从左到右、从上到下、从内到外——仿佛在构建一个看不见的三维模型。这些手势不是装饰性的，而是必要的——他似乎无法不用手势完整表达思想。在兴奋的时刻，双手会离开身体做出扩张的动作，整个人都在"说话"。

### 3.4 工作中的自动化行为

在实验室中，吴启思有一套自动化的仪式：工作台前坐下后，先调整眼镜确保清晰视野，然后双手放在工作区前，眼睛做一次从左到右的快速扫视（检查进度），最后开始操作。工作间隙，他会用食指在空中敲出节奏——这个动作既是思考的外化，也可能是音乐天赋的体现。

### 3.5 与他人互动的冷硬逻辑

吴启思的说话方式直率到几乎冷酷——他会打断对方、跳过寒暄、直奔主题。在与于永智讨论时会用反问来启发："你觉得这个问题的关键是什么？"而非直接给出答案。面对权力人物时，会摒弃客套，用专业术语建立话语权的不对等——让对方感到自己的外行。对学生或下属时则会发挥导师的一面，但批评也是直接而尖锐的："这个想法没有思考过充分的后果"。

**一句话弧线**：从科学理想主义的孤独守护者 → 技术应用的见证者 → 伦理困境的承担者——吴启思的人生是**理想与现实碰撞的记录**。

### 分阶段详解

#### 前期：隐世研究期（ch53-ch60）

- **核心特质**：执着的追梦者，被科学理想驱动的孤独研究者
- **行为表现**：ch53首次出现时沉浸在实验室，对外界人事漠不关心；ch55与于永智的对话中充满导师的严厉与关心；ch57-ch58手握数字人模型原型，眼中闪烁接近成功的兴奋；ch59-ch60准备首次人格导入，面对可能的伦理后果表现出隐隐不安
- **内心驱动**：证明自己几十年研究的价值、打破死亡的界限、获得学术认可（虽然表面不在乎）
- **外貌特征**：蓬乱花白头发、眼神锐利聚焦、脸部线条深刻显示长期思考的痕迹。ch53初识时有明显的实验室痕迹——袖口有灼伤、指甲下有不可洗净的染料、眼镜镜片有细微划痕。随着故事推进，眼神中的执着与紧迫感交织——他知道时间有限，对突破成功有强烈的渴望。ch58-ch60准备首次人格导入时，面部在亢奋与隐忧间摇摆，眼中闪烁的既有科学家的骄傲，也有伦理学家的不安
- **视觉状态**：实验室的冷白灯光照亮他锐利的眼神，蓬乱头发在显微镜前投出参差不齐的影子，手指在精密仪器间舞动——光影从阴冷→明亮清晰，象征思想的聚焦

#### 后期：技术启用期（ch61-ch71）

- **核心特质**：技术后果的见证者，正在蜕变的伦理思考者
- **行为表现**：ch61-ch63指导灵魂宿主的适应期，保持科学家的冷静观察；ch64-ch65见证首次成功时面露满足，但satisfaction很快被questions替代；ch66-ch68在观察宿主的心理变化时开始反思，与于永智的对话中出现了"你觉得我们做的对吗"这样的疑问；ch69-ch71被迫面对权力对技术的利用，眼中的执着逐渐被困惑所蚕食
- **内心驱动**：科学自豪感与伦理责任感的撕裂，对自己创造物的道德评判
- **外貌特征**：维持隐世研究期的基础形象，但面部表情与眼神发生微妙变化。ch61-ch65见证首次成功案例时，眼中闪烁的是科学家的满足感；ch66-ch71随着技术应用带来的伦理困境逐渐显露，眼神中的清晰感开始被困惑和反思所替代。嘴角的下垂变得更明显，显示内心的压力。推眼镜的频率增加，停顿思考的时间拉长——他在自问"我创造的东西是对是错"
- **视觉状态**：从实验室的冷白光逐渐走向混合光线——他离开了舒适的技术殿堂，踏入了充满不确定的现实世界。眼神的清晰度维持但多了阴影，推眼镜的频率加倍——思维在过度运转

### 关键转折点

1. **ch53 与于永智的首次对话**——进入故事的切入点，确立他作为技术源头的身份
2. **ch58 数字人原型成功**——从理论走向现实，也是焦虑的起点
3. **ch61 灵魂首次进入体内**——科学梦想的实现，也是伦理困境的开启
4. **ch66 观察到宿主的心理异常**——技术的"副作用"逐渐显露
5. **ch70 权力的直接干涉**——被迫认识到自己研究的社会化后果

### 视觉叙事线索：眼神清澈度与推眼镜频率

```
眼神锐利专注(ch53-ch58) → 夹杂兴奋与隐忧(ch59-ch61) → 满足但困惑(ch62-ch65) → 逐渐困顿(ch66-ch68) → 深度困惑与反思(ch69-ch71)
```

吴启思的视觉签名是**眼镜与眼神**。眼镜是他与世界的界限——干净的镜片代表清晰的思维，磨损的镜片代表受到现实的冲击。眼神的清澈度与他的信念强度正相关：满怀信心时眼神炯炯有神，伦理困惑加深时眼神会出现短暂的空洞——仿佛内部程序在重新计算。

## 六、人物关系

### 与于永智（学生/合作者）

**关系定位**：亲密的导师-弟子关系，隐含着代际传承与理想寄托

吴启思对于永智的态度是整个故事中唯一显露其温情一面的地方。ch55的对话中，他会为于永智的想法鼓掌，但转身就严厉批评其思考的不够深入——这是典型的"严师关爱"。他将于永智视为自己理想的继承者，有时候甚至是"灵魂的替身"。当于永智提出伦理质疑时，吴启思会陷入长久的沉默——这沉默不是被冒犯，而是被触发了内心已有但未敢直面的疑问。

### 与灵魂宿主（元刚/智刚等）

**关系定位**：专业但逐渐人文化的"造物主与被造物"关系

初期吴启思对宿主的态度是冷静的科学观察——他在做实验，被试者是数据点。但随着观察深入，尤其是看到宿主在数字体内出现心理异常时，他的态度开始柔和。ch67的场景中，他会主动询问宿主的感受，眼神中的科学冷漠被关切所替代。这个转变对他造成了深刻的冲击——他意识到自己不能永远躲在"科学家"的身份后，技术应用的后果最终会落在他的良知上。

### 与权力人物

**关系定位**：紧张而必要的合作，学术独立性与现实妥协的持续拉扯

吴启思坚守科学研究的纯粹性，但他需要资源。这种矛盾在权力的介入后激化。他不会主动向权力屈服，但权力的利用现实（将技术用于军事或控制）让他陷入两难。是否继续提供技术支持？如何在科学伦理与现实需求间做出选择？这个困境在ch69-ch71尤为突出。

### 与学术共同体

**关系定位**：被排斥但最终会震撼的"局外人"

吴启思的古怪让他在学术界保持了距离——他不参加学术会议、不发表论文、不参与学科竞争。但他的研究成果一旦公开，将彻底改写整个领域。这种"局外人终成中心"的身份给他带来了特殊的精神状态：既有不被认可的孤独，也有对自己成就的秘密自信。

## 七、视觉设计

### 整体视觉方案

吴启思的视觉设计遵循一个核心原则：**技术空间与老化身体的反差**。在实验室的冷白灯光中，他显得精神焕发；离开实验室后，混合的暖灰光线揭示了他真实的衰老。

| 要素 | 描述 |
|------|------|
| **主色调** | 枯草黄 #8B7355（身体与衣物） + 冷白灯光 #F0F0FF（实验室，仅限此环境） |
| **次色调** | 混入暖褐色 #A0826D（技术启用期，走出实验室时） |
| **关键道具** | 眼镜、笔记本、数据线、小工具 |
| **衣着** | 米色/浅灰色旧实验服 + 深色棉质衬衫 + 黑色运动鞋 |
| **磨损演变** | 初期：轻微污迹 → 中期：明显污迹与烧伤 → 后期：磨损加重显示频繁离开实验室 |

### 眼神特征

**锐利而专注**（ch53-ch60） → **夹杂满足与疑虑**（ch61-ch65） → **逐渐困顿失焦**（ch66-ch71）

眼神是最重要的表现手段。清澈的眼神代表信念的坚定，困惑的眼神代表思想的重新计算。在每一个关键转折点，眼神都应该出现微妙的变化——从锐利到温和、从清澈到深邃、从坚定到动摇。

### 气质参考

- **隐世研究期**：《知识分子的孤独感》——梁家辉《单身男女》中建筑师的冷静与专注，但更偏向学者的执着和与世隔绝感
- **技术启用期**：《现实的重量》——张丰毅《霸王别姬》中段小楼身上理想与现实碰撞时的那种无奈与困顿，但增加了更多的道德反思

## 八、连贯性约束

### 不变的要素

**视觉标识**：
- 花白蓬乱短发——全片保持，可根据工作强度略显蓬乱程度，但不可改变颜色或长度
- 眼镜——是他的标志，任何阶段不可去掉；镜片磨损程度可递进增加，映射心理状态
- 修长灵巧的手指——标志性的"思考者的手"，布满实验室痕迹但保持其知识分子特征
- 消瘦但挺直的身体——即使在困惑阶段，脊椎也不会弯曲，这是学者的尊严

**行为模式**：
- 对科学的执着——即使陷入伦理困境，他对研究本身的热忱未曾动摇
- 与学生的导师关系——即使面对于永智的质疑，他也保持尊重与倾听
- 思考时的停顿与爆发节奏——贯穿全片的标志性思维方式

**情感内核**：
- 吴启思本质上是理想主义者——即使被现实冲击，他对科学进步的信念不会彻底崩塌
- 他的焦虑来自深思而非恐惧——这是知识分子特有的道德敏感

### 可变的要素

**视觉变化**：
- 推眼镜频率随思维强度增加而递增，随伦理困惑加深而加倍
- 眼神清澈度随伦理困惑加深而递减，从锐利变为困顿再变为深邃反思
- 实验服磨损程度随离开实验室频率增加而加重，从初期的轻微污迹到后期的明显磨损
- 脸部表情线索：嘴角下垂明显程度随压力增加而加重；额头皱纹深度随思考强度增加

**互动风格变化**：
- 与外界的互动频率从隐居→逐渐频繁
- 表达方式从纯科学术语→混入伦理思辨
- 眼光焦点从技术→人的心理现象
- 信心强度从高度自信→夹杂疑虑→深度困惑→最终在困惑中寻求答案

### 禁忌

- **禁止**将吴启思画成"古怪疯狂"的刻板学者形象——他的古怪来自专注而非精神失常
- **禁止**让他在实验室外完全失去学者气质——即使在困惑中，他的挺直背脊和清晰眼神不能消失
- **禁止**使用冷白灯光照亮他的日常场景（非实验室）——这样会抹杀他作为"现实中老化的人"这一维度
- **禁止**让他与权力人物有直接的服从关系——他会妥协但保持尊严，妥协时眼中有不甘
- **禁止**在他的着装中加入"得体"的元素（如整洁领带、正式西装）——他的磨损衣服是身份认同的一部分
