## Ch3 外貌与形态

### 基础形态描述（神兽形态，非人类）

**整体轮廓**：
四足巨兽，站立高度4-5米，比父亲白巨矮约1米但肌肉线条更为暴突分明。身体覆盖约20厘米长的纯白色长毛，但与父亲整齐浓密的毛发不同，灵尤的毛发始终凌乱飞舞，如被永恒的风暴吹拂。面部完全无毛，露出纯黑色皮肤，五官比父亲更尖锐更具攻击性。头顶两只长耳直立但频繁转动，体现持续的警觉与不安。

**面部特征**：
- 面部皮肤：纯黑色，与父亲相同，但表面纹理更粗糙，肌肉紧绷
- 眼睛：尖锐、充血、带有明显攻击性，瞳孔收缩呈现猎食者特征——与父亲悲悯深邃的眼神形成鲜明对比
- 五官：比父亲更尖锐的轮廓——鼻梁更窄、颌线更锋利、嘴部线条更紧绷
- 双耳：同样长而直立，但频繁快速转动，无法安静

**身体结构**：
- 四肢比父亲更为精壮，肌肉线条在毛发下清晰可见
- 爪部宽大，指甲更长更尖锐，呈深黑色
- 背部肌肉隆起，呈现蓄势待发的弓形姿态
- 胸腔宽阔深厚，适合爆发性冲锋

**毛发系统**：
- 颜色：纯白色（#F5F5F5），与父亲相同
- 长度：约20厘米，与父亲相同
- 质感：**关键差异**——灵尤的毛发始终凌乱、飞散、纠结，如被风暴吹乱，绝非整齐梳理状态。即使无风时也呈现蓬乱感
- 颈部鬃毛：同样约30厘米，但向外炸开而非垂落

**动作特征**：
与父亲的极慢仪式感完全相反。灵尤的一切动作都是爆发性的——突然转头、猛烈跺地、急促呼吸。静止时也呈现弹簧压缩般的蓄势状态，身体微颤，随时可能弹射而出。

| 体型 | 5-6米，厚重如山 | 4-5米，精壮如豹 |
| 毛发状态 | 整齐浓密，如丝绸 | 凌乱飞散，如风暴 |

| 颈部鬃毛 | 自然垂落如瀑布 | 向外炸开如狮鬃 |

| 视觉要素 | 隐谷幼兽期 | 山巅躁兽期 | 战场凶兽期 |
|----------|-----------|-----------|-----------|
| 毛发状态 | 凌乱但干净完整 | 被山风吹拂如白旗 | 沾满血污尘土，灰褐杂色 |

### 视觉辨识标记（跨阶段不变）

- **凌乱白毛**：与父亲整齐毛发的核心视觉区别，一眼可辨
- **攻击性眼神**：瞳孔收缩的猎食者凝视，区别于父亲的悲悯深邃
- **蓄势姿态**：身体永远处于弹簧压缩状态，从未真正放松
- **黑色面部+尖锐轮廓**：同为黑脸但线条更锋利

## Ch4 能力与视觉表现

### 核心能力

**1. 毁灭性冲锋**
灵尤最具标志性的能力是直线冲锋——以4-5米的庞大身躯全速奔袭，碾碎路径上的一切人类、战马和工事。冲锋时地面震颤，白毛在气流中向后飞散如白色彗星尾巴。
*视觉表现*：冲锋路线呈直线破坏带，两侧士兵和碎片向外飞溅。灵尤本身如一颗白色炮弹，毛发拖出残影。地面留下深深的爪印沟壑。

**2. 蛮力压制**
纯粹的物理力量——能以爪掌拍碎战马、撕裂建筑木结构、掀翻攻城器械。这种力量没有技巧，只有绝对的暴力。
*视觉表现*：动作简单粗暴——拍、砸、碾、撕。每一击伴随冲击波般的灰尘扩散和碎片四射。

**3. 兽类感知**
超越人类的感官——能感知远处人群的存在（原文记载神兽在远处观察人类活动），听觉和嗅觉极其敏锐。
*视觉表现*：双耳快速转向声源，鼻翼翕动，身体瞬间从蹲伏转为攻击姿态。

### 能力视觉规范

| 能力 | 视觉表现形式 | 禁忌 |
|------|------------|------|
| 毁灭性冲锋 | 直线破坏带+毛发残影+地面沟壑 | 不使用能量尾迹或光效——纯物理破坏 |
| 蛮力压制 | 碎片四射+冲击灰尘+简单暴力动作 | 不使用格斗技巧或武术动作 |
| 兽类感知 | 耳转+鼻翕+姿态突变 | 不使用感知光环或探测波纹 |

### 能力局限

- **无法抗衡父亲白巨**：白巨仅需最小动作即可制止灵尤，体现绝对的力量等级差
- **无法抗衡元神级存在**：威刚以超自然泡泡轻松困住灵尤，灵尤完全无法挣脱
- **无战术智慧**：只会直线冲锋和蛮力攻击，没有策略、不会绕行、不会伏击

## Ch5 角色发展轨迹

### 全局弧线

```
隐谷幼兽期          山巅躁兽期           战场凶兽期
[被压制的幼兽] ──→ [蓄势的困兽] ──→ [被释放的凶器]
    ch9              ch10-ch55           ch56-ch69
```

### 各阶段详细轨迹

**隐谷幼兽期（ch9）**
*情感基调*：躁动不安，幼兽式的暴烈
*关键事件*：与父亲白巨、弟弟灵闽共居凤廊山脉隐谷。围坐篝火用餐时与灵闽争抢食物，暴露暴烈本性。朱围庸坠入隐谷，打破了三兽的日常宁静。
*视觉转变标记*：三兽篝火构图中，灵尤位于三角底部一端，与温顺的灵闽形成对比→人类闯入者打破隐谷平衡

**山巅躁兽期（ch10-ch55）**
*情感基调*：被压抑的怒火持续升温
*关键事件*：与父亲白巨立于凤廊山脉峰顶，作为"神秘山灵"在远处观察人间动态。ch39-40七万难民穿越雪山撤离时，白巨与灵尤从峰顶远距离注视。此后灵尤被杨弘以某种方式收服/控制，成为其秘密军事武器。
*视觉转变标记*：灵尤的姿态从蹲伏变为半起身，从试探性前倾变为全身肌肉绷紧的即将失控状态→体型随时间从亚成年长至完全成年

**战场凶兽期（ch56-ch69）**
*情感基调*：纯粹的杀戮空白——已无愤怒，只剩毁灭本能
*关键事件*：被杨弘作为秘密武器在攻城战中释放。从北门横冲直撞，碾碎士兵和战马。ch69与胡酹正面交锋，胡酹在战斗中被灵尤碾碎。随后威刚出现，以超自然泡泡将灵尤困住。
*视觉转变标记*：纯白毛发被血污和尘土染成灰褐杂色→眼神从怒火变为空洞的杀戮空白→被困泡泡中时毛发湿结、身体静止，首次完全不动

## Ch6 人物关系

### 关系网络

```
        灵闽（弟弟）
         │ 兄弟
    ┌────┴────┐
    │  白巨   │── 灵尤
    │ （父亲）│     │
    └─────────┘     ├── 杨弘（后期驱使者）
                    ├── 胡酹（ch69对手，被碾碎）
                    └── 威刚（ch69捕获者）
```

### 核心关系

**灵尤 → 白巨**：子→被制约者
灵尤在父亲面前从不敢真正爆发。白巨仅需微微抬掌或一个眼神，灵尤便瞬间冻结。这不仅是力量差距，更是族长威压的天然等级压制。灵尤内心的暴烈被父亲的存在持续压缩，如同被按住的弹簧。
*视觉关系语言*：两兽同框时，白巨静止如山，灵尤在旁躁动如困兽。白巨微动→灵尤立刻退缩。

**灵尤 → 灵闽**：兄→弟
灵闽是温柔的弟弟，与灵尤性格完全相反。灵闽的安静友善与灵尤的暴烈冲动形成鲜明对比，两兽同为白巨之子却走向截然不同的性格极端。
*视觉关系语言*：灵闽毛发光滑整洁 vs 灵尤毛发凌乱——同父所出，性格的视觉化对比。

**灵尤 → 杨弘**：兽→被驱使的武器
后期灵尤被杨弘收服并作为攻城秘密武器使用。这一关系将灵尤从"被父亲压制的困兽"转化为"被人类利用的战争工具"，完成了从自然暴力到工具化暴力的堕落。
*视觉关系语言*：灵尤在战场上不再是野兽，而是一枚被发射的炮弹——无方向感、无选择权、纯粹的破坏力。

**灵尤 → 胡酹**：捕食者→猎物
ch69中灵尤从北门冲入，与胡酹正面交锋。胡酹虽为强悍的还阳武将，但在灵尤的绝对体量和蛮力面前被碾碎。
*视觉关系语言*：4-5米巨兽 vs 人类体型的胡酹——绝对的体量碾压。

**灵尤 → 威刚**：凶兽→被困者
威刚作为十八元神之一，轻松以超自然泡泡将灵尤困住。灵尤的一切蛮力在更高维度的力量面前完全无效。
*视觉关系语言*：灵尤在透明泡泡中挣扎但无法破壁，首次呈现完全静止的画面。

## Ch7 视觉设计

### 色彩系统

**阵营归属**：神兽族
**主色调**：纯白 #F5F5F5（毛发）+ 纯黑 #1A1A1A（面部）——与父亲白巨相同种族色

**辅助色**：

| 色彩 | 色值 | 用途 |
|------|------|------|
| 象牙白 | #F5F5F5 | 毛发基础色（隐谷幼兽期、山巅躁兽期） |
| 墨黑 | #1A1A1A | 面部皮肤、爪甲 |
| 深黑 | #0D0D0D | 爪甲尖端，比父亲更深更尖锐 |
| 充血红 | #8B2500 | 眼白充血色，愤怒时加深 |
| 血污灰褐 | #6B5B4F | 战场凶兽期毛发被血污染色 |
| 尘土黄灰 | #9C8C7C | 战场凶兽期尘土附着色 |

### 阶段色彩变化

| 阶段 | 毛发色 | 面部色 | 环境反射色 | 整体氛围 |
|------|--------|--------|-----------|---------|
| 隐谷幼兽期 | #F5F5F5 纯白 | #1A1A1A 墨黑 | 火光暖金 #C9A032 | 野性、年轻、躁动 |
| 山巅躁兽期 | #F5F5F5 纯白 | #1A1A1A 墨黑 | 冷灰日光 #B0B0B0 | 压抑、蓄势、即将爆发 |
| 战场凶兽期 | #F5F5F5→#6B5B4F 白转灰褐 | #1A1A1A 墨黑 | 火光橙红 #CC4400 | 毁灭、血腥、失控 |

### 视觉签名：毛发动态

灵尤的**毛发动态**是贯穿全片的视觉签名：
- **隐谷幼兽期**：毛发蓬乱但随自然风微动——"被搅乱的白色丝绸"
- **山巅躁兽期**：毛发被山风持续吹拂如白色旗帜飘动——"即将撕裂的白帆"
- **战场凶兽期**：毛发沾血结块、在冲锋气流中向后飞散如彗星尾——"染血的白色弹道尾迹"

### 光影方案

**隐谷幼兽期**：篝火从下方照射（与父亲白巨同场景），火光在凌乱毛发中产生更多碎影和高光跳跃。黑色面部在火光中显得更加凶悍。
**山巅躁兽期**：高海拔自然光/阴天散射光，冷灰色调。灵尤身上无暖色——与父亲同场但情感色温更冷。逆光时凌乱毛发边缘产生毛刺般的碎光。
**战场凶兽期**：战场火光（建筑燃烧、火箭）的橙红色照射在灰褐血污毛发上，产生"地狱白兽"效果。烟尘中灵尤的轮廓时隐时现，增强恐怖感。

## Ch8 连贯性约束

### 不可变特征（全阶段恒定）

| 特征 | 规范 | 容错范围 |
|------|------|---------|
| 体型 | 高4-5米，四足巨兽 | ±0.3米 |
| 毛色基础 | 纯白 #F5F5F5 | 战场凶兽期允许血污灰褐覆盖，但可见区域仍为白色基底 |
| 面部 | 纯黑无毛 #1A1A1A | 不同光照下边缘可带环境反射色 |
| 双耳 | 长耳直立，频繁转动 | 耳朵转动频率随情绪加快 |
| 毛发状态 | 始终凌乱飞散 | 任何阶段不得出现整齐梳理状态 |
| 动作风格 | 爆发性、不可预测 | 任何阶段不得出现缓慢仪式感动作（那是父亲白巨的特征） |
| 眼神 | 攻击性、瞳孔收缩 | 不得出现悲悯或智慧感（那是父亲白巨的特征） |

### 可变特征（随阶段变化）

| 特征 | 变化规则 |
|------|---------|
| 毛发清洁度 | 隐谷/山巅期干净→战场期血污覆盖 |
| 体型成熟度 | 隐谷期亚成年→山巅期完全成年 |
| 眼神 | 暴躁活力→压抑怒火→空洞杀戮 |
| 伤痕 | 隐谷/山巅期无→战场期可能有战斗伤痕 |

### 比例约束

| 参照对象 | 比例关系 |
|----------|---------|
| 灵尤 vs 白巨 | 约0.75:1至0.85:1（灵尤比父亲矮约1米） |
| 灵尤 vs 灵闽 | 约1.4:1（灵闽为未成年幼兽） |
| 灵尤 vs 人类 | 约2.5:1至3:1（人类仅到灵尤腹部） |
| 灵尤 vs 战马 | 约2:1（灵尤远大于战马） |

### 与白巨的区分禁忌
