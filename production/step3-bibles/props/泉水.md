## 二、物理特性

### 2.1 外观描述

| 特性 | 描述 |
|------|------|
| **位置** | 牛台山山腰至山顶之间，靠近美女榕 |
| **形态** | 圆形水池，直径约2-3米，深度约0.5米 |
| **水质** | 极度清澈透明，可见底部鹅卵石 |
| **水面** | 平静如镜（无风时），微微闪闪发光 |
| **口感** | 矿物质清爽味道 |

### 2.2 周围环境

- **溪流入口**：北方山谷流入
- **溪流出口**：南侧流出
- **周围平地**：半径10-15米的开放区域
- **标志性巨石**：约2米高的大灰岩，位于池塘东侧

## 三、功能效果

### 3.1 对人类的效果

| 效果 | 描述 |
|------|------|
| **伤口愈合** | 加速伤口愈合 |
| **体力恢复** | 恢复体力和精力 |
| **消除疲劳** | 消除疲劳，增强免疫能力 |
| **返老还童** | 使人仿佛年轻20岁 |
| **思维敏捷** | 思维变得敏捷，记忆力提升 |
| **方向感** | 身体轻松感增加，方向感恢复 |

### 3.2 对受伤生命的效果

- 快速愈合（小鹿喝泉水数小时内完全恢复）
- 可以救治即将死亡的生命

### 3.3 对灵魂的效果

| 效果 | 描述 |
|------|------|
| **结构恢复** | 恢复散开的灵魂结构 |
| **聚拢魂体** | 重新聚拢松散的灵魂形态 |
| **魂魄融合** | 加速魂与魄的融合 |
| **适应肉体** | 帮助灵魂快速适应肉体 |
| **恢复速度** | 约为普通恢复的30倍 |

### 3.4 副作用/诅咒

> 对外人有诅咒作用——一个月内使人变傻

- 长期饮用对主角一家无害反益
- 无其他副作用

## 四、叙事功能

### 4.1 故事触发点

- 老吴在第1章发现泉水，开启整个故事
- 泉水的神奇功效是故事的核心驱动力

### 4.2 战略资源

- 朱围庸将泉水作为控制还阳者的核心资源
- 秘密泉水仓库的建设（泊岗镇）
- 储存在水桶中可长期保存

### 4.3 象征意义

| 象征 | 含义 |
|------|------|
| **生命之源** | 希望与重生的象征 |
| **阴阳之门** | 连接两界的物质表现 |
| **救赎** | 神圣与神秘的力量 |
| **权力** | 控制他人的资源（朱围庸） |

## 五、关键场景

### 场景1：老吴首次发现（第1章）
- 老吴在山林中偶然发现泉水池塘
- 饮用后感受到神奇的恢复效果
- 决定保守这个秘密

### 场景2：小鹿的救治（第1章）
- 受伤的小鹿饮用泉水
- 数小时内完全恢复
- 展现泉水的治愈能力

### 场景3：泉水仓库（泊岗镇）
- 朱围庸建立秘密泉水储存点
- 用于维持还阳者的生命
- 成为权力控制的工具

### 场景4：灵魂恢复（多章）
- 朱围庸利用泉水恢复灵魂状态
- 加速魂魄融合过程
- 维系还阳者军团

```

├─ 连接阴阳
└─ 权力资源
```

### 核心定位
