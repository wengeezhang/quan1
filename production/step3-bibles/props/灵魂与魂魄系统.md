## 二、基本概念

### 2.1 魂与魄的定义

| 组成部分 | 功能 | 特性 |
|----------|------|------|
| **魂** | 主灵气心智 | 意识、记忆、人格 |
| **魄** | 主体力血气 | 生命力、肉体驱动 |

### 2.2 生死循环

```
生前：魂 + 魄 = 完整生命
        ↓
死亡时：魂魄分离
        ↓
    ┌───┴───┐
    ↓       ↓
正常轮回   十恶不赦
（投胎）   （恶灵区）
```

## 三、灵魂形态

### 3.1 外观特征

| 特性 | 描述 |
|------|------|
| **基本形态** | 不规则、布满褶皱的圆形光影团 |
| **大小** | 随情绪和状态变化 |
| **色调** | 正常为淡蓝/白色；罪孽者为暗紫色 |
| **密度** | 与灵魂状态相关 |

### 3.2 状态变化

| 状态 | 表现 |
|------|------|
| **交流时** | 轮廓外围产生水波纹般的震荡 |
| **受创后** | 形态松散、不稳定，轮廓扩大 |
| **濒散时** | 外扩至最大，内部光影紊乱 |
| **恢复中** | 逐渐凝聚，光芒恢复稳定 |

### 3.3 颜色意义

| 颜色 | 含义 |
|------|------|
| 淡蓝/白色 | 纯净灵魂 |
| 暗紫色 | 罪孽印记 |
| 紫色光晕 | 恶灵/还阳者标志 |
| 金色光芒 | 救赎/神圣状态 |

---

## 四、灵魂能力

### 4.1 移魂夺魄

| 属性 | 描述 |
|------|------|
| **原理** | 将灵魂强制进入他人身体，挤出原有灵魂 |
| **执行者** | 朱围庸、李仁（主要操作者） |
| **条件** | 需要一定的灵魂力量和技巧 |
| **风险** | 可能导致灵魂损耗 |

**应用范围**：
- 控制普通人类
- 大规模改造人口（30名新还阳者）
- 操控数字人（第71章）

### 4.2 灵魂探测

- 感知灵魂存在和位置
- 在恶灵区识别同伴
- 招募新的还阳者

### 4.3 吸食精气

- 吸取周围活人的精气
- 维持灵魂生命力
- 恢复灵魂损耗

## 五、恶灵区设定

### 5.1 基本信息

| 属性 | 描述 |
|------|------|
| **位置** | 地下200米以下 |
| **功能** | 关押十恶不赦犯人的灵魂监狱 |
| **环境** | 充满绝望与折磨 |
| **管理者** | 阴间系统（宋小仙等） |

### 5.2 惩罚机制

- 永恒的囚禁
- 无法投胎转世
- 紫色印记标记

### 5.3 逃脱途径

- **十八层塔**：通过考验可获重生资格
- **树根通道**：朱围庸发现的非常规逃脱路径

## 六、灵魂的救赎与重生

### 6.1 正常途径
- 通过十八层塔考验
- 获得新的魄
- 重新投胎转世

### 6.2 非常规途径
- 通过树根逃脱（朱围庸）
- 附身人类肉体
- 成为"还阳者"

### 6.3 数字人途径（第71章）
- 将灵魂植入人造肉体
- 3D打印人体技术
- 科技与超自然的融合
