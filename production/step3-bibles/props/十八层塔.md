| 属性 | 描述 |
|------|------|
| **层数** | 18层 |

### 2.2 考验机制

```
第1层 → 第2层 → ... → 第18层
   ↓        ↓            ↓

   ↓        ↓            ↓
    └────────┴─────┬─────┘

            通过 = 获得重生
            失败 = 返回起点
```

## 三、层级设计

### 3.1 难度递进

| 层级范围 | 难度 | 考验类型 |
|----------|------|----------|
| 第1-6层 | 低 | 基础意志考验 |
| 第7-12层 | 中 | 精神折磨考验 |
| 第13-17层 | 高 | 极限生存考验 |
| 第18层 | 极限 | 最终救赎考验 |

### 3.2 朱围庸的攀塔记录

**第一次尝试**
- 到达第9层后退下
- 灵魂形态变得松散、不稳定
- 边缘模糊，光影暗淡

**第二次尝试（第3章）**
- 到达第15层
- 濒临散架状态
- 发现树根逃脱途径

### 3.3 濒散状态描述

> 圆形光团外扩至最大，内部光影紊乱闪烁，呈半透明的即将崩解状态

## 四、象征意义

### 4.1 救赎主题

| 象征 | 含义 |
|------|------|
| **攀塔** | 自我救赎的过程 |
| **18层** | 完整的考验周期 |
| **顶层** | 重生与新生 |
| **失败** | 永恒的循环 |

- 挑战塔是对命运的抗争
- 发现树根是创新性突破

## 五、叙事功能

### 5.1 世界观建立
- 展示恶灵区的规则系统
- 说明灵魂救赎的可能途径
- 建立"考验-奖励"的逻辑

### 5.2 角色塑造
- 展现朱围庸的意志力
- 体现其突破常规的思维
- 为后续行为提供铺垫

### 5.3 情节推动
- 朱围庸因攀塔发现树根
- 由此开启逃脱之路
- 触发整个故事主线

## 六、视觉设计

### 6.1 整体风格

| 元素 | 设计 |
|------|------|
| **外观** | 高耸入云的黑色塔楼 |
| **材质** | 古老的石材或金属 |
| **氛围** | 压抑、威严、神秘 |
| **光线** | 昏暗，仅有微弱的红/紫光 |

### 6.2 内部设计

**低层（1-6层）**
- 相对宽敞的空间
- 模糊的威胁感
- 灰暗的色调

**中层（7-12层）**
- 空间逐渐狭窄
- 明显的折磨元素
- 深红色的光芒

**高层（13-17层）**
- 极度狭窄和压迫
- 灵魂扭曲的环境
- 紫黑色的氛围

**顶层（18层）**
- 未知的神秘空间
- 强烈的光与暗对比
- 重生之门的象征

### 6.3 色彩基调

| 层级 | 主色 | 辅色 |
|------|------|------|
| 1-6层 | 暗灰 #3D3D3D | 淡紫 #9370DB |
| 7-12层 | 深红 #8B0000 | 黑色 #1A1A1A |
| 13-17层 | 紫黑 #2E0854 | 血红 #DC143C |
| 18层 | 金白 #FFD700 | 纯黑 #000000 |

## 九、设计要点

### 9.1 建筑感
- 高耸、压迫、不可逾越的感觉
- 每层有独特但统一的设计语言
- 越高越狭窄、越压迫

### 9.2 考验感
- 每层都有明显的"折磨"元素
- 灵魂在其中的挣扎要可视化
- 成功与失败的对比

### 9.3 希望与绝望
- 顶层是希望的象征
- 攀塔过程是绝望的体现
- 树根是意外的希望

---

## 十、制作建议

### 10.1 模型设计
- 可拆分的层级结构
- 内部有可见的考验元素
- 灵魂通过的动态展示

### 10.2 光效设计
- 层级递进的色温变化
- 濒散时的闪烁效果
- 树根发现时的光芒突破

### 10.3 音效建议
- 低沉压抑的背景音
- 灵魂挣扎的嘶嘶声
- 发现希望时的清脆音效
