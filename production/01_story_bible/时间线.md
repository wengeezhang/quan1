# 时间线设定文档

## 一、历史背景时间线

### 周朝
- **事件**：灵魂"封"被囚禁入恶灵控制中心
- **角色**：封（男性）
- **身份**：周朝时期犯下大罪的灵魂
- **特殊性**：
  - 已在恶灵区存续数千年
  - 是凤毛麟角般的古老灵魂
  - 大多数同时期灵魂已消散，他仍存续
  - 作为恶灵区的历史见证人存在

---

### 明朝
- **时期**：明朝年间
- **事件**：朱围庸犯下大罪被囚禁
- **角色**：朱围庸（男性）
- **身份**：明朝时期的恶灵
- **在恶灵区时长**：已在此地几百年

#### 朱围庸的性格与信仰
- **自我认知**：不相信"人之初，性本恶"的说法
- **理由**：认为自己犯错是周围环境逼迫所致，非本性使然
- **信心**：坚信自己不应被剥夺投胎机会
- **执着**：与李仁、胡酹三人不甘现状，千方百计寻找出路

---

### 清朝康熙年间
- **事件**：李仁被囚禁入恶灵控制中心
- **角色**：李仁（男性）
- **身份**：清朝康熙年间的恶灵
- **关系**：与朱围庸、胡酹组成反抗三人组
- **特点**：
  - 思想活跃，敢于提问
  - 与朱围庸理想相同
  - 在关键时刻给予朱围庸鼓励和支持

---

### 清朝雍正年间
- **事件**：胡酹被囚禁入恶灵控制中心
- **角色**：胡酹（男性）
- **身份**：清朝雍正年间的恶灵
- **关系**：与朱围庸、李仁组成反抗三人组
- **特点**：
  - 为朱围庸献言献策
  - 性格谨慎，给予现实的建议
  - 关键时刻表现出深厚的同志情谊

---

### 第一章：朱围庸诞生

#### 时间：22世纪初冬日

**1.1 老吴被行政折腾（序幕）**

- 地点：派出所、户政室
- 事件：
  - 老吴为孙子办户籍证明
  - 被折腾三趟，连续挫折

**1.2 老吴出门散步（发端）**
- 时间：之后数天
- 地点：牛台山附近
- 事件：
  - 老伴劝其出门散步
  - 老吴谎称陪老同学吃饭

**1.3 老吴迷路（危机）**
- 时间：黄昏到深夜
- 地点：牛台山深处
- 事件：
  - 向山深处行走
  - 天色渐暗，严重迷路
  - 开始恐惧和绝望

**1.4 严冬求生（困境）**
- 时间：深夜到半夜
- 地点：山上小池塘旁
- 事件：
  - 北风大雪，条件恶劣
  - 发现清澈的小池塘
  - 喝泉水，靠在大石头上昏迷

**1.5 智慧求生（转机）**
- 时间：半夜
- 地点：大岩石周围
- 事件：
  - 被冻醒，开始自救
  - 利用枯树枝搭建小屋
  - 点燃篝火，堆落叶
  - 身体和思维明显改善

**1.6 成功返家（归来）**
- 时间：天亮后
- 地点：回到家中
- 事件：
  - 方向感回复，成功下山
  - 向老伴谎称在朋友家住
  - 身体状态明显好转

**1.7 每日山上疗养（恢复期）**
- 时间：随后数天到数周
- 地点：山上临时小屋
- 事件：
  - 每天上山待半日
  - 饮用泉水
  - 身体逐步改善
  - 思维敏捷，身体强健
  - 决定不告诉家人

**1.8 救助小鹿（良心展现）**
- 时间：随后的某个清晨
- 地点：山上临时小屋附近
- 事件：
  - 发现被藤蔓缠住的小鹿
  - 费力救助小鹿
  - 小鹿喝泉水恢复
  - 悉心照料三天
  - 小鹿成为每日访客

### 第二章：老吴在泉水旁盖房

#### 时间：22世纪初冬日至春初

**2.1 建房的决心（起始）**
- 时间：救助小鹿之后
- 地点：山上临时小屋
- 事件：
  - 老吴决定建造更好的房子
  - 设计三间宽房子（客厅+两卧室）
  - 计划含厨房和竹栅栏院落
  - 选择隐蔽位置

**2.2 建房材料准备（第一阶段，约2周）**
- 时间：建房第一周期
- 地点：山上
- 事件：
  - 采集黄泥土
  - 加水和干草搅拌
  - 开始砌墙（五十公分深，四十公分宽的沟）
  - 每砌半米晾晒半天

**2.3 墙体完成（第二阶段，约1-2周）**
- 时间：建房第二周期
- 地点：山上建筑地
- 事件：
  - 墙体砌至三米高
  - 请亲人帮忙安装大梁
  - 砍伐松木和树干
  - 用钉子固定屋顶框架

**2.4 屋顶与外墙完成（第三阶段，约1-2周）**
- 时间：建房第三周期
- 地点：山上建筑地
- 事件：
  - 铺设茅草覆盖屋顶
  - 两间主屋基本成型
  - 厨房建设完毕
  - 整体花费约一个月

**2.5 竹栅栏院落完成（第四阶段，约1周）**
- 时间：建房最后周期
- 地点：山上小院
- 事件：
  - 砍伐竹子（两米高）
  - 并排围成院子
  - 多次修葺和装饰
  - 小院彻底完成

**2.6 老吴对老伴的告知（关键对话）**
- 时间：建房完成后
- 地点：老吴家中
- 事件：
  - 老吴主动向老伴说明
  - 展示手臂肌肉和茧子
  - 老伴半信半疑
  - 老吴指向山腰说明位置

**2.7 老伴首次入山（感受期）**
- 时间：建房完成后第二天
- 地点：山上小院和周围
- 事件：
  - 老吴和老伴收拾物品
  - 带上床单和生活用品
  - 食物包括红烧肉和糖醋排骨
  - 老伴被小院温馨所打动
  - 老伴决定在此长住

**2.8 小院生活的开始（新阶段）**
- 时间：老伴入住后
- 地点：山上小院
- 事件：
  - 简单布置卧室
  - 准备做饭用品
  - 一起做简单晚餐
  - 在此住下

**2.9 登山游历（浪漫期）**
- 时间：入住后第二天清晨
- 地点：花溪及其上游
- 事件：
  - 老吴带老伴游览周围
  - 发现花溪的美景（宽一米，水流急，榕树高大）
  - 给小溪起名"花溪"
  - 沿溪上行约二十分钟到平台休息

**2.10 美女榕的发现与命名（标志事件）**
- 时间：登山游历中
- 地点：花溪附近的平台
- 地点特征：距院子约二十分钟步程
- 事件：
  - 发现一棵特殊的大榕树
  - 树干粗大（需两人抱下）
  - 树根盘旋交织呈女性人形
  - 两条细"腿"深伸溪水
  - 老伴建议名字"美女榕"
  - 老吴欣然同意
  - 两人继续登上山顶
  - 因起风返回小院

**2.11 新生活的展开（情景转折）**
- 时间：之后的日子
- 地点：山上小院
- 事件：
  - 老吴和老伴在此建立新的生活
  - 享受大山幽静
  - 不知道山下发生了什么
  - "然而这座山的地下，竟是另外一个景象"

### 第三章：朱围庸逃脱

#### 时间：同期（从第二章某时刻开始）

**3.1 恶灵区的绝望（背景）**
- 时间：22世纪初冬日
- 地点：牛台山地下恶灵控制中心
- 事件：
  - 数百万恶灵积累在此
  - 空间开始拥挤
  - 绝望气氛弥漫
  - 晚间聚会是唯一的欢乐

**3.2 古老灵魂的到来（转折）**
- 时间：某个晚上（具体时间不确定）
- 地点：恶灵区聚会地
- 事件：
  - 来自周朝的古老灵魂"封"罕见出现
  - 大家期待他讲述奇闻旧事
  - 李仁突然问出关键问题："怎样才能离开这里？"

**3.3 封的古老传说（希望之光）**
- 时间：同一晚上
- 地点：恶灵区聚会地
- 事件：
  - 封揭示十八层塔的传说
  - 解释通过考验可重获魄
  - 可进入轮回隧道投胎
  - 透露"确实有人做到过"
  - 朱围庸、李仁、胡酹受鼓舞

**3.4 朱围庸的首次挑战（失败的勇气）**
- 时间：次日
- 地点：十八层塔
- 事件：
  - 朱围庸召集李仁和胡酹商议
  - 二人全力支持
  - 朱围庸出发挑战
  - 成功爬到第九层
  - 忍受不住折磨，下来

**3.5 初战失败的恢复（挫折与思考）**
- 时间：失败后数天
- 地点：恶灵区
- 事件：
  - 朱围庸灵魂散开，轮廓扩大一倍
  - 濒临散开消亡
  - 李仁和胡酹悉心照料
  - 几天后恢复一些
  - 开始怀疑传说的真实性
  - 思考上神的用意

**3.6 朱围庸的决绝与勇气（心路历程）**
- 时间：失败后半个月
- 地点：恶灵区
- 事件：
  - 朱围庸持续思考
  - 相信上神不会无谓折磨
  - 相信洗礼的意义在于脱胎换骨
  - 决定再次挑战
  - 召集李仁和胡酹表明决心
  - 说出决绝之言："如果今天晚上没下来，就是我成功了"

**3.7 朱围庸的二次挑战（成功突破）**
- 时间：决定后立即进行
- 地点：十八层塔
- 事件：
  - 成功通过第九层
  - 继续向上攀登
  - 到达第十五层

**3.8 极限时刻与生死抉择（关键转折）**
- 时间：攀塔过程中到第十五层
- 地点：十八层塔第十五层
- 事件：
  - 灵魂力气耗尽
  - 无法聚拢自己
  - 灵魂即将散开
  - 面临两个选择：
    - 退下去（但认为下次无法再次承受15层折磨）
    - 坚持向上（面临灵魂散开的风险）

**3.9 树根奇遇（意外救赎）**
- 时间：灵魂即将散开时刻
- 地点：第十五层附近
- 事件：
  - 突然触碰到一根树根
  - 仅有手指粗细，长约十公分
  - 朱围庸意识到这是凡间生命的通道
  - 决定钻入树根
  - 用尽最后力气缩聚成圆锥状
  - 扎进树根

**3.10 穿越通道（回到阳间）**
- 时间：进入树根后
- 地点：从地下通向地表
- 事件：
  - 魂沿树根缓慢上升
  - 不知升了多久
  - 空间逐渐变大
  - 朱围庸舒展开来
  - 听到潺潺流水声
  - 听到鸟叫声和蝉鸣
  - 恍然发现自己附着在榕树树根上

**3.11 附身美女榕（新生命开始）**
- 时间：重返人间直后
- 地点：美女榕的树根上
- 事件：
  - 朱围庸无法看到，但能感受
  - 感受蚂蚁在身上爬行
  - 感受溪水在脚下流淌
  - 感受照射的阳光温暖
  - 体验久违的生命气息
  - 不再忍受恶灵区的阴冷

**3.12 泉水的第一次滋养（快速恢复）**
- 时间：附身美女榕后的第一天
- 地点：美女榕处，泉水边
- 事件：
  - 脚下的泉水滋润朱围庸
  - 仅一天时间就恢复大量体力
  - 与之前第九层花一个月恢复形成巨大对比
  - 朱围庸感到精力充沛

**3.13 凡人的对话（确认真实）**
- 时间：恢复后某时刻
- 地点：美女榕处
- 事件：
  - 老吴对老伴说树的姿态变了
  - 老伴疑惑树的腿摆放位置是否改变
  - 老吴靠近仔细观察
  - 朱围庸吓得屏气凝神，一动不敢动

**3.14 朱围庸的人生新阶段（幸福时刻）**
- 时间：老吴和老伴离开后
- 地点：美女榕处
- 事件：
  - 朱围庸可以舒展身体
  - 双腿在泉水中荡来荡去
  - 好不自在
  - 确信自己真的来到了凡人世界
  - 虽然看不到花花世界，但不感到忧伤
  - 现在很快乐
  - 希望就这样一直过下去

### 第四章：旅行团

#### 时间：22世纪初春日至冬日

**4.1 旅行团的组织（起始）**
- 时间：春日或初夏
- 地点：牛台山和周围
- 人物：导游丁路（二十多岁）、旅客小周（近30岁，换工作间隙）、中老年游客十几人
- 事件：
  - 丁路热衷探险，观察地势
  - 自组织旅行团
  - 大多数是退休中老年人

**4.2 登山和游玩（冒险期）**
- 时间：白天
- 地点：牛台山各地
- 事件：
  - 旅团从西南侧开始登山
  - 走到山顶
  - 从东南侧下山
  - 下到半山腰听到流水声
  - 发现小溪，兴致高涨
  - 大家开始去泉水中玩耍

**4.3 导游的疲惫与睡眠（关键时刻）**
- 时间：下午
- 地点：花溪旁
- 事件：
  - 丁路感到疲惫
  - 大家去玩水
  - 丁路找地方坐下
  - 困意来临，决定睡觉
  - 脚翘起搭在旁边树根上（美女榕的树根）

**4.4 朱围庸的第一次移魂夺魄（转变时刻）**
- 时间：丁路睡眠期间
- 地点：美女榕处
- 事件：
  - 朱围庸感受到压迫感（丁路的脚踩在他身上）
  - 起初担心是老吴试探
  - 半小时后听到呼噜声（丁路睡着了）
  - 朱围庸微微动弹
  - 感觉被吸引到另一个地方（丁路的身体）
  - 更舒适、柔软、温暖
  - 逐步探索身体（脚、膝盖、肚子、屁股、手、耳朵）
  - 用力一拱，成功将丁路的魂拱出去
  - 朱围庸的魂和丁路的魄合二为一

**4.5 新生的喜悦与适应（初体验）**
- 时间：移魂成功后
- 地点：旅行地
- 事件：
  - 朱围庸睁眼时感到肿胀疼痛
  - 缓慢适应光线（从白茫茫到树叶绿色）
  - 看到深蓝天空和白云
  - 检查双手，感受生命存在
  - 爬起来，感觉口渴
  - 去泉水边喝水
  - 躺下休息，感受生命力流淌
  - 耳朵嗡鸣声逐渐消退
  - 眼睛开始适应光线
  - 双手逐渐恢复力气

**4.6 旅团的询问与朱围庸的困境（突发困难）**
- 时间：下午到傍晚
- 地点：旅行营地
- 事件：
  - 旅团玩了两三小时
  - 大家开始疲惫并提议下山
  - 旅客问导游是否知道下山路和时间
  - 朱围庸什么都不知道，一时不知所措
  - 小周来到身边询问
  - 朱围庸勉强同意下山
  - 凭直觉带队在森林中左拐右转
  - 天色渐暗

**4.7 迷路与露营（被困第一晚）**
- 时间：傍晚到晚上
- 地点：山上某处
- 事件：
  - 旅客焦急询问是否迷路
  - 朱围庸承认迷路
  - 决定露营而不是冒险下山
  - 大家接受意见
  - 小周帮忙搭帐篷
  - 月光明亮，可以看清东西
  - 大家简单进食
  - 大家聚集聊天

**4.8 朱围庸的虚弱与恢复（被困第一晚后）**
- 时间：入夜后
- 地点：帐篷区
- 事件：
  - 朱围庸感到异常疲劳，天旋地转
  - 这是魂与魄的磨合期
  - 容易疲劳虚弱
  - 休息约两小时仍然疲惫
  - 爬起来想小解
  - 蹒跚经过各帐篷
  - 不小心碰到一对夫妇的帐篷
  - 蹲下帮他们关帐篷
  - 感受到一股力量流入体内
  - 瞬间有了力气

**4.9 朱围庸发现精气吸取能力（黑暗转折）**
- 时间：夜晚露营期间
- 地点：各帐篷

## 五、故事的时间跨度与节奏

- **背景历史**：从周朝到清末，跨越超过2000年的灵魂关押历史
- **现代故事**：从22世纪初冬季开始，至同年冬季结束
- **故事密度**：约1年时间内完成第一部分五章的所有事件
- **节奏特点**：
  - 前两章较为缓慢（老吴恢复与建房）
  - 第三章为转折点（朱围庸逃脱）
  - 第四章开始加快（冲突升级）
  - 第五章达到高潮（救援与真相）
