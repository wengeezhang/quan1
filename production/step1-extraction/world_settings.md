# 世界观设定

## 视觉规则体系

### 灵魂的视觉表现
- **普通灵魂（游魂/游丝）**：白色或淡色微弱光点/丝状体，在轮回通道中如人流般缓缓移动
- **恶灵灵魂**：不规则褶皱圆形光团；颜色因罪孽标记深浅不同（深紫色=重罪标记如朱围庸；浅色=较轻者如李仁胡酹）；外缘波纹振动代表情绪；光团密度和大小随状态变化（虚弱时松散扩散，激动时紧缩闪烁）
- **封（周朝古灵）**：比其他灵魂更大更凝实，表面皱纹如古树年轮般深而有序，古铜/深灰色调
- **元神**：十八元神为更高级存在，显现时为半透明发光巨大人形（数十米高），明亮白光

### 移魂夺魄（灵魂附身）的视觉过程
- **附身过程**：灵魂光团从外部侵入睡眠中人体；原主灵魂被挤出体外（白色微弱光点无助飘走）
- **排异反应**：新附身初期，宿主面色苍白、极度虚弱；随适应逐渐消失
- **精气吸取**：附身者通过近距离接触吸取他人"精气"维持附身——被吸取者逐渐变为"植物人"（灰白面色、空洞眼神、全身僵硬）
- **远程附身**：后期发展出远程移魂夺魄能力——需要视线接触；被黑布遮挡可阻断
- **跳体逃脱**：附身者假装倒地昏迷，灵魂瞬间转移到附近另一个人体内——外观上前一个人倒下、旁边一个人突然眼神变化

### 泉水效果的视觉表现
- **治愈/逆转衰老**：饮用后立竿见影——皮肤紧实、皱纹淡化、肌肉恢复力量、视力恢复（可用渐变特效表现）
- **恢复被吸取者**：灰白面色逐渐恢复红润血色，僵硬肢体恢复活动
- **维持还阳者**：还阳者需定期饮用泉水维持灵魂与肉体的稳定连接
- **泉水断供后果**：记忆力衰退、身体机能退化（老吴的案例）

### 数字人的视觉特征
- **制造过程**：原子级扫描重建人体——科幻感的制造场景
- **空壳状态**：完美人类外形但毫无生气，空洞无神的眼睛，如同真人大小的人偶
- **注入灵魂后**：瞬间恢复生命力——眼神变得锐利，展示悬浮、超人力量等能力（光芒闪烁）
- **普通灵魂版（元刚）**：清新面容但天真无害，如新生婴儿般空白
- **元神版（智刚/威刚）**：气质超凡，展现时可变为数十米高发光巨人

## 阵营与视觉对立

### 还阳者阵营（朱围庸一方）
- **代表色**：深紫/暗红——源自恶灵罪孽标记的紫色和朱围庸标志性的红色披风/铠甲
- **视觉基调**：暗色系服装、低调精干、军事化纪律；超自然攻击以紫色光效为主
- **标识**：红领巾/红臂章（控制区安全通行标志）
- **武器特征**：电磁枪（科幻风格非致命武器）；灵魂收集容器

### 人类抵抗军/正义联盟
- **代表色**：无统一色系，以朴素实用为主
- **视觉基调**：便装/军装混搭，战争磨损感，疲惫但坚定
- **防御标志物**：黑布阵（大幅黑布遮挡城墙）；大蒜（融入军粮的天然克制物）

### 杨弘政权（后期独立势力）
- **代表色**：金色/铜色——权力与腐败
- **视觉基调**：初期英雄气质的军服，后期日渐臃肿奢华；雕像、宫殿等纪念物
- **暗面**：秘密警察、地下实验室、数字人制造设施

### 天界/上神
- **代表色**：金色光芒、纯白
- **视觉基调**：宏伟神殿、发光的巨大存在、超凡脱俗的空间感
- **和寻灵**：金色/暖色光芒，人形但非实体
- **智寻道**：暗蓝色雷电主题，带动物图腾（龙、麒麟）

### 阴间/恶灵区
- **代表色**：黑色、深灰、冷色调
- **视觉基调**：无光黑暗、压抑封闭；十八层塔中层层递进的酷刑场景
- **灵魂管理中心**：相对有序的灵魂管理空间
- **恶灵控制中心**：数百万光团在黑暗中飘浮的震撼画面

### 神兽族
- **代表色**：纯白（毛发）+ 纯黑（面部）
- **视觉基调**：5-6米高的原始巨大生物，白色长毛约20cm，面部简化纯黑；动作缓慢仪式感或暴烈极端（性格不同）
- **视觉对比**：白巨的悲悯平静 vs 灵尤的暴烈疯狂

## 空间体系

### 三界空间结构
```
人间（阳间）
    │
    │ 美女榕树根通道（牛台山地下约200米）
    │
阴间
    ├── 魂魄管理中心（管理普通灵魂轮回）
    ├── 恶灵控制中心（关押恶灵）
    │   └── 十八层塔（灵魂赎罪/攀升通道）
    └── 隔离环（无肉身灵魂的维度监狱，灰色无限循环空间）

上神界（天界）
    └── 神殿大厅（上神议事/审判之所）
```

### 空间转换规则
- **人间→阴间**：通过美女榕树根通道（物理通道）或灵魂出窍（精神通道）
- **阴间→人间**：攀爬十八层塔至塔顶触碰树根（朱围庸的方式）或被管理者送回
- **人间→上神界**：只有上神使者（宋小仙）可被召唤前往
- **隔离环**：无肉身锚点的灵魂自动被轮回系统吸入，困在灰色无限循环空间中
- **美女榕树根被切断**：第70章朱围庸切断树根，永久关闭阴阳两界通道——千年超自然循环终结

### 空间视觉差异
- **人间**：正常的自然光线、色彩饱满的自然景观和人类建筑
- **阴间（恶灵区）**：无自然光源的黑暗空间，数百万灵魂光团在黑暗中飘浮，冷色调
- **隔离环**：永恒的灰色虚空，无边界感，无限循环
- **上神界**：金色光芒充盈的宏伟空间，非人类尺度的巨大建筑/存在，超凡脱俗
- **十八层塔内部**：层层递进，每层有不同酷刑/试炼景象，越往上越艰难
