# 第四部 第三十五章 人物画像

### 1. 朱围庸
- **基本信息**：还阳者势力的最高领袖，战略家与决策者。
- **外貌锚点**：本章更多体现其深思熟虑的枭雄气质。可以有他独自考察三镇时，站在高处俯瞰地形的背影，眼神锐利，嘴角或带一丝冷笑。在向手下解释新武器时，神情从容，带有一种洞悉世事的嘲讽。
- **服饰与道具**：便服，但质地精良，便于行动。关键道具是新采购的**电磁枪**，造型应具有未来感，区别于传统火器。
- **关键场景表现**：
    - **决策扩军**：听到“肉身追随”现象后，不是简单的兴奋，而是立刻意识到其战略价值，说出“真是冥冥中在帮助我们啊！”时，应有一种掌握天意的自信。
    - **介绍新武器**：面对胡酹和李仁对电磁枪的质疑，他显得很有耐心，像一个导师在阐述一种新的哲学。他用枪瞄准胡酹的动作，是带着玩笑和展示性质的，而非真正的威胁，体现其对手下的掌控力。
    - **部署总攻**：在地图前进行部署，条理清晰，命令果决，展现其军事才能。
- **连贯性约束**：延续其作为团队大脑和绝对核心的定位，本章其“枭雄”形象因其战略布局和对时代荒诞性的深刻理解而更加丰满。

### 2. 李仁
- **基本信息**：朱围庸的副手，团队的理论家与主要管理者。
- **外貌锚点**：神情依然沉稳，但在讨论召唤恶灵和管理千人队伍时，眼神中透露出精于计算与规划的智慧。在面对新武器时，眉头微蹙，流露出对传统方式的路径依赖。
- **服饰与道具**：深色系的合身便服，显得干练而低调。
- **关键场景表现**：
    - **解释现象**：当罗勇提出“肉身追随”时，他立刻从理论层面（魂魄融合与吸引）给出解释，奠定其“军师”地位。
    - **执行运输**：指挥手下在夜间大规模运输“空壳”肉身，场景应体现其行动的隐秘、高效和有条不紊。
    - **质疑新武器**：他认为电磁枪不如火枪“靠谱”，代表了团队中相对务实和保守的一面，关心的是效率和结果。
- **连贯性约束**：保持其作为朱围庸左膀右臂的形象，是战略的完美执行者和补充者，但在思想上不如朱围庸那般激进和具有前瞻性。

### 3. 胡酹
- **基本信息**：团队的武力担当，行动派的代表。
- **外貌锚点**：性格直率，情绪外露。在听到可以扩充千人队伍时，表现出毫不掩饰的兴奋。在把玩新武器时，眼神充满好奇，像个得到新玩具的孩子。被朱围庸用枪指着时，会下意识地做出夸张的躲闪动作。
- **服饰与道具**：穿着便于活动的劲装，可能随身携带某些旧时代的武器作为习惯。
- **关键场景表现**：
    - **发现并上报**：他能意识到罗勇发现的现象的重要性，并立刻上报，体现了他的责任心和一定的判断力。
    - **玩枪与躲闪**：与朱围庸互动时，他的反应是本章中一个轻松的调剂。从把玩新枪的好奇，到不解其用，再到听闻其效果后的惊慌躲闪，生动地刻画了他勇猛但略带鲁莽的性格。
    - **接受任务**：在接受带领一队人马从海上进攻的任务时，应表现出坚毅和自信。
- **连贯性约束**：延续其勇猛、直率、忠诚的“大将”形象，是团队中最锋利的矛。他对新事物的反应也符合其简单直接的思维模式。

### 4. 罗勇
- **基本信息**：胡酹手下的基层头目，关键现象的发现者。
- **外貌锚点**：外形普通但眼神敏锐，善于观察。
- **服饰与道具**：普通的还阳者成员着装。
- **关键场景表现**：向胡酹汇报时，神情严肃，语气肯定，体现其观察的细致和对异常情况的警觉。在结尾被朱围庸委以留守泊岗镇的重任，应表现出领命时的坚定。
- **连贯性约束**：作为基层骨干的代表，其作用是串联起高层决策与底层执行。

### 5. 新还阳者群像（一千二百人）
- **基本信息**：被新召唤的恶灵填充的“空壳”肉身，组成了恶灵军团的主力。
- **外貌锚点**：可以描绘他们被集中管理、训练的宏大场面。他们的表情可能还带有一丝初生的迷茫，但在统一的号令下行动一致，形成强大的视觉冲击力。
- **服饰与道具**：统一或制式化的着装，以区别于普通民众，并装备了新式电磁枪。
- **关键场景表现**：在李仁和胡酹的带领下，作为两路大军向三镇进发，队列整齐，气势磅礴，宛如乌云压境。插画可以采用广角或航拍视角，强调其数量上的压倒性优势。
- **连贯性约束**：他们是朱围庸野心和力量的具象化体现，是开启大反攻时代的关键“资源”和“武器”。
