
# 第五十四章 人物画像

本章是典型的双线叙事，舞台聚焦于AA大学校园内部，核心人物是新登场的科学家吴启思，以及作为其对手盘的马戈逸与朱亮。他们的互动构成了本章的核心戏剧冲突。

### 1. 吴启思

- **外貌与神态**：
  - **核心锚点**：**“理想主义的疯子”科学家**。他是一位年长的学者，外貌可能有些不修边幅，但眼神中透着智慧与偏执。当他出现在食堂时，姿态从容、慢条斯理，面对一群背景不明的“占领者”，他毫无惧色，反而带着一种高级知识分子特有的自信与疏离感。
  - **细节**：可以设定他有花白的头发，戴着一副老式眼镜，衣着是有些陈旧但很干净的实验服。

- **服饰与道具**：
  - **服饰**：白色或浅色的实验室工作服，可能有些褶皱。
  - **道具**：食堂的餐盘、实验室里的各种仪器、以及最重要的——躺在实验台上的“数字人”。

- **关键场景建议**：
  - **食堂初登场**：他独自一人走进喧闹后归于沉寂的食堂，坦然地打饭、吃饭，与周围充满疑虑和警惕的氛围形成鲜明对比。
  - **实验室“引诱”**：他坐在实验台前，背对或侧对门口，专注地工作，但眼角的余光却时刻留意着窗外的窥探者。这个场景需要表现出他“导演”一切的掌控感。
  - **主动“求助”**：在食堂与马戈逸对话时，他的姿态是平等的，甚至是略带一丝“考究”的，仿佛在面试一个合作对象。他说出“需要帮手”时，语气应该是非常自然的，不带任何乞求的意味。

- **连贯性约束**：
  - 吴启思是本章引入的关键人物，他身上混合了对科学的纯粹追求和为了实现理想不择手段的疯狂。后续章节中，他的这种复杂性需要保持。

### 2. 马戈逸

- **外貌与神态**：
  - **核心锚点**：**谨慎的决策者**。作为这支潜伏部队的最高指挥，他始终保持着高度的警惕。在面对吴启思这个不确定因素时，他的第一反应是怀疑和防备。他的神情总是很凝重，习惯性地锁着眉头。
  - **细节**：在与朱亮讨论时，他可能会不停地来回踱步，这是他内心焦虑和思考的外在表现。

- **关键场景建议**：
  - **食堂观察**：当吴启思出现时，他坐在人群中，不动声色地观察着，眼神锐利，试图从对方的言行中分析出其真实目的。
  - **决策的犹豫**：当朱亮提出“数字人”的巨大价值时，他的脸上会闪过一丝兴奋，但立刻又被忧虑所取代。他担心这是荣康城设下的陷阱，这种内心的挣扎可以通过微表情来展现。

- **连贯性约束**：
  - 马戈逸的性格基调是多疑和谨慎，这与他所承担的巨大责任有关。他与朱亮的“激进”形成了鲜明的对比和互补。

### 3. 朱亮

- **外貌与神态**：
  - **核心锚点**：**大胆的执行者与机会主义者**。与马戈逸的谨慎不同，朱亮更倾向于抓住眼前的机会。他看到了“数字人”的巨大潜力，并愿意为此冒险。他的眼神中充满了渴望和兴奋，行动力极强。

- **关键场景建议**：
  - **潜入实验室**：他带领手下进入吴启思的实验室，对那个“人形生物”充满了好奇，会凑得很近去观察。
  - **说服马戈逸**：他向马戈逸分析为什么吴启思不像是陷阱时，言辞恳切，逻辑清晰，充满了说服力。
  - **主持移魂实验**：在实验室里，当机会出现时，他没有过多犹豫，果断下令进行尝试。实验成功后，他的兴奋溢于言表。

- **连贯性约束**：
  - 朱亮是团队中的“行动派”，他的勇敢和果断推动了情节的发展。后续他将继续作为探索吴启思秘密的先锋。

### 4. 数字人

- **外貌与神态**：
  - **核心锚点**：**完美的“空壳”**。它拥有完美的人类外形，皮肤、肌肉、毛发都栩栩如生，但眼神空洞、呆滞，没有任何神采。当被植入灵魂后，它的眼神会瞬间发生变化，从“物”变成了“人”。
  - **细节**：身上连接着各种导线和输液管，躺在冰冷的金属实验台上，与它栩栩如生的外貌形成强烈的科技与生命的冲突感。

- **关键场景建议**：
  - **苏醒的瞬间**：当移魂成功，它的眼睛会突然聚焦，然后缓缓地眨一下。这个特写镜头将是本章最具震撼力的画面之一。接着，它可以尝试性地活动手指，感受新的身体。
