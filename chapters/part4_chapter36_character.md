# 第四部 第三十六章 人物画像

### 1. 朱围庸
- **基本信息**：还阳者军团的最高领袖，本章中同时扮演着“主帅”“王牌”与“顶级间谍”的多重角色。
- **外貌锚点**：在战场上，他应展现出不怒自威的强大气场，眼神锁定目标时充满杀气。在制定和执行“木马计时”，又恢复了那种深不可测的冷静。附身老者时，需通过佝偻的姿态和浑浊的眼神来伪装，但在无人注意的瞬间，眼神会闪过一丝精光。
- **服饰与道具**：自己的身体穿着便于战斗的深色劲装。附身的老者则穿着本地常见的、陈旧的平民服饰。
- **关键场景表现**：
    - **亲临前线**：当战局僵持时，他并非在后方督战，而是选择亲自成为攻破僵局的尖刀，体现其“兵王”本色。插画可描绘他如鬼魅般穿梭于敌阵，轻易夺取他人肉身的瞬间。
    - **定计与说服**：在解释“木马计”时，他应表现出绝对的自信，对李仁和胡酹的担忧，他并非强硬压制，而是通过分析人性和局势来使其信服，展现其领导艺术。
    - **戏仿老者**：在城门下求救、在指挥部谎报军情、向指挥官“乞求”安身之所，每一幕都需在老者的外表下，精准传达出卑微、急切又无助的表演层次。
    - **回归真身**：在最终胜利后，他从指挥官身体中脱离，回归自己肉身，那一刻可以有一种大局已定的淡然与疲惫。
- **连贯性约束**：本章极大地丰富了朱围庸的形象，他不再仅仅是运筹帷幄的枭雄，更是敢于将自己置于险境并精准执行高难度任务的“全能战士”，其冷酷、智谋与胆识都达到了新的高度。

### 2. 李仁
- **基本信息**：朱围庸最倚重的副手，负责正面战场推进与后续接应。
- **外貌锚点**：稳重冷静，但在听闻朱围庸要亲自冒险时，脸上会流露出明显的担忧和不赞同，眉头紧锁。
- **服饰与道具**：标准的作战服饰。
- **关键场景表现**：
    - **战术执行者**：在初期进攻和最终的攻城战中，他是完美的执行者，带领部队冲锋在前。
    - **担忧与劝阻**：他与胡酹一同劝阻朱围庸，代表了团队内部理性的声音，关心领袖安危，也体现了他对朱围庸在整个计划中不可替代作用的认知。
    - **耐心等待**：在城外埋伏时，应表现出紧张而专注的神情，时刻准备响应城内信号。
- **连贯性约束**：延续其作为团队“压舱石”和“大管家”的定位，忠诚、可靠，是朱围庸可以放心将后背交予的伙伴。

### 3. 胡酹
- **基本信息**：团队的武力担当，对朱围庸绝对忠诚。
- **外貌锚点**：性格更为外露，听到大哥要冒险，他的反应比李仁更激烈，是第一个请缨代替的人。
- **服饰与道具**：作战服饰，可能随身携带标志性的武器。
- **关键场景表现**：
    - **请缨出战**：在朱围庸提出计划后，他毫不犹豫地要求代替大哥去执行最危险的部分，体现其赤胆忠心和悍不畏死的勇士本色。
    - **攻城先锋**：城门打开后，他应是第一个率队冲入城中的人，动作迅猛，势不可挡。
- **连贯性约束**：保持其勇猛、直率、忠诚的“第一猛将”形象，是朱围庸最锋利的矛。

### 4. 葛家堡最高指挥官
- **基本信息**：三镇抵抗力量的最后领袖，一个悲剧性的角色。
- **外貌锚点**：中年男性，面带愁容，但在部署防御时仍保持着镇定和威严。
- **服饰与道具**：可能是当地某种民兵或自卫队的制服。
- **关键场景表现**：
    - **打断会议**：被朱围庸扮演的老者打断会议时，虽有不悦但仍保持了基本的礼貌和耐心。
    - **轻信谎言**：在听到“挖地道”的假情报后，他与同僚虽有疑虑但最终还是选择相信并调动兵力，反映了在信息不对称和巨大压力下的决策失误。
    - **引狼入室**：出于对老者的同情和进一步获取情报的考虑，他做出了让“老者”与自己同住的致命决定。
- **连贯性约束**：他代表了正常人类在面对未知诡异力量时的无力与脆弱，他的失败并非因为愚蠢，而是因为无法想象敌人会以何种方式进行渗透。

### 5. 还阳者军团群像
- **基本信息**：作为攻城主力的两百名核心成员与被控制的两万名“肉身”。
- **外貌锚点**：在攻城时，他们应如潮水般涌入，动作统一，没有恐惧，形成强大的视觉压迫感。
- **关键场景表现**：在城门打开的瞬间，从黑暗中蜂拥而出的场景，是本章最具冲击力的画面之一。可以描绘他们在李仁和胡酹带领下，迅速控制城内各处要道，使抵抗瞬间瓦解。
