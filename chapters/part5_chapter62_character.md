# 第六十二章 人物画像（插画参考）

## 于永龙

- **整体定位**：从“小跟班研究员”正式迈入权力场的新人部长，本章情绪起伏最大的人物之一。
- **外形神态锚点**：
  - 年近三十的青年，身材中等略偏瘦，动作略显急躁；说话时经常前倾身体，表情丰富。
  - 在家中拉着于崇宝出门时，眼睛发亮、带点狡黠；见到杨弘宣布升官时，则是难掩笑意，眉飞色舞，甚至有些失态的小得意。
- **服饰与道具**：
  - 在实验室多为简洁的工作服、白大褂或朴素中山装，袖口经常挽到小臂，方便操作仪器。
  - 可在腰间或手上安排笔记本、夹板等小道具，强调他同时兼顾“技术汇报”和“跑官”的角色。
- **典型场景构图**：
  1. 站在元刚身边，用手比划着墙高，表情诧异地说“连墙都上不去”，凸显他心中落差。
  2. 在杨弘面前听到“升为二级部长”时，眼睛瞪大、嘴角上扬，身形微微前探，和一旁淡定的永智形成对比。
  3. 走出实验室后，仰头大笑或兴奋地挥拳，背影中透出一个突然被权力点燃的青年。
- **跨章连贯性提示**：
  - 后续章节中，于永龙会更频繁出入权力中枢，他的衣饰可以逐渐从纯技术感向略带官气的打扮过渡，但表情与动作应始终带有一点“爱现”“好热闹”的特质。

## 于永智

- **整体定位**：数字人计划的技术核心，性格冷静克制，与永龙形成鲜明对照。
- **外形神态锚点**：
  - 外形与前章保持一致：瘦削、高个，戴眼镜或目光略显专注，给人“书生工程师”的感觉。
  - 在与杨弘对话时，回答简洁、语气平稳；当永龙沉浸在升官的幻想中时，他只是侧身离开，眉头微皱，流露出对元神问题的忧虑。
- **服饰与道具**：
  - 一贯的实验服或深色简朴衣物，胸前可配笔袋、口袋里塞着折叠的实验记录。
  - 实验室背景中可以有仪器、瓶瓶罐罐、数据本等，强化其技术角色。
- **典型场景构图**：
  1. 与永龙一起站在元刚身后，低声讨论测试结果，表情严肃。
  2. 面对杨弘时，神情克制，略带紧张但不卑不亢，用手轻托眼镜或低头翻看记录。
  3. 最后一幕，他转身离开正在做官梦的永龙，一个人走回实验台前，背影略显孤独，突出“担忧未解”的情绪。
- **跨章连贯性提示**：
  - 在后续涉及数字人技术的章节中，于永智应始终保持“冷静、理性、略带焦虑”的气质，让读者一眼就能识别他是技术与良心的代表。

## 于崇宝

- **整体定位**：掌握水晶棒和元神秘密的关键人物，本章只短暂出场，却推动剧情朝“回山查探”发展。
- **外形神态锚点**：
  - 与前文保持：略显精干结实，比永龙更粗犷，常带几分江湖气。
  - 刚被叫醒时表情困倦、烦躁；听到“数字人醒了却没本事”后，立刻从迷糊变为警觉，眼神一紧，坐直身子。
- **服饰与道具**：
  - 居家场景中可以是随意的背心、短袖或半扣的衬衫，床边凌乱，增加生活气息。
  - 提到要回少林山时，可以加一个简单的行囊、背包，暗示即将远行。
- **典型场景构图**：
  1. 被永龙推醒，从床上坐起，眉头紧皱、头发略乱，房间光线偏昏黄。
  2. 在元刚住处，一手叉腰、一手指着元刚，神情专注地追问“还记得威刚吗？”，元刚在对面茫然摇头。
  3. 临别前对永龙、永智说“明天就动身”，侧身站在门口，背后是渐暗的天空，预示他将走向隐藏秘密的山洞。
- **跨章连贯性提示**：
  - 后续若出现水晶棒、少林山洞等情节，可在画面中延续他略带江湖痞气又精明的形象，让观者自然联想到“元神调度”的那条暗线。

## 元刚

- **整体定位**：看似普通的新数字人，缺乏超凡能力，却是后续剧情的重要棋子。
- **外形神态锚点**：
  - 年轻男子，面容清爽干净，眼中带有一点迷茫和好奇，没有威刚那种凶悍或锐利的气质。
  - 面对几位“前辈”时略显拘谨，说话前会看向永智，仿佛在寻找提示。
- **服饰与道具**：
  - 简洁、合身的校服或实验室统一服装，颜色偏浅，象征“白纸一张”的状态。
  - 桌上可以摆放简陋的饭碗、筷子，突出他刚刚学会日常生活的感觉。
- **典型场景构图**：
  1. 在简陋房间里端着饭碗，被永智叫住，半起身、半坐着，表情局促。
  2. 面对杨弘检查时，被围着上下打量，微微低头，却努力保持镇定。
  3. 回答“我叫元刚”时，语气认真，脸上带着一丝希望，暗示他也想拥有一个完整身份。
- **跨章连贯性提示**：
  - 之后若展现他与元神、威刚之间的联系，应保持“纯净外表 + 内心渐生困惑”的双重感受。

## 杨弘

- **整体定位**：权力顶端的统治者，在数字人面前既像挑选武器，又像审视新仆，从怀疑快速转为狂喜。
- **外形神态锚点**：
  - 中年男性，体态略显发福，但眼神锐利，习惯双手背在身后或负手踱步。
  - 检查元刚时，眉头先紧后舒，嘴角慢慢上扬，呈现从谨慎到兴奋的变化。
- **服饰与道具**：
  - 延续前文的高位官服或军装，颜色偏深，辅以肩章、徽记等权力象征。
  - 在实验室里可以配以随行侍卫或书记官，强化其地位；背景可有高背椅、徽记旗帜的暗影。
- **典型场景构图**：
  1. 站在实验室中央，单手指着元刚，绕圈打量，其他人恭敬地站在一侧。
  2. 宣布成立数字人部门、任命永龙为二级部长时，抬手示意、嘴角含笑，画面中永龙激动、永智沉静，形成对比。
  3. 临走时，一手搭在元刚肩上，把他领出实验室，背影并行，像牵走一件新得到的“秘密武器”。
- **跨章连贯性提示**：
  - 后续涉及杨弘与数字人、还阳者对抗时，应保持他“算计深、情绪起伏大”的特点，让插画中的表情既有雄心也有隐忧。

---

以上人物设定建议与前几章保持统一美术风格：现实主义基础上略带夸张的表情与动作，用光影和站位突出权力关系与心理微妙差异，为后续数字人故事线提供连贯的视觉基调。
